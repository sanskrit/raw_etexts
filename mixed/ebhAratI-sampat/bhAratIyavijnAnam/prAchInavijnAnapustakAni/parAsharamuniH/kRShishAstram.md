**कृषिशास्त्रम्¹।**

**कृषिसमयनिर्णयः**

**अथातः सम्प्रवक्ष्यामि सङ्कीर्णविषये नृणाम्।  
क्रियाणामापि सत्कालमुहूर्तश्चविशेषतः॥**

**प्रथमं सम्प्रवक्ष्यामि समयं कृषिकर्मणः।  
शुभं वज्रिन् नृणाश्चैव देवानाञ्च विशेषतः॥**

**कृषिमूलंहि सर्वत्र प्राणिनां सम्पदस्सदा।  
कृष्यभावेनसम्पत्स्यात् तस्माच्चप्रारभेत्कृषिम्॥**

**पूर्वं सुभूप्रवेशः स्यान्निमित्तञ्चावलोकयेत्।  
मैत्रार्यमाश्विरोहिण्यां साहिर्बुध्न्यां कूजाहनि॥**

**स्वान्युत्तरेषुपुष्येच रौद्रे याम्येच चित्तभे।  
कुर्यात्प्रवेशनं भूमेः प्रथमं ... ...॥**

**वारेषु कौजः श्रेष्ठोऽत्र मध्यमौबुधजीवयोः।  
शेषावाराविवर्ज्याः स्युः सोमवारः कदाचन॥**

**... ... ... ... ... ...।   
प्रवेशोगो... नान्तु गवां सम्पदमिच्छतः॥**

**एवं प्रवेशनङ्कृत्वा वनच्छेदनमेवच।  
दहनादिकमप्येवं कृत्वा कृषिमथारभेत्॥**

**रोहिण्यादित्य पौष्णाश्च मूलहस्तोत्तराः क्रमात्।  
पुष्यस्तथानुराधाच कृषिकर्मणिपूजिताः॥**

**वारनक्षत्रयोगेषु ... ... भुवम्।  
एष्वेवशुभयोगेषु कृषिमप्यारमेत्सुधीः॥**

**पक्षच्छिद्रेतरास्सर्वेतिथयः कृषिकर्मणि।  
पूजितावृष्टिवर्ज्यश्च ...स्युः शुभावहाः॥**

**••• ••• तकरणाश्चविवर्जिताः।  
शुद्ध सम्पूर्ण... वर्जनीयास्तु कर्षकैः॥**

**वाराः शुक्रेन्दुजीवानां बुधस्यच शुभावहः।  
मन्दारवारौ द्वौ वर्ज्यौ सर्वतः कृषिकर्मणि॥**

**कर्किगोमृगमत्स्याश्वनृयुग्माश्चसुपूजिताः।  
शेषाश्चराशयः सर्वे वर्ज्याः स्युः कृषिकर्मणि॥**

**शुभग्रहोदयः श्रेष्ठो वर्ज्यः पापग्रहोदयः।  
निरीक्षणं तथैवैषां ग्रहाणान्तुशुभाशुभम्॥**

**एवङ्कालः समादिष्टः कृषिकर्मणिशोभनः।  
अन्ये दोषा न दोषाः स्युः कृषिकर्मणिकर्मणिवृत्रहन्॥**

**कृषिविधिः**

**अथवज्रिन्प्रबक्ष्यामिबीजनिर्वापणायच।  
कालं सर्वामरणाञ्च हितेप्सया...... ॥**

**गृहस्थाचारधर्मस्य मूलङ्कृषिरुदाहृतम्।  
अन्येषामाश्रमाणाञ्चगृ...हस्थाश्रमतः फलम्।**

**तस्मात्कृषिप्रधानेन बीजनिर्वापणेन च।  
नृणां पुरुषार्थसिद्धिः स्यादतस्तत्काल ...॥**

**तदास्यादार्तवो भूमेस्तदाबीजान्निवापयेत्।  
बीजानां तुविशेषेण धरार्तवविशेषतः॥**

**यस्यबीजाह्वयः कालो भूमेरार्तवसम्भवः।  
कालेतुतं... निजं शुभं निर्वापयेत्सुधीः॥**

**ज्येष्ठाख्ये मासि मूलाख्येत्वाषाढेच विशेषतः।  
सर्वेषाम्बीजजातानां धरायामार्तवो भवेत्॥**

**वारुणं वैष्णवं स्वातिपुष्य... रास्त्रयः।  
रोहिणीमूलमैत्राश्वहस्तपौष्णमघास्तथा॥**

**चतुर्दशीतास्ताराः स्युर्बीजनिर्वापणे शुभः।  
वासवादित्यसौम्याश्विताराः ... ... ॥**

**अन्या नवतु ताराः स्युर्वर्ज्याः स्युर्बीजवापने।  
वर्ज्येष्वपिचयो...च्छुभंस्याद्बीजवापनम्॥**

**... तिथयः श्रेष्ठाः पक्षयोरुभयोरपि।  
...नवमीयुक्तममावास्यां च वर्जयेत्॥**

**... ... ... गोगर्दभाह्वयाम्।  
गजकोलाहलाश्चैव वर्ज्याः स्युर्बीजवापने॥**

**द्वितीयादशमीषष्ठी मध्यमास्तिथयः परे।  
वर्ज्याः स्युस्तिथयोयुग्माबीजनिर्वापणेबुधैः॥**

**परेषु करणेष्वेवविष्टिवर्ज्येषुषट्सुच।  
कदाचिद्बीजवैशिष्ट्यं शुभम्बीजनिवापनम्॥**

**मृगगोकर्किसिंहाःस्युर्मीनश्चशुभदाः सदा।  
तुलामिथुनकुम्भाःस्यु र्मध्यमावर्जिताः परे॥**

**चन्द्रज्ञजीवशुक्राणां वारवर्गादयः शुभाः।  
... ... वर्ज्य क्षेष्वपिशोभनाः॥**

**... ... ... ... ... ।  
... ... ... ... ... ॥**

**सहजारि भवेत्क्रूरः शुभाः सौम्या न शोभनाः।  
विभवे चानयोः सौम्यखवर्गाश्वहयाछते॥**

**... ... ... ... ... ।  
अष्टमस्थाग्रहास्सर्वे नेष्टाः शुक्रस्तु कामगः॥**

**कदाचित् षट् भवर्ज्येषु ग्रहेषुशुभदाः शुभाः।  
अष्टमोदयवर्ज्येषु पापराशिषु शोभनाः॥**

**सप्तमे दिवसे ... ... ... ... ।  
शेषापञ्चशुभाःप्रोक्तादिवसाबीजवापने॥**

**सर्वदोषेषुवर्ज्याः स्युः बीजनिर्वापनाबुधैः।  
अपवादादिभिः कालेनिर्दोषे निवपेद्बुधः॥**

**देवप्रतिष्ठाः क्षौरञ्चवीजनिर्बापणं तथा।  
राजाभिषेकं ... ... मुपनीतिन्नकारयेत्॥**

**बीजनिर्वापणं कुर्यान्नक्षत्रे प्रोक्तदिङ्मुखे।  
सप्तसप्त ... ... प्राच्यादिषुचवापयेत्॥**

**प्राच्यां तु कृत्तिकाद्यास्तुमखाद्यादक्षिणामुखाः।  
मैत्राद्याः पश्चिमायांस्युः श्रविष्टाद्युत्तरामुखाः॥**

**कोणेषु भचतुष्केषु तारास्वष्टसुवर्जयेत्।  
परिखाख्यातुशून्यत्वात् बोजनिर्वापणेबुधैः॥**

**परिखादिङ्मुखाबीजान् गृहीत्वानव्रजेत्सुधीः।  
ग्रामदिङ्मुखनक्षत्रे बोजं तद्दिक्षुवापयेत्॥**

**सप्तमे दिवसे पूर्वं पञ्चमे बीजवापनात्।  
बीजाञ्जलाशये सिञ्चेत्साङ्कुरांस्तानथावपेत्॥**

**... ... बोजान्निर्वपेदन्यसस्यकान्।  
... ... ... ... ... ... ॥**

**सर्वान्दोषगुणानत्र निरपेक्षं प्रवक्ष्यते।  
बीजनिर्वापणे योगान् चतुर्मुखमुखोदितान्॥**

**आर्किवारे खमध्येर्के बीजान् ...... वापयेत्।  
मन्दवारे रुणे लग्ने बीजानुक्तानथावपेत्॥**

**भौमवारे कुजे लग्ने निर्वपेत्कोद्रवान् बुधः।  
भृगुवारे भृगोर्लग्ने कौसुमं तरुमावपेत्॥**

**गुरोर्वारेगुरोर्लग्ने स्थापयेत्फलिनस्तरून्।  
अन्यान्फलसमायुक्तान्स्थापयेदत्रवापयेत्॥**

**गुरोर्वारे गुरौलग्ने सर्वबीजानिवापयेत्।  
गुरौ चतुर्थगे कुर्याल्लवनं सर्वसस्यके॥**

**गुरावस्तमये धान्यान्सङ्गृह्णीयात्सदाबुधः।  
गुरौ दशमगे सर्वान्वपेद्भुक्तिशुभावहा॥**

**सूर्यवारे रवेर्लग्ने रक्तबीजान्प्रवापयेत्।  
सोमवारेचतल्लग्ने जलपुष्पाङ्कुरान्न्यसेत्॥**

**बुधवारेबुधेलग्ने निवपेत्कुमुदान् बुधः।  
सुगन्धसुमनोबीजान् शुक्रलग्नेतु विन्यसेत्॥**

**एवंवारोदये योगेप्रहाणां बीजवापनम्।  
शुभावहं प्रतिष्ठानां सद्यस्कालीन वापने॥**

**वृक्षाणां रोहिणी पूज्याशीतगोरूदयेबुधः।  
भरणीबृहतीनान्तु मेषलग्ने निवापनम्॥**

**आदित्येऽनुदये श्रेष्ठमिक्षूणान्तुनिवापनम्।  
स्वातीचसर्वसस्यानां श्रेष्ठं चन्द्रोदयं सदा॥**

**त्वष्ट्रभम्पूगसस्यानां जीवलग्नेशुभावहम्।  
अनूराधातु कङ्गूनां चन्द्रलग्नेशुभावहा॥**

**शततारासुतल्लग्ने कृष्णधान्यान्निवापयेत्।  
ताम्बूलवल्लि भेदानान्नैर्ऋृतेनैर्ऋृतोदये॥**

**मूलन्तुकन्दजातीनां जीवलग्नेशुभावहम्।  
इन्दुभेशालिजातीनां जीवेलग्नेशुभावहम्॥**

**अश्विन्यां पूगजातीनां शुभलग्नेशुभावहम्।  
कूश्माण्डादिषुवल्लीषु रोहिणीषु शुभावहा॥**

**हस्तपौष्णाश्विसौम्याश्चपुष्यमैत्रानिलोऽनलः।  
रोहिणी च प्रशस्ताः स्युः सर्वबीजनिवापने॥**

**एवमेवैषु कालेषु वैषाम्फलपरिग्रहे।  
एषां फलाशने चैव कालस्सम्यगुदाहृतः॥**

**रोहिण्यार्दमघाहस्तभरणीश्रवणेषुच।  
मन्ददृष्टस्थिरेलग्ने धान्यसङ्ग्रहमिष्यते।  
याम्यवैष्णववैशाखे बाल्येकर्कटवृश्चिके॥**

**प्रथमंलवनं कुर्यात्समृद्धिः शत्रुनाशनम्।  
कृत्तिकारौद्रसर्पेषु स्वात्यामर्यमभेतथा॥**

**विश्वभेशततारासु रेवत्यङ्गारवारगाम्।  
विष्टिरिक्ताविवर्ज्येषु तिथिषूक्तां...।  
शुभवारे शुभेलग्नेलवनं प्रथमंहितम्॥**

**......... तौतु पुरैवकथितं मया।  
भोक्तृ भुक्तिविशेषश्चतत्कालस्याशनंशुभम्॥**

**प्राजापत्यमघशततारासु युक्तयोः।  
अर्केन्दौचतयोर्लग्ने धान्यप्रच्छादनं शुभम्॥**

**वृषचापयमेशुक्र जीवसौम्याय भोदितः।  
कन्यकासम्भृतं धान्यं गर्भिणीसन्निवेषयेत्॥**

**वारनक्षत्रयोर्योगेशुभेगुरु ••• •••।

नन्दायां स्थिरलग्नेवाकण्टके शुभसंयुते॥**

**सर्वव्याधिविनाशायविदध्यादौषधम्पुनः।  
शुभवारतिथिष्वेषु शुभांशैकर्कटोदये॥**

**मैत्रेमित्रश्रविष्ठायाम् अश्विन्यां पुष्यचित्रयोः।  
औषधं व्याधिनाशायकुर्या ... कुलीरयोः॥**

**क्षिप्रोग्रचरभेवारे सौम्येरक्तासुमेहिनाम्।  
अष्टादशसु मेहेषु कर्तव्याप्यौषधक्रिया॥**

**पापवारोडुतिथ्यंशे पापराश्युदयेनृणाम्।  
पापदष्टे क्रियां कुर्यादौषधीः कुष्ठरोगिषु॥**

**शुभग्रहयुतेलग्ने षष्ठेशशुभसंयुते।  
चन्द्रे शुभक्रियायुक्ते कुर्यादौषधमुत्तमम्॥**

**शुक्रर्क्षे शुभवारेषु शुभनक्षत्रसंयुते।  
अरोगिणस्तु तुष्ट्यर्थं कुर्यात्तत्रौषधान्बुधः॥**

**शुभयोगेषुवारेषु शुभतिथ्यंशकेशुभे।  
लग्नेशुभग्रहे ... रसायनमुदाहरेत्॥**

**मन्दांशेमन्दलग्नेचमन्ददृष्टियुतोदये।  
मन्दराशौ क्रियाः कुर्याच्छिरसः कण्ठउत्तमाः॥**

**साधारणाङ्ख्यनक्षत्रेवक्रग्रहनिरीक्षणे।  
विष्टिवर्ज्येतिथौ कुर्यात्सत्क्रियाराजयक्ष्मणः॥**

**अपत्मारादि ... ... ... ।  
कर्तव्याः प्रोक्तकालेषुमन्त्रौषधिविशारदैः॥**

**विपरीताख्यविष्ठ्यान्तुकर्तव्यं शुभमिच्छताम्।  
औषधानाडिकाकाले अमृताख्येविषा... ॥**

**मूलश्विकृत्तिकापुष्यापुनर्वस्वसुवारगाः।**  
**यदिरोगी ... ... ... सुरवीभवेत्॥**

**रोहिण्युत्तरफल्गुन्योः आहिबुध्न्येतथामयी।  
सप्ताहाद्वा ...... सुखीभवतिवृत्रहन्॥**

**रोगचिकित्सायोग :—**

**सोमवारयुतेमूले चापलग्नेमहाद्रुमान्।  
स्थापये ज्जीवलग्नेवारेवत्यां गुरुवारगे॥**

**सार्पभेसौरिवारेण सहिते कर्कटोदये।  
स्थापयेदिक्षुवर्गादीन्कदलीक्रमुकानपि॥**

**सोमभेसोमवारेचसहिते कर्कटोदये।  
भेदात्कलमजान्सर्वान्निर्वपेत्तद्विवर्धये॥**

**माहेयवारे सौम्यर्क्षेक्रियलग्ने प्रियङ्गवः।  
निर्वाप्यविष्टिरिक्ताभिः वर्जितैस्तिथिभिर्युते॥**

**अहिबुध्न्याङ्गुरौवारेशुकांशेकर्कटोदये।  
निर्वपेञ्जलपुष्पाणां बीजं तद्वृद्धयेबुधैः॥**

**कोद्रवान्निर्वपेद्वृद्ध्यैकुजवारेयमोदये।  
अश्विनीरोहिणीमैत्रानक्षत्रसहितेयदि॥**

**सितवारायुतेशूर्पे आरण्यांश्चकृषीन् न्यसेत्  
.... .... .... .... ....  
.... .... चन्द्रैशशिवारेविषूदये।  
रक्षार्थं सस्यवर्गाणान्ध्वजः कार्योविचक्षणैः॥**

**मघा... श्चतुर्थांशे चाहिबुध्न्यस्यमध्यमे।  
ऐन्द्रस्य प्रथमेकुर्यात्सिंहपापोदयेध्वजम्॥**

**चापसिंहोदये .... .... .... ।  
सस्यमध्येथवारामेरौद्रर्क्षेमध्यमांशयोः॥**

**पुनर्वस्वाश्चतुर्थांशेचन्द्रजीवौयदोदितौ।  
तदा .... .... कदलीक्रमुकास्तथा॥**

**अदितेस्तुचतुर्थांशे चन्द्रजीव ...।  
उद्यन्तश्चेत्तदाकुर्यादायुष्कामीसुरक्षणम्॥**

**... ... .... .... ।  
सोमवारयुतेस्वातौ रोहिण्यांवापियूपकृत्॥**

**कृष्णपक्षेष्टमीयोगेचतुर्दशीमथापिवा।  
सौम्यभेसोमवारेणयूपः स्थाप्योमहर्द्धये॥**

**सूर्यवारेमहायोगे हस्तेवानैर्ऋतेपिवा।  
गोष्टागारन्तुसम्पोष्य स्थिरलग्नेगुरूदये॥**

**याम्यवैष्णवपैत्राणां चतुर्थांशे विधौस्थिरे।  
.... .... कुर्याद्धान्यसङ्ग्रहणं बुधः॥**

**चन्द्रेचसभृगौलग्ने स्वहोरायां स्थिरांशके।  
जीवद्दष्टेगृहे धान्यान्निक्षिपेद्वृद्धयेशुभान्॥**

**... ... चन्द्रेविलग्ने भौमवारगे।  
निखनेत्कूपवाप्यार्दांस्तटाकपरिखादिकान्॥**

**मन्दवारे ... वृषलग्नेशुभे...।  
सेतूनां बन्धनं कुर्यात्तटाका.भेदमागते॥**

**मन्दवार... वृषलग्नेचबन्धनम्।  
सेतूनाम... कुर्याच्छिलाबन्धादिबन्धम्॥**

**स्थिराण्यन्यानि सर्वाणिद्रव्यभेजीवमन्दयोः।  
पाठेदृष्टेगृहेचाभ्यां स्थिरेसेतुं निबन्धयेत्॥**

**पूर्वाषाढेरवेर्वारेवारूणर्क्षेभृगोर्गृहे।  
... ... कुर्यात्कीलद्वारेतटाकतः॥**

**रोहिण्यां मैत्रभेऽथवा  
... ... ... सिंहलग्ने शुभे... ॥**

**पूर्वाषाढेचवारुण्यां कुजवारे यमोदये।  
प्राकाराणां ... ... ... ॥**

**... ... ... ... ... **

**सस्यरक्षादिविधि :—**

**... ... निशानाथेशशिवारे स्थिरांशके।  
निलयान्नगरान्वापि सिंहलग्ने तथारभेत्॥**

**जीवशुक्रयुतेचन्द्रेस्वराशौ लग्नगेतदा।  
गृहारम्भः शुभः प्रोक्तो... ... ...॥**

**कृष्णपक्षेचतुर्दश्यां रेवतीरोहिणीयुते।  
यदात्तदा गुरौ लग्ने गृहार्थं संहरेद्द्रुमान्॥**

**वारनक्षत्रयोगेषु शुभाख्येष्वखिलेष्वपि।  
जीवलग्ने गुरौवारेगृहारम्भः शुभावहः॥**

**ध्रुवभेशुभवारेषु स्थिरलग्नेशुभेतिथौ।  
द्वारिस्थाप्योद्रुमञ्चित्रः सर्वसम्पद्विवर्धनम्॥**

**चरस्थिरेऽथनक्षत्रे बुधशुक्रदिनेतिथौ।  
शुभेकवाटयोगः स्या ... ... गृहे॥**

**चन्द्रलग्ने स्थिरांशेचबुधलग्नेशुमेक्षणे।  
मन्दिरादौ गृहान्तेषु स्तम्भः स्थाप्यस्तथोदितम्॥**

**जीवलग्ने भृगौकेन्द्रे शुभतारासुमूर्धनि।  
इष्टकामन्दिरेकार्ये सुरराजगृहादिषु॥**

**शुक्रलग्नेगुरौकेन्द्रे खगेराशौ गृहोपरि।  
तृणादिभिः समाच्छाद्य ... ... गृहे॥**

**रौद्रेन्द्रसार्पयाम्यानित्रिपूर्वानशुभावहाः।  
शेषास्ताराः शुभाख्यासुशुभकर्मणिशोभनाः॥**

**... तिथयस्सर्वे कुहूञ्चनवमीं विना।  
द्वितीयादशमीयुक्ताविष्टिवर्ज्याः शुभावहाः॥**

**शुभतारासु सर्वासु सर्वासुशुभवारे शुभांशके।  
शुभलग्ने शुभैर्दृष्टेशुभाख्यैस्तिथिभिर्युते॥**

**एषुयोगेषु कृत्यानिशुभान्याखण्डलशृणु।  
गजाश्वपशुभृत्यानां सङ्ग्रहं ... ... ॥**

**एषामारोहणं चैवकर्म ... ... ।  
राज्ञामास्थानपूर्वाणां मण्डपानां प्रवेशने॥**

**गार्हस्थ्यवेशनेचैवनराणां बन्धुदर्शने।  
निर्गत्यानुप्रवेशादि ... ... सम्भवान्॥**

**चिह्नान्वाहनपूर्वांश्चप्रतिपत्युदितांस्तथा।  
पुष्यस्थे ... ... शुभैः॥**

**दृष्टे कुर्याद्धयागारङ्गवाञ्चकरिणान्तथा।  
अस्मिन्योगेगृहारम्भगोष्ठागारादीनामपि॥**

**गृहेऽस्मिंश्चोरमाराद्या ... ... ... ।  
त्वाष्ट्रमैत्रान्त्यपादस्थेशीतगौमिथुनोदये॥**

**... विषमैर्दोषैर्विदध्यादमुनाबुधः।  
शुभतारासुसल्लग्ने शुभांशेशोभनेतिथौ॥**

**अङ्कुशं ... योज्यः शनिलग्नेशनेर्दिने।  
शुभवारे शुभलग्ने शुभेमासिशुभेतिथौ॥**

**शुभेदिने शुभेलग्ने खलीनादिहयादिना।  
सन्नाहादिनियोजयेत्**

**शूर्पभेसोमवारस्थे गृह्णीयात्प्रथमं घृतम्।  
... ... सोमवारेविधूदये॥**

**स्वतारांशयुतेचन्द्रे लग्नस्थे ... शुभम्।**  
**अस्मिन्योगेगवान्दोहः शुभक्षीर... ॥**

**सूर्यवारेरवौलग्ने कुजमन्दांशसंयुते।  
पौष्णोजीवांशके कुर्यात्कस्यचिन्मन्थनं बुधः॥**

**गृहारम्भस्तम्भकवाटाद्यश्वादीनाङ्गवाञ्च**

**... ... जीवलग्नेस्मरेम्वुगे।

**शुक्लपक्षस्यस्वातीयोगेविधूदये ...।  
विक्रयेसिद्धिर्यात्रायातिशुभप्रदाः ॥**

**सर्वत्रजीवलग्नेन्दोः ........ ।  
खस्थेऽर्केऽर्थान्सुगृह्णीयात्प्रारब्धञ्च ...॥**

**विष्णुशूर्पयमान्त्यां शेष्वलीक ... वदेत्।

**रोहिणीशूर्पयाम्यानाम् अन्त्यांशेवृश्चिकोदये।

**कृष्णाष्टम्यां च सप्तम्यां रौद्रभेयस्य कस्यचित्।

**यमभेकुजवारस्थे चतुर्दश्यष्टमीमिते।

**हस्ते शुभांशकेचन्द्रे गुरुलग्नेतिशोभनम्।

**शुभवाराः शुभाः प्रोक्ताः स्थितयः शोभनाश्चये।
