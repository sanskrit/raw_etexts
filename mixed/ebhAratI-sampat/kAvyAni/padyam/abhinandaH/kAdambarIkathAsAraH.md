**कादम्बरीकथासारः।**  
**———**

**प्रथमः सर्गः।**

**श्रियं दिशन्तु वः शौरेर्द्वये तुल्यश्रमाः क्रमाः।  
ये चादौ गोष्पदं पश्चात्रैलोक्यं क्राम्यतश्च ये॥१॥  
सरसाः सदलङ्काराः प्रसादमधुरा गिरः।  
कान्तास्तातजयन्तसय जयन्ति जगतीगुरोः॥२॥**

**——————————————————————————————————————————**

१.अयमभिनन्दमहाकविः कश्मीरेषु ख्रिस्ताब्दीयाष्टशतकोत्तरभागे समुत्पन्न इति प्रतीयते, यतोऽस पञ्चमः पूर्वपुरुषः शक्तिस्वामी कर्कोटवंशोद्भवस्य मुक्तापीडनरपतेर्मन्त्रिप्रवर आसीदिव ग्रन्थारम्भे स्फुटमस्ति मुक्तापीड इति तुं सप्तमशतकोत्तरभागे कश्मीराज्पालयतो ललितादित्यस्य नामान्तरमस्तीति राजतरङ्गिप्यां (४।३९—४३) प्रसिद्धमेव. तस्माद्धाणभट्टानन्तरं किञ्चिन्न्यूनाधिकसार्धशतवर्षेतीतेष्वभिनन्दः समुत्पन्न इति सुव्यक्तमेव, दशमशतकसमाप्तिसमुद्धूतः श्रीमदाचार्याभिनवगुप्तोऽपि ध्वन्यालोकलोचने तृतीयोद्द्योते कादम्बरीकथासारस्य स्मरति स्म. एवं क्षेमेन्द्रः सुबृत्ततिलके तर तृतीयबिन्यासे “तस्यां निजभुजोद्योग'— (१।१६) इत्यादि पद्यम्‌ भोजदेवश्च सरस्वतीकण्डाभरणे पश्चमपरिच्छेदे ‘चद्धापीडं सा च जग्राह कण्ठे— (८।७९) इत्यादि पद्यं कादम्बरीकथासारस्थमुदाहृतवान्‌ ,सुवृत्ततिलके “अनुष्टुप्सततासक्ता साभिनन्दस्य नन्दिनी। विधाधरस्य वदने गुलिकेव प्रभावभूः॥' इत्यनुष्टुप्छन्दोनिर्माणेऽभिनन्दकविं क्षेमेन्द्रः स्तौति स्म, कादम्बरीकथासारमपहायान्यः कोऽपि ग्रन्थोस्य कवेर्नोपलभ्यते, रामचरितकाब्यस्य प्रणेता तु कश्चिदन्य' एवं शताननन्‍दसूनुरभिनन्दोऽस्ति. ग्रन्थोऽयं पण्डितापरनाम्नि काशीविद्यासुधानिधौ मासिकपत्रे १८६७ खि्रस्ताष्दे प्रथमद्वितीयभागयोमुद्रितोऽपि न सर्वसुलभो नातिशुद्धश्चेति पुनरेतस्य मुद्रणमस्माभिरूपकान्तम् ,तत्र काश्मीरकपुस्तकस्य प्रतिरूपकमेकं पुस्तकं प्रायः शुद्धमष्टत्रिंशत्पत्रात्मकं श्रीपण्डितसरयुप्रसादशर्मणां सकाशादभिगतं क-सञ्ज्ञकम्‌. अपरं चतुरशीतिपत्रात्मकं १८०९ मिते विक्रमाब्दे लिखितं नातिशुद्धं, पण्डितमुद्रितपुस्तकस सहोदरमिव राजगुरुपर्वणीकरनारायणभट्टानां सङ्ग्रहात्प्राप्तं ख-सञ्ज्ञकम्‌. एतत्पुस्तकद्वयाधारेणैतन्मुद्रणमारभ्यत इति शिवम्‌।, 'क्रामतः ख. २. उदभुत्तारगम्भीरो मध्याह्नपटहध्वनिः॥४४॥**

**बहिः प्रकल्प्यतामस्या[^40] सुखायावसथादिकम्[^41]।  
अयं चाभ्यन्तरं पत्ररथभूपः प्रवेश्यताम॥४५॥**

[^40]: # ". प्रकल्पतां"

[^41]: # " ‛सुखं ख’."

**इत्यादिश्य प्रतीहारीं सभामौज्झन्महीपतिः।  
विससर्ज च राजन्यचक्रं सेवार्थमागतम्‌॥४६॥**

**ततः कृताल्पव्यायामः स्नातः पूजितेदैवतः।  
हुताग्नि श्रुतविप्राशीर्भुक्त्वा श्य्यागृहं ययो॥४७॥**

**तत्र पर्यङ्कमारुह्य ध्यात्वैव[^42] किमपि क्षणम्‌।  
तमन्तः पुरविश्रान्तं शुकमानाययत्पुनः॥४८॥**

[^42]: # "ध्यात्वा वै ख."

**तं च कञ्चुकिनानीतमादौ पृष्ट्वा सुखासिकाम्‌[^43]।  
पप्रच्छ कौतुकाक्षिप्तस्तद्वृत्तान्तमनन्तरम्‌॥४९॥**

[^43]: # "'सुखादिकं ख"

**वैशम्पायन को नु त्वं क्व जातः क्वोषितश्चिरम्‌।  
कुतस्त्यमिदमीदृक्ते कलाशास्रादि कौशलम्‌॥५०॥**

**मन्ये जातिस्मरः कोऽपि भवान्भूमिमिमां[^44] गतः।  
नह्यस्त्यतिशयप्राप्तिरियती पक्षिजातिषु॥५१॥**

[^44]: # ". इहागतः’ क."

**केन वा भवतो नाम कृतं विप्रजनोचितम्‌।  
ईदृशान्यभिधानानि कुतोऽन्त्यजनवेश्मसु[^45]॥५२॥**

[^45]: # "अन्यजन"

**कथं चण्डालसम्पर्कं[^46] इह वा कथमागमः।  
इति कार्त्स्न्येन वृ्त्तान्तमात्मनो वक्तुमर्हसि॥५३॥**

[^46]: # " ‛चाण्डाल’— ख."

**इति पृष्टः क्षितीशेन बहुमानपुरःसरम्‌।  
क्षणं बुद्ध्यानुसन्धाय शुको वक्तुं प्रचक्रमे॥५९॥**

**देव विन्ध्याटवी तावत्तव श्रुतिपथं गता।  
पूर्वापरपयोराशिवेलासंस्पर्शशालिनी॥५५॥**

**यदेकदेशे कण्डूलगण्डेभदलितद्रुमे।  
उषितौ दण्डकारण्ये ससीतौ रामलक्ष्मणौ॥५६॥(युग्मम्‌)**

**यत्रागस्त्यमुनेरासीदाश्रमः शमिनां मतः।  
लोपामुद्राकराक्षिप्तजलसंवर्धितद्रुमः॥५७॥**

**तस्योपकण्ठे पम्पाख्यमस्ति पुण्यजलं सरः।  
सीताकरतलोल्लूनश्रवणाभरणोत्पलम्॥५८॥**

**तस्यास्ति पश्चिमे तीरे जीर्णः शाल्मलिपादपः।  
आरोहपरिणाहाभ्यां व्याप्तव्योमदिगन्तरः॥५९॥**

**तस्मिन्कृतकुलायानि वीतभीतीनि दुर्गमे।  
विहङ्गमसहस्राणि निवसन्ति निरन्तरम्‌॥६०॥**

**दिवा तान्यभिनिर्वर्त्य[^47] प्राणयात्रामितस्ततः।  
तत्रागत्य[^48] स्वनीडेषु सुखं रात्रिषु शेरते॥६१॥**

[^47]: # " ‘अभितो गत्वा’ ख. "

[^48]: # " ‘कृत्वागत्य ख. "

**तत्रैव बद्धबसती अभूतां पितरौ मम।  
जातोऽहमेकस्तनयस्तयोर्वयसि[^49] पश्चिमे॥६२॥**

[^49]: # " 'एकतनयोतयोः ख"

**उद्वेढुमसमर्था सा तीव्रां प्रसववेदनाम्‌।  
तयैव पञ्चतां नीता जननी मे दुरात्मनः॥६३॥**

**जायावियोगदग्धोऽपि[^50] तातः[^51] पुत्रैषणातुरः।  
शोकमुत्सृज्य मत्प्राणपोषणे यत्नमग्रहीत्॥६४॥**

[^50]: # " ‘विरह’ ख. "

[^51]: # " ‘पिता’ ख. "

**न शशाक कचिद्गन्तुमात्मनः प्राणवृत्तये।  
मां वक्षसि निधायास्त नीड एव दिवानिशम्‌॥६५॥**

**अन्यपत्ररथानीतभुक्तोज्झितफलादिना।[^52]  
मदाप्यायनशिष्टेन सोऽकरोद्वृत्तिमात्मनः॥६६॥**

[^52]: # " ‘फलाम्बुना' क"

**अतिक्रामति कालेऽथ किञ्चिदुद्भिन्नपक्षतौ।  
तातवक्षःस्थलस्थेऽपि मय्युड्डयनलालसे॥६७॥**

**अकस्मादेव सम्भ्रान्तमृगपक्षिकुलश्रुतः।  
अभूत्कलकलस्तसिन्नरण्ये यमदुन्दुभिः॥६८॥(युग्मम्‌)**

**ततस्तातोरसः किञ्चिदुन्नमय्य शिरोधराम्‌।  
कुतूहलवशाच्चक्षुर्दिक्षु निक्षिप्तवानहम्[^53]॥६९॥**

[^53]: # "‘विक्षिप्तवान्’ ख"

**अथापश्यमसङ्ख्यातकिरातपरिवारितम्‌[^54]।  
वने शबरसेनान्यमायान्तं मृगयारसात्‌॥७०॥**

[^54]: # " ‘परिवारिते’ ख. "

**ततो मृतमिवात्मानमुत्खातमिव शाल्मलिम्‌[^55]।  
परिवृत्तमिवारण्यं भयेनाहमचिन्तयम्‌[^56]॥७१॥**

[^55]: # " 'शल्मलिं क"

[^56]: # ".‘अकल्पयम' ख."

**हतप्राणिसह[^57]स्राङ्गस्रवद्रुधिरचर्चितैः।  
मुहूर्तादेव तद्व्याधैर[^58]थारण्यमपूर्यत॥७२॥**

[^57]: # " ‘सहस्रौधैः’ ख."

[^58]: # " 'तदरण्यः’ ख "

**सेनापतिः स[^59] निस्त्रिंशो मृगयाजनितश्रमः।  
निषसाद ततः[^60] पम्पासरसस्तीरशाद्वले[^61]॥७३॥**

[^59]: # " ‘तु’ क. "

[^60]: # " 'मुहुः’ क"

[^61]: # " 'शाद्वले’ ख. "

**क्षणं विश्रम्य पीत्वापः प्राश्य बालमृणालिकाः।  
पुलिन्दैः सह सेनानीः स जगाम यथागतम्‌॥७४॥**

**एको जरत्पुलिन्दस्तु पापस्तत्र व्यलम्बत[^62]।  
प्रांशुर्भयङ्कराकारः कृतान्तस्येव किङ्करः॥७५॥**

[^62]: # " ‘विलम्बितः’ ख"

**अजातपिशितप्राप्तिरनुभूतवृथाश्रमः।  
यमदूतीमिव दृशं स ददौ तत्र शाल्मलौ॥७६॥**

**स दुरारोहतां तस्य जानन्नपि महातरोः।  
जातानध्यवसायोऽपि यत्नमारोहणेऽकरोत्‌॥७७॥**

**अथाहम[^63]समत्रासविपर्यासितमानसः।  
अविशं पितुरुत्सङ्गं तत्पक्षा[^64]वृतविग्रहः॥७८॥**

[^63]: # " ‘ततोऽहं ख. "

[^64]: # "'पक्षावित—’ख"

**स तु पापस्तमारुह्य तरुं नीडनिवासिनः।  
हत्वाहत्वा यथाशक्ति शुकान्भूमावपातयत्‌॥७९॥**

**क्रमेण स दुराचारस्तातस्यापि शिरोधराम्‌।  
बभञ्च तं च[^65] चिक्षेप क्षितो विगतजीवितम्‌॥८०॥**

[^65]: # " ‘तु’ ख."

**तदा मम न तच्छोको नान्यर्किचिञ्च[^66] चेतसि।  
केवलं सहजातेन भयेनैवास्मि विप्लुतः॥८१॥**

[^66]: # "‘किञ्चन’ ख."

**ततोऽहं पितुरुत्सङ्गाद्विनिर्गत्य ससाध्वसम्‌।  
अन्तरं तरुशीर्णानां पर्णानां तूर्णमाविशम्‌॥८२॥**

**पापः सोऽप्यवरुह्याथ शाल्मलेर्मूलवर्तिनः।  
शुकान्गतासूनादाय व्याघाननुसरन्नगात्‌॥८३॥**

**अहं विधिनियोगात्तु तेन दुष्कृतकारिणा[^67]।  
जीर्णपर्णसवर्णत्वात्तदानीं नावधारितः॥८४॥**

[^67]: # " ‘कर्मणाः’ ख,"

**ततो दूरमतिक्रान्ते तस्मिन्दारुणकर्मणि।  
सञ्जातजीविताशं मां पिपासा पर्यबाधत॥८८॥**

**अनुड्डयनसामर्थ्यात्पभ्द्यामेव यथातथा।  
पम्पापुलिनपर्यन्तं गन्तुमैच्छं पिपासितः॥८६॥**

**तथाप्यति[^68]निरुत्साहः पतन्मुहुरितस्ततः[^69]।  
तमेव पापकर्माणं शङ्कमानः पदे पदे॥८७॥**

[^68]: # "‘तथा पथिः’ ख"

[^69]: # " मुकलितेक्षणः ख."

**चिरं हा तात तातेति शोचन्कण्ठान्तवर्तिभिः[^70]।  
प्राणैरायास्यमानोऽहमतिष्ठमतिनिष्ठरः॥८८॥(युग्मम्‌)**

[^70]: # "‘कण्ठाग्र’—ख. "

**तस्मादनतिदुरेऽस्ति [^71]सरसस्तपसां निधेः।  
निर्धुतक्लेशजालस्य जावालेराश्रमो मुनेः॥८९॥**

[^71]: # " ‘अथः’ ख. "

**ततस्त[^72]स्मिन्सुतस्तस्य सरसि स्नातुमागतः।  
हारीतनामा मां तत्र तदवस्थं व्यलोकयत्‌॥९०॥**

[^72]: # "‘ततस्तस्य सुतस्तस्मिन्’ ख"

**कष्टं केनापि पापेन पातितः शुकशावकः।  
कुलायात्क्लाम्यतीत्युक्त्वा[^73] कृपया मां करेऽग्रहीत्‌॥९१॥**

[^73]: # "‘एवं’ ख."

**अपाययत्पयोबिन्दून्मरुताश्वासयन्मुहः।  
कृताह्निकश्च कारुण्यादनयत्पितुराश्रमम्‌॥९२॥**

**समाश्वस्तस्य हारीततातं दृष्ट्यैव पश्यतः।  
पदं स्थिरमिव न्यस्तं हृदि मे जीविताशया॥९३॥**

**नीडच्युतोऽयमानीतो मया विहगपोतकः।  
इति पित्रे प्रणम्यासौ[^74] पाणिस्थं माभदर्शयत्‌[^75]॥९४॥**

[^74]: # "‘आदौ’ क"

[^75]: # "‘अदीदृश्यत्' क."

**अथ जाबालिमद्राक्षं साक्षादिव पितामहम्‌।  
महात्मभिः परिवृतं सानुकम्पेैस्तपोधनैः॥९५॥**

**जराधवलमूर्धानं धाम्ना प्रसरता वहिः।  
अनतिस्फुटसंलक्ष्यशरीरावयवाकृतिम्‌॥९६॥ (युग्मम्‌ )**

**स तु मां क्षणमुद्वीक्ष्य स्मयमान इवावदत्‌।  
इदं हि भुज्यतेऽनेन फलं स्वयैव कर्मणः॥९७॥**

**त्रिकालदर्शी भगवान्स हि दिव्येन चक्षुषा।  
विश्वं पश्यति पाणिस्थ[^76]बदरामलकादिवन्‌॥९८॥**

[^76]: # "‘पाणिस्थं करस्थामलं यथा’ ख."

**वचस्तदाकर्ण्य कुतूहलाकुलाः समेत्य सर्वे मुनयस्तमब्रुवन्।  
वभूव कोऽयं भगवन्भवान्तरे किमस्य[^77] वा दुश्चरितं तदुच्यताम्‌॥९९॥**

[^77]: # " ‘किमद्य यत्‌’ क."

**अभ्यर्थितो[^78] मुनिभिरित्युपरुध्यमान-**  
**स्तानब्रवीत्कुरुत तावदहः क्रियाः स्वाः।**  
**सान्ध्यं विधाय विधिमस्खलितावधान-**  
**माकणेयिष्यथ कथां पृथुमेतदीयाम्‌॥१००॥**

[^78]: # " ‘अभ्यर्चितो मुनिभिरित्यनुबध्यमानः’ ख."

**उक्तास्तथा[^79] भगवता मुनयो निजेन**  
**कर्मक्रमेण दिवसं गमयाम्बभूवुः।**  
**हारीतहस्तविहितैश्च[^80] मनोविनोदै-**  
**स्तैस्तैरनीयत मयापि दिनावशेषः॥१०१॥**

[^79]: # " ‘च ते’ क"

[^80]: # " ‘निहतेन’ ख. "

**इति श्रीविपश्चिद्वराग्रगण्याचार्यभट्टश्रीजयन्तसूनोरभिनन्दस्य[^81] कृतौ कादम्बरीकथासारे प्रथमः सर्गः।**  
**—————————————————————————**

[^81]: # " ‘इति श्रीनिपश्चिद्वराग्रगण्याचार्य’ इति ख-पुस्तके नास्ति"



**द्वितीयः सर्गः।**  
**—————————**

**अथ सन्ध्यासमाध्यन्ताः[^82] समाप्य मुनयः क्रियाः।  
शुश्रूषवः समागम्य यथा[^83]स्थानमुपाविशन्‌॥१॥**

[^82]: # " ‘सान्ध्याः’ख, "

[^83]: # "‘यथास्थलम्’ख."

**अहमप्यात्मवृत्तान्तविस्तरश्रवणोत्सुकः।  
संहृत्य चापलं सर्वमासं स्थिरमनास्तदा॥२॥**

**अथ दुग्धाब्धिकल्लोलधवलैरमृातञ्चितैः।  
सिञ्चन्निव करैरिन्दुरुदगादाश्रमस्थलीम्‌[^84]॥३॥**

[^84]: # "‘स्थलीः’ ख. "

**भगवानथ[^85] जाबालिः सुखासीनः स्वविष्टरे।  
वल्कलाहितपर्यङ्कवन्धो वक्तुं प्रचक्रमे॥४॥**

[^85]: # "‘अपि’ क. "

**सर्वे दत्तावधानाः स्थ श्रृणुतेमां तपोधनाः।  
अमुष्य पूर्ववृत्तान्तप्रस्तावोपनतां[^86] कथाम्‌॥५॥**

[^86]: # "‘उपचितां’ ख,"

**अस्त्यवन्तिषु संसारसलक्ष्म्याः प्रसवभूरिव।  
निधिरुज्जयिनी नाम नगरी गुणसम्पदाम्‌॥६॥**

**लङ्कारकामरावत्यो येषां वसतयः प्रियाः।  
तेऽपि यामभिपश्यन्तः सुखं तासु न शेरते॥७॥**

**किमन्यद्यत्र[^87] कैलासनिवासप्रीतिमुज्झता।  
महाकालाभिधानेन रुद्रेण खयमास्यते॥८॥(तिलकम्‌[^88])**

[^87]: # ". ‘यस्यां किमन्यत्कैलासनिवासरसमुज्झता’ ख. "

[^88]: # "ख-पुस्तके नास्ति. "

**तस्यां [^89]भरतमाङ्घातृभगीरथपृथूपमः।  
तारापीड इति श्रीमान्बभूव[^90] पृथिवीपतिः॥९॥**

[^89]: # " ‘यस्यां’ ख."

[^90]: # " ‘उवास’ क."

**परस्परविरोधेन विप्रकीर्णैरितस्ततः।  
विस्त्रब्धमास्यते यत्र सम्भूय सकलैर्गुणैः[^91]॥१०॥ (युग्मम्‌)**

[^91]: # "‘कुशलैः’ ख"

**विहसन्बुद्धिविभवैर्भृगूद्वहबृहस्पती[^92]।  
अमात्य शुकनासाख्यो ब्राह्मणस्तस्य चाभवत्‌॥११॥**

[^92]: # " ‘बृहस्पतीन्’ ख."

**त्रिविक्रमपदाक्रम्य स राजा पृथिवीमिमाम्‌।  
तस्मिन्मन्त्रिणि तद्भारं निचिक्षेप मरक्षमे॥१२॥**

**तथा च[^93] पालयामास स मन्त्री सकलाः प्रजाः।  
यथा त्रिविधमप्येता न दुःखं किञ्चिदस्पृशत्‌[^94]॥१३॥**

[^93]: # ".‘एव’ क."

[^94]: # " ‘अस्पृशन्’ ख."

**यथा यथा च लोकोऽस्मिन्पितरीवान्वरज्यत[^95]।  
राजा तथा तथात्मानं कृतकृत्यममन्यत॥१४॥**

[^95]: # "‘अनुरञ्ज्यते’ ख."

**अक्लेशमर्पितं[^96] तेन स्वभुजे जगद[^97]र्जनम्‌।  
सम्भोगविघ्नकारीति पालनं मन्त्रिसात्कृतम्॥१५॥**

[^96]: #

[^97]: # " "

**कृतकत्यतया चासौ कार्यशेषमचिन्तयन्।  
विषयेष्वाहितप्रीतिरासाञ्चक्रे यथासुखम्‌॥१६॥**

**शोभा हि कृतकृत्यस्य राज्ञो भोगविभूतयः।  
असमाप्तजिगीपस्य ता एव तु विडम्बनाः॥१७॥**

**स युवा मधुराकारस्तरुणीनयनोत्सवः।  
कलासु[^98] कुशलः कामी बुभुजे भोगसम्पदः॥१८॥**

[^98]: # "यथाभिलषिता"

**न ते भोगा[^99] न ताः क्रीडा न ता गोष्ट्यो न ते रसाः।  
न ते विलासा येष्वस्य [^100]पारगं नाभवन्मनः॥१९॥**

[^99]: # ". भावाः ख."

[^100]: # ". "

**समग्रसुखसम्भोगभूमिज्ञोऽपि स[^101] भूपतिः।  
सुखमेकं तु नाज्ञासीदपत्यालोकनोद्भवम्‌॥२०॥**

[^101]: # "“महीपतिः क"

**यथा यथास्य कालेन व्यतीयाय नवं वयः।  
निर[^102]पत्यतया शोको वर्धते स्म तथा तथा॥२१॥**

[^102]: # " “अनपत्यतया' ख"

**मुख्यभूता च सर्वस्मिन्नन्तःपुरकदम्बके।  
राज्ञी विलासवत्याख्या तस्य[^103] प्राणसमाभवत्‌॥२२॥**

[^103]: #

**सा राजमहिषी गौरीमभ्यर्चयितुमेकदा।  
देवतायतनं बाह्यं ययौ भर्तुरनुज्ञया॥२३॥**

**पुराणे वाच्यमानेऽथ[^104] प्रसङ्गपतितां श्रुतिम्‌।  
अपुत्रस्य कुतो लोका इति शुश्राव तत्र सा॥२४॥**

[^104]: # " 'तु प्रसङ्गात् ख,"

**प्रतप्तलोहसूचीव सा श्रुतिः श्रवणान्तरम्‌।  
विवेश ददती[^105] तस्याः परां कामपि वेदनाम्‌॥२५॥**

[^105]: # ".दधतीः क."

**ततः प्रतिनिवृत्ताथ प्रविश्य भवनं निजम्‌।  
वस्रावृत्तमुखी भूमावुपविश्य रुरोद सा॥२६॥**

**सखीभिः प्रार्थ्यमानापि न स्वामभजत स्थितिम्‌।  
नोद[^106]स्थात्कर्तुमाहारं विरराम न रोदनात्‌॥२७॥**

[^106]: # " “नो दुःखातः ख"

**अथोपलभ्य तत्पीडा[^107]वार्तां परिजनान्नृपः।  
प्रविश्यान्तःपुरं प्राह स्वैरमेनां प्रसादयन्‌॥२८॥**

[^107]: # "तच्चित्तपीडा क"

**प्रिये किमिदमारब्धं रुद्यते किमकारणम।  
न त्वया मत्कृते[^108] मन्युरनुभूतः कदाचन॥२९॥**

[^108]: # " 'मत्कृतो” ख"

**अन्योऽपि भन्मुखप्रेक्षी[^109] मदधीनात्मजीवितः।  
कस्तवोत्पादयेन्मन्युं यस्मै कुप्यसि सुन्दरि॥३०॥**

[^109]: # " “मत्सुखापेक्षी” ख,"

**इति प्रसाद्यमाना सा यदा नोवाच किञ्चन।  
तदा विदितवृत्तान्तस्तत्सख्या नृपतिः[^110] कृतः॥३१॥**

[^110]: # " 'भूपतिः” क"

**तेन व्यतिकरेणाथ सोऽपि शोकपरायण[^111]:।  
रुरोद वक्षसि न्यस्तसबाप्पदयितामुखः॥३२॥**

[^111]: # "'दुःख–ख. "

**चिरं रुदित्वा निःशब्दं स्वयमुन्सृज्य लोचने।  
जगाद गद्गदं राजा महिषीमुपसान्त्वयन्‌॥३३॥**

**ममापि नन्वियं देवि चिरात्प्रभृति दुःसहा।  
दुनोति हृदयं चिन्ता[^112] निष्प्रतीकाररोगवत्‌॥३४॥**

[^112]: # " पीडाः ख,"

**विफलं बत मे राज्यं विफला मम सम्पदः[^113]।  
विफला मम सम्भोगा विफलं मम जीवितम्‌॥३५॥**

[^113]: # "देवताः ख,"

**कुतोऽस्माकमभव्या[^114]नामियत्यो भाग्यसम्पदः।  
यद्द्रक्ष्यामस्तवोत्सङ्ग बालकापत्यमण्डितम्‌॥३६॥**

[^114]: # "'अभाग्यानां"

**वाञ्छन्क्रीडनकं किञ्चित्कदा[^115] मे पापकर्मणः।  
आकृष्यति सुतः कूर्चं किमप्यव्यक्तमालपन्‌॥३७॥**

[^115]: # "ख-पुस्तके द्वितीयचतुर्थचरणयोर्व्यत्ययः"

**इत्यादि विलपत्यस्मिन्बाष्पाकुलितलोचने।  
शुकनास इमां वार्तां[^116] श्रुत्वा सत्वरमाययौ॥३८॥**

[^116]: # ". 'ज्ञात्वा वार्ता"

**निरपत्यः कृतः सोऽपि राजेव हतवेधसा।  
तस्याप्यनेन शोकेन गृहे रोदिति गेहिनी॥३९॥**

**स तु स्वामिगतं दुःखं दुःखमित्यवधारयन्‌।  
तद्दुःखापनयोपायप्रवणो धीरमभ्यधात्‌[^117]॥४०॥**

[^117]: #

**क्व धीरप्रकृतिर्देवः क्वेयं विप्लवचित्तता।  
शोकोऽवकारां लभते हृदये न भवादृशाम्‌॥४१॥**

**यदि[^118] हि स्त्रीस्वभावेन देवी विक्लवतां गता।  
तत्किं प्रलम्बकूर्चस्य युक्तं देवस्य रोदितुम्‌॥५२॥**

[^118]: # ". 'यदीह” ख"

**कोऽयं हिताहितप्राप्तिपारिहारक्रियाक्रमः।  
यत्कपोलतलन्यस्तहस्तं निःश्वस्यते चिरम्‌॥४३॥**

**यदि दुःखासिका[^119] सत्यमनपत्यतया तव।  
बुद्धिराधीयतां धर्मे धर्मात्सर्वमवाप्यते॥४४॥**

[^119]: # " दुःखस्थितिः"

**दीयतां वस्तु विप्रेभ्यः पूज्यन्तां सर्वदेवताः।  
प्रवर्त्यन्तां प्रपारामाः स्वन्यन्तां सलिलाशयाः॥४५॥**

**नास्तिनाम[^97] जगत्यस्मिन्पुरुषार्थः सं कश्चन।  
यो धर्मनिरतैर्धीरै[^121]स्त्वादृशैर्नाधिगम्यते॥४६॥**

[^121]: # "वीरैः क"

**इत्यादि शुकनासोक्तमाकर्ण्य विगतज्वरौ।  
उत्थाय स्वस्थवत्सर्वा दपन्ती चक्रतुः क्रियाः॥४७॥**

**ततः प्रभृति दानानि यागहोमतपांसि तौ[^122]।  
चित्राण्य[^123]कुरुतां सर्बा व्यस्मयन्त ततः[^124] प्रजाः॥४८॥**

[^122]: # " च क,"

[^123]: # "“चक्रतुस्तु विचित्राणिः कः"

[^124]: #

**कदाचिदथ भूपालः स्वप्नेऽपश्यन्निशाकरम्‌।  
विलासवत्या वदनं प्रविशन्तमखण्डितम्‌॥४९॥**

**तथा मनोरमा नाम शुकनासस्य गेहिनी।  
स्वप्ने सपुण्डरीकाङ्का सापि तेन व्यकोक्यत॥५०॥**

**अन्येद्युरुदये भानोर्मङ्गलाचारपूर्वकम्‌।  
तं स्वप्नं कथया[^125]मास शुकनासाय भूपतिः॥५१॥**

[^125]: # ". 'वर्णयामासः क."

**सानन्दं सोऽभ्यधादेनं सम्पन्ना[^97]स्ते मनोरथाः।  
अचिरात्सोष्यते देवी पुत्रं पूर्णेन्दुसुन्दरम्‌॥५२॥**

**दिनेष्वथ व्यतीतेषु केषुचिद्गर्भमग्रहीत्‌।  
महिषी तत्सखीभ्यश्च तमज्ञासीन्नरेश्वरः॥५२॥**

**ताभ्यः प्रादात्स वासांसि चित्राण्याभरणानि च।  
क्रमेण गर्भलिङ्गेषु व्यक्तेषु मुमुदेतराम्‌॥५४॥**

**शुकनासाय चोवाच प्रहर्षोत्फुल्ललोचनः।  
चक्राङ्क्ष चातको मेघानिव प्रसववासरान्‌॥५५॥**

**सम्पूर्णे[^127] दशमे मासि देवी सुतमसूत सा।  
हर्षेण च जगत्सर्वं परिवृत्तमिवाभवत्‌॥५६॥**

[^127]: # "“प्राप्तेऽथ दशमे मासे"

**राजा महोत्सवानन्दपूर्णपात्रमुपाहृतम्‌[^128]।  
दृष्ट्वा लोकेषु माति स्म त्रिविक्र[^129]मवत्रिषु॥५७॥**

[^128]: # " “उदाहृतम्‌ ख. "

[^129]: # ". 'त्रिधिक्रमवरर्तिषु क-ख. "

**शुकनासस्तु राजानं कण्टे जग्राह निर्भरम्[^130]।  
ओत्सुक्यात्सूतिकावासं ताबुभावषि जग्मतुः॥५८॥**

[^130]: #

**ततो ददृशतुर्व्यक्त[^131]सकलाङ्गं कुमारकम्‌।  
इन्दोरुदयरागार्द्रं नवं बिम्बमिवोदितम्‌॥५९॥**

**शुकनासो निरूप्याङ्गमेकैकं निपुणं शिशोः।  
हस्ते भूपालमालम्ब्य हर्षगर्भमभाषत॥६०॥**

**अयं शङ्ख इदं चक्रमियमूर्णेदमम्बुजम्‌।  
पश्य [^132]वालस्य चिह्नानि दृश्यन्ते चक्रवर्तिनाम्‌॥६१॥**

[^132]: #

**योऽसौ स्वप्ने त्वया दृष्टश्चन्द्रो देवीमुखं[^133] विशन्।  
स एवायमिहोत्पन्न इति मा संशयं कृथाः॥६२॥**

**इति तद्दर्शनानन्दमुधाप्लावितयोस्तयोः।  
शुकनासस्य[^134] राज्ञश्च हर्षाब्धिर्ववृधेतराम्‌॥६३॥**

[^134]: # " शुकनासगृहावृद्धा तत्पुत्रोत्पत्तिमभ्यघात्"

**अत्रान्तरे[^135] मन्त्रिगृहादागत्य द्वाररक्षकः।  
शुकनासगृहे पुत्रजन्मोत्सवमचीकथत्‌॥६४॥**

[^135]: # "अयं श्लोकः क-पुस्तके नास्ति"

**शुकनासमथा[^136]भीक्ष्णमालिलिङ्ग महीपतिः।  
उत्तरीयं जहारास्य पूर्णपात्रमिति स्वयम्‌॥६५॥**

[^136]: # "अथोदारं ख, "

**उवाच चैनं पश्येदं सुखस्योपरि नः सुखम्‌।  
इमे ते पतिताः सत्यं तुहिने सोमरश्मयः॥६६॥**

**साध्विदं वेधसारब्ध॑ चित्ता[^137]नुगुणमावयोः।  
सहैव पात्रतां नीतौ यदावां सुखदुःखयोः॥६७॥**

[^137]: # " “चिन्ता—* ख"

**ततस्तूर्यरवैर्नृत्यैश्चर्चरीतालनिःस्वनैः।  
सिन्दूरघनसारादिक्षोदैः कुङ्कुमकर्दमैः॥६८॥**

**अनपेक्षितनीचोच्चव्यवहारनिरर्गला।  
केलिः प्रववृते तत्र राजगेहमहोत्सवे॥६९॥ (युग्मम्‌)**

**राजा तु स्वगृहे केलिविमर्दं वीक्ष्य तादृशम्‌।  
शुकनासगृहं गत्वा द्विगुणं तमकारयत्‌॥७०॥**

**कृतेऽथ जातकर्मादौ[^138] वृत्ते षष्ठीप्रजागरे।  
विलासवत्यां स्नातायां शुचौ काल डपस्थिते॥७१॥**

[^138]: #

**प्रशस्तेऽहनि देवज्ञैरनुज्ञातो महीपतिः।  
स्नातः सुवर्णलक्षेण[^139] पूजयित्वा द्विजोत्तमान्‌॥७२॥**

**स भूषणानि वासांसि वर्षन्परिजने शिशोः।  
चन्द्रापीड इति स्वप्नसदृशं नाम निर्ममे॥७३॥ ( तिलकम् )**

**शुकनासोऽपि सम्पूज्य विप्रान्विप्रजनोचितम्‌।  
वैशम्पायन इत्यात्मसूनोर्नाम प्रणीतवान्[^140]॥७४॥**

[^140]: # "प्रशासिवान्‌ः ख."

**क्रमेण वर्धमानोऽथ कृतचूडादिमङ्गलः।  
चन्द्रापीडोऽभवत्पित्रोः परमुत्सङ्गमण्डनम्‌॥७५॥ **

**आकारेणेन्दुका[^141]न्तेन मधुरैर्बालवल्गितैः।  
अव्यक्तवर्णैरालापैर[^142]हरत्स तयोर्मनः॥७६॥**

[^141]: # " 'इन्दुतुल्येन' ख, "

[^142]: #

**प्राप्तेऽथ सप्तमे वर्षे विद्यागृहमकारयत्‌।  
चन्द्रापीडस्य नगराद्बहिर्बहुगुणं नृपः॥७७॥**

**कृतोपनयनं चैनं काले क्षत्रजनोचिते।  
तत्र न्यवेशयद्राजा विद्याधिगमसिद्ध्ये॥७८॥**

**तर्कव्याकरणाभिज्ञान्धनुर्वेद्विशारदान्‌।  
दण्डनीत्यादिनिपुणानाचार्यान्पर्यकल्पयत्‌॥७९॥**

**शुकनासोऽपि तं पृत्रमुपनीय यथाविधि।  
राजपुत्रानुगं चक्रे विद्यागृहनिवासिनस्‌॥८०॥**

**नृपतिस्त्वगमद्द्रष्टुं पुत्रं गुरुकुले[^143] स्थितम्‌।  
प्रार्थितोऽपि[^144] स तन्मात्रा नैनमानाययद्धहम्‌॥८१॥**

[^143]: # "गुरुगृहे ख."

[^144]: # " स प्रार्थितोऽपि' क"

**पूर्णषोडशवर्षोऽथ[^145] राजपुत्रः कुशाग्रधीः।  
सकृदध्यापितो विद्याः पूर्वाभ्यस्ता इवाग्रहीत्‌॥८२॥ .**

[^145]: # "'तु” ख. "

**सर्वशास्त्रार्थतत्त्वज्ञः सर्वेभाषासु सत्कवीः।  
सर्ववादेषु[^146] वक्ताभूत्सर्वशास्त्रेषु कोविदः॥८३॥**

[^146]: #

**प्रोढः कलासु सर्वासु दक्षः सर्वेषु कर्मसु।  
सर्वेषु[^147] व्यवहारेषु सोऽभूदस्खलितक्रमः॥८७॥**

**क्रमेण वर्धमानश्च स व्यायाममसेबत।  
शुशुभे च[^148] तदभ्यासप्राप्तसर्वाङ्गसौष्ठवः॥८५॥**

[^148]: # ". स क."

**प्रलम्बबाहुः पीनांसः पृथुवक्षास्तनूदरः।  
गम्भीरबाहुर्वृत्तोरुस्तरुणः करिसद्गतिः॥८६॥**

**शुकनाससुतस्तस्य प्रतिबिम्ब[^149] इवाभवत्‌।  
गुणैरशेषैरन्यत्र शरीरप्राणगोरवात्‌॥८७॥**

[^149]: # "प्रतिबिम्ब"

**राजा तु तं सुतं बुद्धा कृतार्थं गुरुवेश्मनः।  
शुकनासगिरा गेहमानेतुमुपचक्रमे॥८८॥**

**स बलाहकनामानमाहूय ध्वजिनीपतिम्‌।  
पुण्येह्नि ग्राहिणोत्पुत्रमानेतुं कृतमङ्गलः॥८९॥**

**सहस्त्रं राजपुत्राणां विनीतं बाहुशालिनाम्‌।  
विससर्ज[^150] च तेनैव सह पुत्रा[^151]नुवर्तिनम्‌॥९०॥**

जितोच्चैः श्रवसं तैस्तैरनन्यसदृ[^152]शैर्गुणैः॥९१॥**

[^152]: # "सुलभैः'ख. "

अद्राक्षीद्राजपुत्र तं प्रणनाम च सादरम्‌॥९२॥**

[^153]: # " “कृतास्पदम्‌ ख. "

**ततो बलाहकाख्यातशौर्यादिगुणगौरवाः।  
राजपुत्राः प्रणेमुस्तं पुरं[^97]दरमिवामराः॥९४॥**

**पर्याणरत्नज्योतिर्भिजनितेन्द्रायुधप्रभम्‌।  
इन्द्रायुधं प्रवेश्याश्वं[^156] पुनराह बलाहकः॥९५॥**

[^156]: # "अथ' ख"

**इममारोह सिन्धूस्थमश्वरत्नमयोनिजम्‌।  
पारसीकाधिपतिना देवस्योपायनीकृतम्‌॥९६॥**

**चन्द्रापीडस्तु तं दृष्ट्वा[^157] तुरङ्गममनुकत्तमम्‌।

[^160]: #

रम्भा किलाप्सरःकन्याशापेन वडवाभवत्‌॥९९॥**

**गरुत्मतीदमुच्च[^162]त्वमुच्चैःश्रवसि वा भवेत्‌।  
तेजसामाकृते[^163]र्वापि न सामान्येषु वाजिषु॥१००॥**

[^162]: # " 'उच्चैस्त्वं ख."

[^165]: # " 'मङ्गल ख, "

**सवैशम्पायनः सोऽथ निर्गत्य भवनात्ततः।

[^199]: # ". तथातथा ख."

**कामं भवान्प्रकृत्यैव धीरः पित्रा च संस्कृतः।  
त्वद्गुणेष्वति[^200]सन्तोषो मुखरीकुरुते स्म[^201] माम्‌॥३९॥**

[^200]: # "अपि' ख़."

[^201]: # " हि"

**युवराज धुरं पित्रा कल्पितामुद्वह सिराम्‌।  
विजितामपि चानेन पुनर्जय वसुन्धराम्॥४०॥**

**इत्युक्त्वा विररामासौ तत्कल्याणपरायणः।  
चन्द्रापीडस्तु तद्वाग्भिररुकृत इवाभवत्‌॥४१॥**

**अथ पुण्येऽहनि स्नातं विधिवत्कृतमङ्कलम्।  
यौवराज्ये प्रियं पुत्रमभ्यषिञ्चन्महीपतिः॥४२॥**

**अभिषेकोत्सवस्तस्य जन्मलाभोत्सवादपि।  
अधिकः सर्वकोलस्य बभूवानन्दवर्धनः॥४३॥**

**सर्वाभ्यो दिग्भ्य आजग्मुर्गहीतोपायना नृपाः।  
द्विजाश्च तुतुषुः प्राप्त[^202]गोभूकनकदक्षिणाः॥४४॥**

[^202]: # "“प्राप्तबहुसर्वस्व—"

**अन्य[^203]त्राहनि दैवज्ञश्लाघ्ययोग्यगुणान्विते।  
ददौ दिग्विजये यात्रां युवराजो नृपाज्ञया॥४५॥**

[^203]: # " “अन्याहनि च क"

**कृत[^204]प्रस्थानहोमोऽसो विधिज्ञेन पुरोधसा।  
शस्त्रास्त्रचक्रमभ्यर्च्य शास्त्रोक्तैर्मन्त्रविस्तरैः॥४६॥**

[^204]: # "“गीत— ख"

**कृतस्वस्त्ययनो विप्रैः प्रचुरस्वर्णतोषितैः।  
यत्नरुद्धा[^205]स्रया मात्रा कृतप्रस्थानमङ्गलः॥४७॥**

[^205]: # " रुद्धाश्रुको"

**शङ्खदुन्दुभिनिर्घोषमुखरीकृतदिङ्मुखः।  
स्तुतस्तारं परद्भिश्च सूतमागधबन्धिभिः॥४८॥**

**स्रग्विलेपनदीपादिसम्भारार्चितदैवतः।  
पितरौ शुकनासं च प्रणम्य निरगाद्गृहात्‌॥४९॥ (चक्कलकम्‌ )**

**इन्द्रायुधमथारुह्य पौरजानपदाशिषः।  
शृण्वन्नगर्याः पूर्वेण द्वारेण निरियाय सः॥५०॥**

**ततोऽनुगम्यमानोऽसौ बलाहकनिवेदितैः।  
सहस्रसङ्ख्यैः क्षितिपैरुल्ल[^206]सच्छत्रचामरै॥५१॥**

[^206]: # "अलस—क,"

**वैशम्पायनमेकत्र पार्श्वे श्वेताश्चवर्तिनम्‌।  
अन्यत्र पत्रलेखां च पश्यन्नश्वतरीस्थिताम्‌॥५२॥**

**तस्मिन्नहमि स स्तोकं प्रस्था[^207]योच्छ्रिततोरणम्‌।  
चित्रं निवासमध्यास्त सिप्रा[^208]कूले प्रक[^209]ल्पितम्‌॥५३॥ (तिलकम्‌)**

[^207]: # "“प्रस्थानोच्छ्रित—” क,"

[^208]: # " 'प्राक्‌ प्राक्कूले ख,"

[^209]: # "अत्र कल्पितम्‌ क"

**अथापरेद्युरुत्थायाशेषाभरणमभूषितः।  
प्रावर्तत प्रबन्धेन प्रस्थातुमतुलोद्यमः॥५४॥**

**पीडयन्नथ[^210] सैन्येन पातालतलवासिनः।  
रजसा च तदुत्थेन स्थगयन्गगनस्थितान्‌॥५५॥**

[^210]: # ". इव ` ख."

**नमयन्नुन्नतान्भूपानान[^97]तान्परिपालयन्‌।  
अवशा[^212]नुत्स्वनन्कांश्चिदुत्खातान्प्रतिरोपयन्॥५६॥**

[^212]: # " 'अवश्यान् ख."

**विश्वासयन्नविश्वस्तान्संरक्षञ्शरणागतान्।  
दस्यूनु[^213]त्सादयन्दण्डैः साधून्सम्मानयन्धनैः॥५७॥**

[^213]: # " “उत्सारमन्‌ कः,"

**ब्राह्मणेभ्यो ददद्भ्रामान्कुर्वन्देवकुलाबलीः।  
नवानि[^214] निजचिह्णानि तत्र[^215] तत्र प्रवर्तयन्‌॥५८॥**

[^214]: # "“वनानिः ख"

[^215]: # "“यत्र तत्र क. "

**वर्षत्रितयमात्रेण समग्रामर्णवावधिम्‌।  
स वभ्राम प्रबन्धेन मान्धातेव वसुन्धराम्‌॥५९॥ (कुलकम्‌)**

**जिगाय प्रथमं प्राचीं दिशं तदनुं दक्षिणाम्‌।  
ततः[^216] प्रतीचीं पश्चाच्च प्रतापनिधिरुत्तयाम्‌॥६०॥**

[^216]: #

**अथ हेमजटाख्यानां किरातानां निकेतनम्‌।  
स सुवर्णपुरं जित्वा[^217] दण्डं जग्राह हेलया॥६१॥**

**या[^218] च कैलाससीमान्तर्भूर्नितान्तममनोरमा।  
तस्यां दिग्विजयश्रान्तः कियतोऽप्यास्त[^219] वासरान्‌॥६२॥**

[^218]: # "याथ' क"

[^219]: # "अध्यास्त क"

**एकदा तु[^220] स तत्स्थानादारुह्येन्द्रायुधं वने।  
मृगयानिर्गतोऽपश्यत्किन्नर्या सह किन्नरम्॥६३॥**

[^220]: # "तत्स्थ एवासौ क."

**आलोक्य कौतुका[^221]कृष्टदृष्टिरश्वमुखौ च तौ।  
आदित्सुः प्रेरयामास तावेव प्रति वाजिनम्‌॥६४॥**

[^221]: # "कौतुकाबिष्ट -” क."

**अपूर्वपुरुषालोकत्रासात्कैलासकन्दरस।  
प्रवेष्टुकामौ महता[^222] वेगेनानुससार तौ॥६५॥**

[^222]: # " “सहसा ख,"

**पार्ष्णिप्रहरणाभ्यासग्रहितोद[^223]ग्ररंहसा।  
पञ्चयोजनमाना[^224] भूस्तेनाश्वेन व्यलङ्घयत॥६६॥**

[^223]: # " “उदार—’ क."

[^224]: # "मात्रा क"

**तौ च पर्वतमारूढावग्र[^225]तस्तस्य किन्नरौ।  
ततस्त[^226]द्ग्रहणाकाङ्क्षां हृदयादुत्ससर्ज सः॥६७॥**

[^225]: # "पश्यन्निःश्वस्य क. "

[^226]: # ". ग्रहणे काङ्क्षां ख"

**अशक्नोदनु[^227]सर्तुं च कश्चित्परिजनो न तम्‌।  
तार्क्ष्येणाप्यनुगम्येत धावन्निन्द्रायुधः कुतः॥६८॥**

[^227]: # " “उपसर्तुं क. "

**अथ वल्गां समाकृष्य श्रमस्वेदार्द्रविग्रहम्‌।  
दृष्ट्वैकाकिनमात्मानं श्वसन्तं च तुरङ्गमम्‌॥६९॥**

**निर्वर्णयंस्तमुद्देशं मानुषा[^228]णामगोचरम्‌।  
विहस्थैवात्मनात्मानं सानुतापमचिन्तयत्‌॥७०॥**

[^228]: # "“मनुष्याणां कथातिगम्‌ क,"

**अहो किमपि मे मोर्ख्यमस्थानाभिनिवेशिनः।  
यदात्मा बालकेनेव[^229] व्यर्थमायासितो मया॥७१॥**

[^229]: # " 'दारकेणेव का,"

**अगृहीतेन को[^230] वार्थः कोऽर्थः प्राप्तेन वामुना।[^231]  
किन्नरद्वितयेनेति मया मूढेन नेक्षितम्‌॥७२॥**

[^230]: # " कोऽनर्थः क"

[^231]: # "पुनः क,"

**पन्था निरूपितो नायमाविष्टेनेव धावता।  
न चात्र लभ्यते मर्त्यो मम मार्गस्य देशिकः[^232]॥७३॥**

[^232]: # "दैशिकः क"

**श्रूयते किल वर्षस्य भारतस्यावसानभूः।  
कैलाससन्निकृष्टेयमिह मर्त्यस्य का कथा॥७४॥**

**वने पर्णतृणाकीर्णे पादमुद्राश्च वाजिनः।  
न स्युर्यदन[^233]ुसारेण यास्यामि शिबिरं निजम्‌[^97]॥७५॥**

[^233]: # " शनैः ख"

**केवलं दक्षिणामाशां पुरस्कृत्य यथातथा।  
गन्तव्यं कि तु मध्याह्नसमयोऽयसुपस्थितः॥७६॥**

**अनुभूतोऽतिमात्रं[^235] च स्वेदस्तदिह साम्प्रतम्‌।  
लब्ध्वा कुतोऽप्यपस्ताभिर्यास्यामि विगतश्रम॥७७॥**

[^235]: # " अतिन्मात्रेणः ख,"

**इति[^236] सञ्चिन्त्य पानीयमन्वेषितुमितस्ततः।  
परिभ्रमन्घनस्निग्धं शाखिनां खण्ड[^237]मैक्षत ॥७८॥**

[^236]: # " इत्थं विचिन्त्य” ख"

[^237]: # " षण्डं क"

**ततस्तदनुकूलेन पथा गच्छन्समीरणेः।  
आह्लाद्यत नवोत्फुल्लकमलामोदवाहिभिः॥७९॥**

**मत्तद्विपपदा[^238]सक्तमृणालदलदन्तुरम्‌।  
स पन्थानमथापश्यदम्भसामनुमापकम्‌॥८०॥**

[^238]: # ", मदासिक्त क "

**तेन गत्वागखण्डस्य[^239] मध्ये जलधिसन्निभम्‌।  
अदृष्टापारपर्यन्तमालुलोके महत्सरः॥८१॥**

[^239]: # "षण्डस्य क. वृक्षसमूहस्येत्यर्थः,"

**तदालोकनमात्रेण दूरादपगतक्लमः[^240]।  
प्राप्य तत्पुलिनोपान्तं वाजिनोऽवततार सः॥८२॥**

[^240]: # ". श्रमः ख."

**कृत्वापनीतपर्याणं तं स्थलीलुठितोत्थितम।  
गृहीतशाद्वलग्राममाप्लु[^241]ताधूतकेसरम्‌॥८३॥**

[^241]: # "आप्लुतोद्भूत क."

**उत्तार्य सरसस्तीरतरौ संयम्य वाजिनम्‌।  
क्षिप्त्वासिधेनुलूनानि तृणान्येतस्य चाग्रतः॥८४॥**

**स्वयं सस्नौ पपौ वारि बुभुजे बिसकन्दलीः।  
उत्तरीयशिरोधानो निषसाद शिलातले॥८५॥ (तिलकम्‌)**

**ततो मुहूर्तविश्रान्त उत्तरस्मिन्सरस्तटे।  
समुच्चरन्तं शुश्राव गीतध्वनिममानुषम्॥८६॥**

**तस्यां दिशि दृढासक्तचक्षुरुत्क्षिप्तकन्धरः।  
तमादौ निबिडश्रोत्रशुक्तिरिन्द्रायुधोऽशृणोत्‌॥८७॥**

**कुतोऽत्र विजने गीतमित्याविष्कृतकौतुकः।  
उत्थाय चन्द्रापीडोऽथ तामेव दिशमभ्यगात्‌॥८८॥**

**गीतश्रवणलुब्धेन सलीलमभिधावता।  
अपृष्टेनापि[^242] मार्गोऽस्य मार्गे[^243]णैव न्यवेद्यत॥८९॥**

[^242]: # " 'अथ” ख"

[^243]: # " मृगसमूहेनेत्यर्थः"

**गत्वा प्रासादमैक्षिष्ट स्फाटिकं नातिदूरतः।  
हसन्तमिव कैलासशोभा[^244]विभवमंशुभि॥९०॥**

[^244]: # " लक्ष्मी” ख. "

**देवं चतुर्मुखं तत्र रत्नपीठप्रतिष्ठितम्‌।  
भगवन्तमुमाकान्तं ददर्श प्रणनाम च॥९१॥**

**तस्याथ दक्षिणां मूर्तिमाश्रित्य विहितासना।  
उत्सङ्गनिहितां वीणां वादयन्ती कलस्वनम्[^245]॥९२॥**

[^245]: # " “कलस्वना' क."

**देहप्रभावितानेन कुन्दगौरेण सर्पता।  
साक्षाल्लक्ष्मीरिव क्षीरजलधर्मध्यवर्तिनी॥९३॥**

**कोपाग्निशमितानङ्गप्रत्युज्जीवनवाञ्छया।  
रतिः प्रसादयन्तीव तपसा पार्वतीपतिम्[^246]॥९४॥**

[^246]: # "प्रियम्” ख."

**नवेऽपि वयसि प्राप्तवैराग्याभ्यासवासना।  
व्यलोकि कन्यका तेन धृतपाशुपतव्रता॥९५॥ \`**

**चन्द्रापीडोऽथ[^247] तां दृष्ट्वा विस्मयाविष्टमानसः।  
केयं स्यात्कन्यका नास्या दिव्यतां प्रति संशयः॥९६॥**

[^247]: # " तु ख."

**नान्तर्धानमियं गच्छेन्न वा गगनमुत्पतेत्‌।  
न मौनमवलम्बेत मर्त्यसन्दर्शनेन वा॥९७॥**

**इत्यादि चिन्तयन्नास्त स्तम्भा[^248]च्छादितविग्रहः।  
गीतावसानसमयप्रतीक्षणपरायणः॥९८॥ (तिलकम्)**

[^248]: # "स्तम्भावासित— ख"

**साथ[^249] गीतं समापय्य विनिधाय च[^250] वल्लकीम्।  
प्रणम्य चन्द्रचूडाय चन्द्रापीडमभाषत॥९९॥**

[^249]: # " सा तुः ख."

[^250]: # " तु ख"

**अतिथे स्वागतं तेऽस्तु[^251] सौम्य विश्रम्यतामिह[^252]।  
प्राप्तोऽसि स्वामिमां भूमिमित आगम्यतामिति॥१००॥**

[^251]: # " तुभ्यं ख"

[^252]: # " 'इति' ख. "

**चन्द्रापीडस्तदाकर्ण्य बह्वात्मानममन्यत।  
व्रजन्तीमनु वब्राज देवतावत्प्रणम्य ताम्‌॥१०१॥**

**विवेश नातिदुरेऽथ[^253] [^96]निजा[^255]श्रमगुहामसौ।  
स्थिताजिनकुशलाबुपात्रदण्डकमण्डलुम्॥१०२॥**

[^253]: # "च क"

[^255]: # "निजासन—”ख."

**तत्र दर्भासनासीनमुपविष्टा मृगाजिने।  
पप्रच्छ चन्द्रापीडं सा स्वरूपागमनादिकम्‌॥१०३॥**

**स तु सर्वं यथावृत्तमाचचक्षे सविस्तरम्‌।  
सा च[^256] विज्ञातवृत्तान्ता निर्जगाम ततो[^257] बहिः॥१०४॥**

[^256]: # " तु विज्ञाय बृत्तान्तं ख "

[^257]: # "'बहिर्मुहुः' ख"

**गृहीतभिक्षापात्रा च क्षणमास्त तरोरधः।  
आपूर्यत स्वयं शीर्णैः फलैस्तदमृतोपमैः॥१०५॥**

**ततस्तदुपयोगाय चन्द्रापीडं न्ययुङ्क्त सा।  
अन्वति[^258]ष्ठद्वचस्तस्याः स च विस्मयमानधीः॥१०६॥**

[^258]: # ".खादति स्माज्ञया तस्याः ख"

**तस्मिन्नथ कृताहारे गिरिप्रस्रवणान्तिकम्[^259]।  
गत्वा सापि फलहारप्रक्रियामात्रमाचरत्‌॥१०७॥**

[^259]: # " 'अन्तिकेः ख“"

**आगत्य च पुनस्तसिन्नुपविष्टा मृगाजिने।  
तारापीडोज्जयिन्यादिकथया क्षणमास्त सा॥१०८॥**

**अपि नाम स्ववृत्तान्तं मद्वदेषापि[^260] वर्णयेत्‌।  
पृष्टेत्येव[^261] क्षणं दध्यौ चन्द्रापीडः सकौतुकः॥१०९॥**

[^260]: # " 'महादेशेन' ख."

[^261]: # " “पृष्टेति क्षणमादध्योः ख. "

**दृष्ट्वा चास्याः प्रश[^262]ममधुरामाकृतिं भावशुद्धिं  
दाक्षिण्यं च प्रकृतिसरलं [^263]लब्धवाक्यावकाशः।**

[^262]: # "'प्रकृति— ख."

[^263]: # "“सरसं ख."

**आत्मोदन्तं नियतमखिलं वक्ष्यति प्रार्थितेयं  
चन्द्रापीडः क्षण[^264]मथ कृतप्रश्नसम्प्रत्ययोऽभूत्‌॥११०॥**

[^264]: # " 'क्षणकृतरतिः” ख. "

**इति श्रीविपश्चिद्वराग्रगण्याचार्यभट्टश्रीजयन्तसूनोरभिनन्दस्य कृतौ कादम्बरीकथासारे**

**तृतीयः स्वर्गः।  
———————————————**



**चतुर्थः स्वर्गः।**

**स तद्गुणप्रभावर्द्धिवर्धमानकुतूहलः।  
तामवोचत माधुर्यविनयोदारया गिरा॥१॥**

**त्वत्प्रसादकृतेनैव प्रागल्भ्येन ममा[^265]मुना।  
मनो भगवति[^266] प्रश्नचापले विनियुज्यते॥२॥**

[^265]: # " अधुना ममः क,"

[^266]: # " भगवती ख"

**न मादृशदृशां भूमिरियमाश्चर्यसन्ततिः।  
तदाख्यानप्रसादेन जनोऽयमनुगृह्यताम्‌॥३॥**

**कतमो मुनिगन्धर्वसिद्धविद्याधरादिषु।  
पुण्येन जन्मना वंशो[^267] भगवत्या विभूषितः॥४॥**

[^267]: # " वासो ख"

**पञ्चभूतात्मकं चेदं धत्ते धवलतां वपुः।  
कथं कामपि दिग्दन्तिदन्तकान्त्यतिशायिनीम्‌॥५॥**

**नवे वयसि किं चातितीव्रवैराग्यकारणम्‌।  
येनारण्यनिवासादि दुःखमीदृग्विषह्यते॥६॥**

**इति तस्मिन्वदत्येव स्थूलाश्रुकणवर्षिणी।  
रुरोद वल्कलच्छन्नवदना दिव्यकन्यका॥७॥**

**अकम्पत मनश्चैनां चन्द्रापीडस्य पश्यतः।  
इयमप्याकृतिर्नाम दुःखै[^268]र्यदभिभूयते॥८॥**

[^268]: # " द्वन्द्वैः ख"

**असौ चिरं रुदित्वा तु[^269] वल्कलोन्मृष्टलोचना।  
दीर्घमुष्णं च निःश्वस्य व्याहर्तुमुपचक्रमे॥९॥**

[^269]: # "च ख"

**राजपुत्र किमेतेन शोकैकवसतेर्मम।  
पापाया मन्दमाग्याया वृत्तान्तेन श्रुतेन ते॥१०॥**

**तथापि यदि ते चित्तं कुतूहलवशीकृतम्‌।  
तदिदं वर्ण्यते सर्वमाकर्णय यथाक्रमम्‌॥११॥**

**सुप्रसिद्धमिदं तावत्तव श्रुतिपथं गतम्‌।  
यत्सन्त्यप्सरसो नाम सुरसद्मनि कन्यकाः॥१२॥**

**चतुर्दश च तद्वंशा पृथक्पृथगवस्थिताः।  
सोमामृतसमीरार्कदक्षकन्या[^270]दिसम्भवाः॥१३॥**

[^270]: # " मन्वादि ख"

**दक्षकन्योद्भवं तेषु गन्धर्वाणां कुलद्वयम्‌।  
कन्ये मुनि[^271]ररि[^272]ष्टेति द्वे दक्षस्य बभूवतुः॥१४॥**

[^271]: # "मतिः क"

[^272]: # " 'च रिष्टेति' ख, "

**तत्र चित्ररथो नाम गन्धर्वाणामधीश्वरः।  
मुनेः[^273] श्रुतयशाः पुत्रश्चित्रसेनादिपूर्वजः॥१५॥**

[^273]: # " मतेः क"

**तुम्बुरुप्रभृतीनां तु ज्येष्ठोऽरिष्टासुतः कृती।  
हंसाभिधानो गन्धर्वराजः प्रथितपौरुषः॥१६॥**

**अकृत्रिममनुत्पन्नमन्युकालुष्यदूषणम्‌॥  
रूढं[^274] परस्परं प्रेम तयोर्गन्धर्वराजयोः॥१७॥**

[^274]: # " गाढं ख"

**इतश्च नातिदूरेऽस्ति नानारत्नोज्ज्बलो गिरिः।  
हेमकूट इति ख्यातः स निवासस्तयोर्द्वयोः॥१८॥**

**तस्मिन्वसन्त्यसङ्ख्यातास्तद्भजस्तम्भसंश्रिताः।  
गन्धर्वास्तौ[^275] च विस्रम्भभूमिर्जम्भद्विषः परा[^276]॥१९॥**

[^275]: # ". ते क."

[^276]: # "“पुरा क."

**अथ यत्सोमसम्भूतमेकमप्सरसां कुलम्।  
गौरी नामोदभूत्कन्या तस्मिन्प्रकृतिर्विधोः[^277]॥२०॥**

[^277]: # ". 'श्रियः ख."

**हंसो गन्धर्वराजस्तामुदूहे तुहिनप्रभाम्।  
ततो[^278] यथार्थं नामाप सर्वथा हंसगामिनी॥२१॥**

[^278]: # " यतो यथार्थतामाप क. "

**चिरसम्भृतविस्त्रम्भनिर्भरं गृहवासिनोः।  
तयोरुदभवत्सूनुरेकोऽपि न महात्मनोः॥२२॥**

**अहं तु मन्दभाग्यैका कन्या दुष्कृतकारिणी।  
उदभूवं तयोरस्य दुःखभारख भाजनम्‌[^279]॥२३॥**

[^279]: # " भागिनी क"

**अनपत्यतया तातो मज्जन्म सुतजन्मवत्‌।  
अभ्यनन्दत्कृतानन्दः[^280] कृतोदारमहोत्सवः॥२४॥**

[^280]: # "'धृतानन्दः क. "

**देहच्छायामिमां वीक्ष्य तातः[^281] सोमक्रमागताम्‌।  
चक्रे वर्णानुरूपं स महाश्वेतेति नाम मे॥२५॥**

[^281]: # "ततः क"

**साहं पितृगृहे बाला वल्लकीव कलस्वना।  
सञ्चरन्ती स्वबन्धूनामङ्कादङ्कमहर्निशम्‌॥२६॥**

**अनवाप्तरसास्वादस्नेहादिद्वन्द्वसन्ततिः।  
अत्यवाहयमक्लेशपेशलस्थिति शैशवम्‌॥२७॥ (युगलकम्‌)**

**अथ बाल्यात्परं प्रापमहं[^282] नानारसास्पदम्‌।  
मनोभवविक्राराणामेकमायतनं वयः॥२८॥**

[^282]: # " प्राप्तं मया नानारसोचितम्‌ः ख, "

**कदाचिदथ रम्येषु मधुमासदिनेष्वहम्।  
अच्छोदसरसि स्नातुमागतेह[^283] सहाम्बया॥२९॥**

[^283]: # "अस्मि ख. "

**अदृष्टपूर्वामन्यत्र विभूति[^284]मिह माधवीम्‌।  
सखीभिः सह पश्यन्ती कानने व्यचिरं चिरम्‌॥२०॥**

[^284]: # "'मञ्जरीमहमार्तवीम्‌” ख. "

**एकस्मिंस्तु[^285] प्रदेशेऽन्यपुष्पगन्धाभिभाविनम्‌।  
अभ्यजिघ्रमनाघ्रातपूर्वमामोदमद्भुतम्॥३१॥**

[^285]: # "कस्मिंश्चिदपि देशे ख"

**तस्य प्रभव[^286]जिज्ञासारसेन विहितत्वरा।  
नातिदूरं ततो गत्वाऽपश्यं मुनिकुमारकम्‌॥३२॥**

[^286]: # ".“प्रभाव” ख."

**अस्पृष्टमिव कन्दर्पमीशाननयनार्चिषा।  
कलङ्कविप्रुषा बिम्बमदूषितमिवैन्दवम्‌॥३३॥**

**विरुद्धैरपि रूपस्य मेखलाजिनवल्ककैः।  
कृताधिकमनोहारिरूपातिशय विभ्रमम्॥३४॥**

**तेन कर्णे कृता काचिदपूर्व[^287]तरुमञ्जरी।  
ततः प्रवृत्तमामोदं समाघ्रा[^288]तवती मुहुः॥३५॥**

[^287]: # "अपूर्वा ख"

[^288]: # "समर्थितवती क"

**तं कुतोऽप्यागतं कान्तं लावण्यामृतनिर्झरम्‌।  
चक्षुषा[^289] सम्पिबन्तीव कौतुकोत्क्षिप्तपक्ष्मणा॥३६॥**

[^289]: # "'चक्षुषैब पिबन्तीव ख"

**रूपैकपक्षपातेन गुणदोषानपेक्षिणा।  
ततः कुसुमबाणेन नीतास्मि वशमात्मनः॥३७॥**

**ततोऽपसर्तुकामाहमप्रगल्भतया मुहुः।  
केनापि प्रेर्यमाणेव तदन्तिकमुपागमम्[^290]॥३८॥**

[^290]: # "उपागता क"

**वन्द्यो मुनिजनश्चेति[^291] प्रचलन्मणिकुण्डलम्।  
अकार्षं वन्दनं तस्मै तन्मुखन्यस्तलोचना॥३९॥**

[^291]: # ". 'तेन क"

**ततः कृतप्रणामायां मयि सोऽपि मनोभुवः।  
पतितो गोचरे चित्रमलङ्घया[^292] भवितव्यता॥४०॥**

[^292]: # ".“अवन्ध्मा' ख, "

**स्मरोपदिष्टयेवाथ बृद्ध्वा तद्विकृतिं धिया।  
लब्धास्था[^293] तत्स्वरूपादिप्रश्नप्रागल्भ्यमागमम्‌॥४१॥**

[^293]: # ". “लब्धावकाशरूपादिः ख"

**अथ पार्श्वस्थमप्राक्षमन्यं मुनिकुमारकम्।  
तुल्यतेजस्तपोवेषं तस्य सब्रह्मचारिणम्‌॥४२॥**

**भगवन्कथ्यतां कोऽयं किमाख्यः किमिहा[^294]गतः।  
कर्णे च निहितानेन कस्येयं मञ्जरी तरोः॥४३॥**

[^294]: # "आगम,' ख"

**सोऽब्रवीत्सुरलोके[^295]ऽस्ति सुरासुरशिरोर्चितः।  
श्वेतकेतुरिति ख्यातो रुपेणाप्रतिमो मुनिः॥४४॥**

[^295]: # "लोकेषु ख"

**पुण्डरीक[^296]सरः स्नातुमगम[^297]त्स कदाचन।  
नित्यं सन्निहिता तत्र लक्ष्मीरेनं व्यलोकयत्‌॥४५॥**

[^296]: # " पुण्डरीकं ख."

[^297]: # ".अयासीत् ख"

**सा तदालिङ्गनेनेव दर्शनेनैव केवलम्‌।  
पुण्डरीके कृतार्थत्वमवाप स्वासनीकृते॥४६॥**

**उदपादि तदैवास्य[^298] कुमारः स तया श्रिया।  
तव पुत्रोऽयमित्युक्त्वा न्यवेदि श्वेतकेतवे॥४७॥**

[^298]: # " अथ ख"

**सोऽपि योगदृशा दृष्ट्वा यथावत्सुतमग्रहीत्‌।  
नाम जन्मो[^299]चितं चास्य पुण्डरीक इति व्यधात्‌॥४८॥**

[^299]: # ". 'जन्मानुगं' ख,"

**योजयामास संस्कारैरेनं मुनिजनोचितैः।  
अतीते[^300] बालभावे च न्ययुङ्क्त तपसे पिता॥४९॥**

[^300]: # "अतीव ख"

**अयं स पुण्डरीकस्ते वर्णितः शृणु मञ्जरीम्‌।  
चतुर्दशीति कृत्वाद्य[^301] प्रस्थितोऽयं शिवालयम्‌॥५०॥**

[^301]: # "मत्वाद्य क,"

**व्रजतो नन्दनेनास्य[^302] वनदेवतया स्वयम्‌।  
त्वदाकृतेरलङ्कारस्तुल्योऽयमिति सादरम्‌॥५१॥**

[^302]: # "चास्य ख. "

**पारिजातद्रुमादेषा दुग्धोदधिसमुद्भवात।  
आदाय मञ्जरीकर्णेऽनिच्छतोऽपि निवेशिता॥५२॥ (युगलकम्‌)**

**इति तस्मिन्वदत्येव चपले गृह्यतामियम्‌।  
किमनेन तव[^303] प्रश्नलोभेनेत्यभिधाय माम्‌॥५३॥**

[^303]: # "'तनुग्राहखेदेन ख, "

**अनायि मुनिना तेन सा सुरद्रुममञ्जुरी।  
अपनीय निजात्कर्णान्मम कर्णावतंसताम्‌॥५४॥ (युगलकम्‌)**

**मत्कपोलतलस्पर्शप्रकम्पतरलाङ्गुलेः।  
तेन [^304]प्रस्खलिता हस्तादक्षमाखा न लक्षिता॥५५॥**

[^304]: # " भ्रश्यन्ती तेन नाज्ञायि स्वाक्षमाला निजात्करात्‌” ख"

**अप्राप्तैव च सा भूमिं गृहीता[^305] सहसा मया।  
अंसिन्न[^306]वसरे छत्र्तग्राहिणी मामवोचत॥५६॥**

[^305]: # " “अग्राहि' ख."

[^306]: # "तस्मिन् ख"

**स्नाता देवी चिरं भर्तृदारिके गम्यतामिति।  
ततः प्रतिनिवृत्तायां मयि तं[^307] स सखावदत्‌॥५७॥**

[^307]: # ".तस्यः ख, "

**वयस्य किमिदं चेतस्तव विप्लुतमीदृशम्‌।  
हृतामप्यनयाज्ञासीरक्षमालां न कन्यया॥५८॥**

**हृता नामेयमनया हृदयं ननु रक्ष्यताम्‌।  
ततोऽसौ पुनरागत्य कृतकोपमिवावदत्‌[^308]॥५९॥**

[^308]: # ".'अब्रवीत' ख"

**अपहृत्याक्षमालां मे क्व चौरि ननु गम्यते।  
अथ मन्दं विहस्येव मयास्य निहिता करे॥६०॥**

**अवमुच्या[^309]क्षमालेति कण्ठान्मुक्तावली निजात्‌[^310]।  
साप्यलक्ष्यत नानेन मन्मुखासक्तदृष्टिना॥६१॥ (युगलकम्‌)**

[^309]: # "'मुक्तेयं” कः"

[^310]: # "निजा क, "

**सखीभिरपि मे दृष्टमिदं चापलमित्यहम्।[^311]  
किञ्चित्सञ्जातलज्जेव सरसस्ती[^312]रमागमम्‌॥६२॥**

[^311]: # ".उच्चकैः ख,."

[^312]: # ".“सरस्तटमुपागमम्‌' ख, "

**अत्राह[^313]मस्वतन्त्रैव क्षणं[^314] स्नात्वा यथातथा।  
अगच्छ[^315]मम्बया साकं तद्गुणाध्यायिनी गृहम्‌॥६३॥**

[^313]: # "तत्र ख"

[^314]: # " मुहुः ख, "

[^315]: # "अगच्छम् इत्यादिसार्धश्लोकः ख-पुस्तके नास्ति,"

**गत्वा सवेदनास्मीति विसृज्य सकलाः सखीः।  
प्रविश्यैकाकिनी गेहमासं मोहविसंस्थुला॥६४॥**

**क्व गतास्मि किमुक्तास्मि[^316] तेन कोऽसावहं च का।  
किमिदं कः प्रकारोऽयमिति नाज्ञासिषं तदा॥६५॥**

[^316]: # " 'किमयाता दृष्ट ख"

**मुहूर्तादेव[^317] ताम्बूलकरङ्कग्राहिणी मम।  
प्रिया तरलिका नाम प्रविश्येदमथाब्रवीत्‌॥६६॥**

[^317]: # ".इव ख"

**कर्णे मुनिकुमारेण कृता ते येन मञ्जरी।  
त्वय्यागतायां पृष्टास्मि स्वरूपं तेन तावकम्‌॥६७॥**

**तच्छ्रुत्वा च स्वयं तेन लिखित्वा वस्त्रपट्टिका।  
एकाकिन्यै त्वया तस्यै देयेत्युक्त्वा समर्पिता॥६८॥**

**ततस्तरलिकायास्तां हस्तादादाय पट्टिकाम्‌।  
तस्यां तु लिखितामार्यामिमामहमवाचयम्‌॥६९॥**

**“दूरं मुक्तालतया बिससितया विप्रलोभ्य[^318]मानो मे।  
हंस इव दर्शिताशो मानसजन्मा त्वया नीतः”॥७०॥**

[^318]: # " विप्रलभ्यमानः क."

**सा तु तैमिरिकस्येव बभूव बहुलक्षपा।  
मोहायार्याप्यनार्येव[^319] सुतरां मनसो मम॥७१॥**

[^319]: # " एव ख"

**वद तेन किमुक्तासि स किमुक्तस्त्वयापि वा।  
तदन्तिके च विस्त्रब्धमुषितासि कियच्चिरम्‌॥७२॥**

**इत्थं तरलिकां तत्त[^320]दालपन्ती पुनः पुनः।  
काञ्चिदह्नस्ततो मात्रामुन्मत्तेवात्यवाहयम् ॥७३॥**

[^320]: # " तत्र च ख"

**अथ प्रलम्बमानेऽर्के प्रतीहारी प्रविश्य मे।  
भर्तृदारिकया दृष्टौ यौ तो मुनिकुमारकौ॥७४॥**

**तयोरन्यतरो द्वारि तिष्ठतीति न्यवेदयत्‌।  
शीघ्रं प्रवेशयेत्युक्ता मया तदभिशङ्कया॥७५॥**

**सा च प्रवेशयामास तस्य तं सहचारिणम्‌।  
सत्कारेणोचितेनाहं भक्त्या चैनमपूजयम्‌॥७६॥ (तिलकम्‌)**

**मुहूर्तमिव विश्रम्य पर्यङ्के मदुपाहृते।  
मदुत्तरीयसंमृष्ट[^321]सुधौतचरणद्वयः॥७७॥**

[^321]: # " 'सम्मृष्टमाधौतचरणद्वयम ख."

**दृष्टिं तरलिकायां स वक्तुकामः क्षणं न्यधात्‌[^322]।  
मया सोऽज्ञा[^323]यि पृष्टस्तु लज्जमान इवाभ्यधात्‌॥७८॥**

[^322]: # "व्यधात्‌ ख"

[^323]: # " स ज्ञाय ख"

**अहं केपिञ्जलो नाम तस्य बालसुहृन्मुनेः।  
न वाक्प्रवर्वते मेऽस्मिन्ननभ्यस्त इवाध्वनि॥७९॥**

**क्वायं मुनिजनः शान्तः क्व चामी रागविभ्रमाः।  
तथापि व्यवसायोऽयं[^324] सुहृत्स्नेहवशा[^325]नुगः॥८०॥**

[^324]: # " मे ख"

[^325]: # " 'रुजाश्रितः' क."

**तदुच्यते[^326] प्रयातायां राजपुत्रि तदा त्ययि।  
हस्ते मुक्तावलीं बिभ्रत्स मदन्ति[^327]कमागतः॥८१॥**

[^326]: # "विदूयसे” ख"

[^327]: # " ममान्तिकं ख,"

**अहो रम्याक्षमालेयं त्वयानीतेति तत्क्षणम।  
विहस्य स मया नोक्तः किञ्चिच्चिन्तो[^328]पशान्तये॥८२॥**

[^328]: # "चिन्तोपपत्तयेः ख "

**नोपदेशप्रकारोऽसौ न स[^329] दृष्टान्तविस्तरः।  
न सोऽस्त्युपायो मन्त्रो वा यो मया नाभ्यधीयत॥८३॥**

[^329]: # "“च क. "

**वचोभिः खिद्यमानो मे त्रपानतशिरोधरः।  
क्षणमन्तार्हितो मत्तः सुतरां मामपीडयत्‌॥८४॥**

**कृच्छ्रादिव मया लब्धो विचित्य[^330] सकलं वनम्‌।  
अन्य एव स संवृत्तो[^331] वचसां यो न गोचरः॥८५॥**

[^330]: # " “विविच्यः ख"

[^331]: # " वृत्तान्तो' ख."

**तदवस्थं समालोक्य विगताशः[^332] प्रबोधने।  
अहं[^333] तत्प्राणरक्षार्थमेवं यत्न[^334]परोऽभवम्‌॥८६॥**

[^332]: # "विरताशः ख"

[^333]: # "अथः कः"

[^334]: # " यक्षकरो” क"

**आस्तीर्य नलिनीपत्र्तशयनं चन्दनोक्षितम्‌।  
तत्रस्थ[^335]स्यास्य कर्पूरपांसूनङ्गे[^336]ष्ववाकिरम्॥८७॥**

[^335]: # " ततस्तस्यापि क"

[^336]: # "'पांसुं” क. "

**त्रपया तदवस्थोऽपि स किचिद्वक्तुमक्षमः।  
सबाष्प[^337]मलिनां दीनां मयि दृष्टिमपातयत्‌॥८८॥**

[^337]: # ". 'सवाष्पमातुरां क."

**ततोऽहमिदमेवात्र प्राप्तकालमचिन्तयम्‌।  
राजपुत्रि सुहृत्प्राणदक्षिणा[^338]र्थमुपागतः॥८९॥**

[^338]: # "रक्षणार्थं क"

**अतः परं तु सामर्थ्यगतार्था मम या गिरः।  
कण्ठे विपरिवर्तन्ते नूनं त्वमपि वेत्थ ताः॥९०॥**

**इति तस्मिन्वदत्येव प्रविश्य प्राह कञ्चुकी। \`  
त्वमस्वस्थशरीरेति ज्ञात्वा देवी समागता॥९१॥**

**तच्छ्रुत्वा जनसम्मर्दभीरु[^339]र्यातः कपिञ्जलः।  
न[^340] स प्रतिवचः किञ्चिन्मामकीनमपैक्षत॥९२॥**

[^339]: # " भीतो क"

[^340]: # " नैव क"

**तदुक्तां तु दशां तस्य जनस्यावेत्य[^341] तादृशीम्।  
आगतां नागतां वापि नाहमम्बाम[^342]लक्षयम्॥९३॥**

[^341]: # "आवेद्य ख"

[^342]: # ".ह्यम्बामहं क."

**ततः कमलिनीं रक्तां[^343] त्यक्त्वा क्वापि गते रवौ।  
उदिते सर्वथा वामे विधौ वैधुर्य[^344]दायिनि॥९४॥**

[^343]: # "त्यक्त्वा रात्रौ ख"

[^344]: # "वैधेयनाशिनि ख "

**उन्मूलितेव कामेन दग्धेव विरहाग्निना।  
पुनस्तरलिकामेव साश्रुकण्ठमवादिषम्‌॥९५॥**

**अहं यथातथा वापि भवेयं स तु मत्कृते।  
कथं भविष्यतीत्येषा चिन्ता दलयतीव[^345] माम्‌॥९६॥**

[^345]: # "दोलायतीव” ख"

**इतः शास्त्रं कुलाचारो लज्जा गुरुजनाद्भयम्‌।  
इतः कामो[^346] वसन्तोऽपि विरहो मरणाद्भयम्‌॥९७॥**

[^346]: # ". 'कामोऽतिदुर्धर्षो ख, "

**वद किं करवाण्यत्र मुह्यतीव मनो सम।  
माभून्मे नरके[^347] विप्रप्राणबाधेन मज्जनम्‌॥९८॥**

[^347]: # "दुस्तरे विप्रवधैनसि निमज्जनम्‌` ख"

**इत्यादि विलपन्त्या मे तत्कालसुलभोदया।  
जहार चेतनां मूर्च्छा निद्रेव नयनद्युतिम्‌॥९९॥**

**अथ मां ग्राहयित्वासौ सञ्ज्ञा शीतजलानिलैः।  
निपत्य पादयोः साश्रुनेत्रा तरलिकाब्रवीत्‌[^348]॥१००॥**

[^348]: # "अवदत्‌” ख. "

**किं[^349] शास्त्रं गुरवः के च का लज्जा का कुलस्थितिः।  
रक्ष्यन्तामात्मनः प्राणा युनस्तस्य च सर्वथा॥१०१॥**

[^349]: # ", कि शास्त्रम् इत्यादिश्लोकः ख. पुस्तके नास्ति"

**गन्धर्वाणां विवाहोऽयसमिच्छयान्योन्यसङ्गमः।  
शास्त्रेषु पठितस्तस्मात्तूर्णमुत्तिष्ठ गम्यताम्‌॥१०२॥**

**तयै[^350]वमुक्ता तत्कालयोग्यवेषग्रहा गृहात्‌।  
तामेकामग्रतः कृत्वा निर्गताहमलक्षिता॥१०३॥**

[^350]: # "तथा ख, "

**अभ्यागममिमां भूमिमौत्सुक्यचटु[^351]लैःपदैः।  
तदा मे दक्षिणं चक्षुरस्पन्दत मुहुर्मुहुः॥१०४॥**

[^351]: # "चतुरैः क"

**अव्यक्तमिव चाश्रौषं कानने रुदितध्वनिम्‌।  
तेन चोत्पन्नशङ्काहमगच्छमधि[^352]कत्वरम्॥१०५॥**

[^352]: # "अधिकज्वरा” क,"

**पदानि कतिचिद्गत्वा हा वयस्य तपोनिधे।  
पुण्डरीक क्व यातोऽसि शून्यं[^353] कृत्वा जगत्रयम्‌॥१०६॥**

[^353]: # " शून्यीकृत्यः क. "

**हा हतोऽस्मि विलीनोऽस्मि दग्धोऽस्मि करवाणि किम्‌।  
गत्वा तातस्य वक्ष्यामि श्वेतकेतोः किमग्रतः॥१०७॥**

**अयि[^354] पापे महाश्वेते न तुष्टास्यक्षमालया।  
हृता मे सुहृदोऽमुष्य प्राणवृत्तिरपि त्वया॥१०८॥**

[^354]: # " अहो क"

**वयस्य देहि मे वाचं न जीवामि त्वया विना।  
ततोऽशृणवमित्यादिविलपन्तं कपिञ्जलम्‌॥१०९॥ (चक्क[^355]लकम्‌)**

[^355]: # " कलापकं चतुर्भिः ख"

**अथ प्रवेप[^356]मानाङ्गी विसंस्थुलपदक्रमा।  
केनाप्युत्क्षिप्य नीतेव तं प्रदेशमयाशिषम्‌॥११०॥**

[^356]: # " प्रकम्पमानाङ्गी विसंष्ठुल' ख."

**गत्वा तमहमद्राक्षं तत्क्षणो[^357]त्कान्तजीवितम्‌।  
किं चिरेणागतासीति रुषित्वेव तथा सितम्‌॥१११॥**

[^357]: # " तत्क्षणात्क्लान्त क"

**एतावदुक्त्वा मुर्च्छाधिं[^358] स्ववृत्तान्तमतः परम्‌।  
विवक्षन्तीव सा कन्या निपपात शिलातरे॥११२॥**

[^358]: # " 'मूर्च्छर्ता क"

**प्रनष्टचेतनां चैनां चन्द्रापीडोऽवलोक[^359]यन्‌।  
प्रश्नकारिणमात्मानं कृतागसममन्यत॥११३॥**

[^359]: # "व्यलोकयत्‌ ख"

**ससम्भ्रमं समुत्थाय कदलीदलमारुतैः।  
तोयैश्च शिशिर[^360]स्वच्छैः सञ्ज्ञामेनामजिग्रहत्‌॥११४॥**

[^360]: # "“शिशिरैः ख"

**अवाप्य[^361] चेतनामेषा स्मृत्वा तद्व्यसनं निजम्‌।  
तदेवानु[^362]भवन्तीव रुरोद विललाप च॥११५॥**

[^361]: # ". 'अवाप्तचेतना चासौ' खं."

[^362]: # " तथैवः क"

**तरूणामप्यभज्यन्त हृदयानि तदा[^363] शुचा।  
प्रतिशब्द[^364]निभेनोच्चैः ककुभोऽप्यरुदन्निव॥११६॥**

[^363]: # "“यतस्तदा क"

[^364]: # ". प्रतिशब्देन चैवोच्चैर्दिशोऽपि रुरुदुश्चिरम्‌ ख "

**चन्द्रापीडोऽथ तामाह बाष्प[^365]रुद्धविलोचनः।  
आस्तां न शक्यते वक्तं श्रोतुं चान्य[^366]दतः परम्‌॥११७॥**

[^365]: # "वाष्पैः क"

[^366]: # "वा ख"

**कथमप्युपसंहृत्य शोकावेगं तु सावदत्‌।  
किमतः परमस्त्यन्यद्वाच्यं श्रोतव्यमेव वा[^367]॥११८॥**

[^367]: # ". चः ख."

**किं तु क्लेशाय येनाशा[^368]लेशेन हतजीवितम्‌।  
राजपुत्र बिभर्मीदं तदाश्चर्यं ब्रवीमि ते॥११९॥**

[^368]: # "“आशाबन्धेन' ख,"

**तदा तस्यातितीव्रस्य दुःख[^369]भारस्य तादृशी[^370]।  
न काचिन्मरणादन्या मयाग्राहि प्रतिक्रिया॥१२०॥**

[^369]: # ". 'दुःखदावस्य ख"

[^370]: # " तादृशः ख."

**मरणव्यवसायनिश्चला[^371] कृतवत्यसि च संविधानकम्‌।  
निरियाय च चन्द्रमण्डलात्पुरुषः कोऽपि हिमोपमद्युतिः॥१२१॥**

[^371]: # "“निश्चयं ख."

**सोऽवतीर्य गगनान्महीतले[^372] तं निधाय भुजपञ्जरान्तरे।  
उत्पतन्दिवभुदार[^373]या गिरा मामवोचत पितेव सान्त्वयन्‌॥१२२॥**

[^372]: # " 'महीतलं क"

[^373]: # "उदात्तया' ख"

**विरम मरणो[^374]द्योगादस्मात्प्रियेण तवामुना  
पुनरपि महाश्वेते वत्से भविष्यति सङ्गमः।**

[^374]: # "'मरणोपायात्‌” ख"

**इति गिरमहं तस्माच्छ्रुत्वा ससम्भ्रमविस्मया[^375]  
किमिदमिति तं तत्रापृच्छं प्रसृत्य कपिञ्जलम्‌॥१२३॥**

[^375]: # ". 'विस्मयात्” क."

**प्रियसखमपहृत्य क्वाधुना गच्छसीति  
स्फुरितविकटकोपाटोपमुक्तो[^376]र्ध्वदृष्टिः।  
मम वचनमदत्त्वा किञ्चिदेवातिवेगा  
कृतपरिकरबन्धः सोऽप्यगादन्तरिक्षम्‌[^377]॥१२४॥**

[^376]: # "“मुक्तार्द्र—” क"

[^377]: # "अन्तरिक्षे क."

**पश्यन्त्यां[^378] मयि ते त्रयोऽपि विविशुः शीतत्विषो मण्डलं  
तत्सम्प्रत्यसंहृतोचितविधिः साहं[^379] श्रितेदं व्रतम्‌।**

[^378]: # " प्रत्यक्षं ख."

[^379]: # "सास्मि ख."

**तातेनैत्य ततः प्रभृत्यहमिहासीनाथ[^380] नेतुं गृहं  
यत्नेनापि न पारितेति सुचिरं[^381] स्थित्वा गतः स्वान्गृहान्‌॥१२५॥**

[^380]: # " एवं” ख, "

[^381]: # "“स चिरं क"

**इति श्रीविपश्चिद्वराग्रगण्याचार्यभट्टश्रीजयन्तसूनोरभिनन्दस्य कृतौ कादम्बरीकथासारे**

**चतुर्थः सर्गः।**  
**——————————————**



**पञ्चमः सर्गः।**

**इत्यावेदितनिर्व्याजनिजवृत्तान्तविस्तरा[^382]।  
विरराम महाश्वेता निःश्वासग्लपिताघरा॥१॥**

[^382]: # ". वृत्तान्तबहुविस्तरा ख"

**चन्द्रापीडस्तु तद्ध्यायन्मनोभवविजृम्भितम्‌।  
सकम्पबिस्मयोऽप्यन्तस्तामवोच[^383]त सान्त्वयन्‌॥२॥**

[^383]: # " “अथोवाच' ख."

**मया ते[^384] चिरवृत्तोऽपि परिणाममपश्यता।  
शोकः प्रत्यग्रतां नीतः कौतुका[^385]विष्टचेतसा॥३॥**

[^384]: # "तव मनोवृत्तेः क."

[^385]: # "'कोतुकाक्षिप्त क. "

**सा त्वमर्हसि कल्याणि न शोकं कतुमीदृशम्।  
आश्वासितासि[^386] देवेन नूनं चन्द्रमसा स्वयम्‌॥४॥**

[^386]: # "आश्वासिता तदा तेन क"

**कोऽन्यस्तथाविधाकारस्तच्छक्तिर्मण्डलात्ततः।  
निर्गत्य तत्रैव विशन्निशि सम्भाव्यते नरः॥५॥**

**श्रूयन्ते गतयश्चित्रा देवतानामनेकशः।  
वरशाप[^387]तपःशक्तियोगवैचित्र्यनिर्भिताः॥६॥**

[^387]: # " दान क. "

**श्वेतकेतुश्च तस्यास्ते[^388] प्रजापतिसमः पिता।  
नास्त्येव तस्य तपसामसाध्यं[^389] नाम किञ्चन॥७॥**

[^388]: # " असौ क"

[^389]: # " अवश्यं ख"

**इत्यादियुक्त्युपन्यासैरितिहास[^390]निदर्शनैः।  
महाश्वेतां समाश्वास्य पुनः पप्रच्छ तामसौ॥८॥ **

[^390]: # "'इतिहासैः ख,"

**सा ते तरलिका नाम विस्रम्भनिभृता सखी।  
गता भगवति क्वाद्य नहि सा त्वाम विमुञ्चति॥९॥**

**अथोवाच[^391] महाश्वेता क्वासौ याति तपस्विनी।  
शृण्वत्रापि निमित्तं मे यथा सा[^392] प्रहिता मया॥१०॥**

[^391]: # "पृष्टोवाचः ख"

[^392]: # " असौ ख"

**यः स[^393] चित्ररथो नाम गन्धर्वाधिपतिर्मया।  
वर्णितो येन देवोऽयं प्रत्यष्ठापि महेश्वरः॥११॥**

[^393]: # "“च' ख."

**येन [^394]खानितमत्यच्छमच्छोदाख्यमिदं सरः।  
येन चैत्ररथं नाम कृतमेतच्च काननम्‌॥१२॥**

[^394]: # " येनाखानि तदत्यच्छ—” कः "

**तस्या[^395]मृतसमुद्भूते प्रसूताप्सरसां कुले।  
महिषी मदिरा नाम जीवितादपि वल्लभा॥१३॥**

[^395]: # "“यस्यामृतात्समुत्पन्नेः ख,"

**तयोः कादम्बरी नाम कन्यास्ति नवयौवना।  
लक्ष्मीमुपहसन्तीव रूपलावण्यविभ्रमैः॥१४॥**

**तया सह महत्प्रेम बाल्यात्प्रभृति मे तथा।  
यथैकमेव जानाति जनो जीवितमावयोः॥१५॥**

**विदित्वा दग्धवृत्तान्तमिमं सपदि मामकम्‌।  
साग्रहीन्नियं वाला वार्यमाणापि बन्धुभिः॥१६॥**

**महाश्वेतास्य[^396] दुःखस्य यावन्न लभतेऽवधिम्‌।  
तावन्मया न कर्तव्यं[^397] पाणिग्रहणमात्मनः॥१७॥**

[^396]: # "“स्वदुःखस्या ख"

[^397]: # ".“सोढव्यं ख. "

**अतो[^398] यथायथा तस्या यौवनश्रीर्विजृम्भते।  
तथात[^399]था ततो धत्ते खेदं चित्ररथः परम्‌॥१८॥**

[^398]: # "यतो ख"

[^399]: # " .“ततस्तथा तथा खेदं धत्ते ख"

**सन्दिष्टं तेन मे पुत्रि यतस्वावहिता तथा।  
यथेमं दुर्ग्रहग्रन्थिं विमुञ्चति तव स्वसा॥१९॥**

**अतस्त[^400]रलिका तत्र मया प्रकरणोचितम्‌।  
प्रहिता बहु सन्दिश्य श्वश्चागन्ते[^401]ह सा पुनः॥२०॥**

[^400]: # "ततः ख. "

[^401]: # " 'आगत्र्येव” ख."

**इतीत्थं कथया नीत्वा महाश्वेताश्रमे दिनम्‌।  
चन्द्रापीडः प्रयातेऽस्तमर्के सन्ध्याविधिं व्यघात्‌॥२१॥**

**तत्रेन्द्रायुधमानीय संयम्य निचुले क्वचित्‌।  
महाश्वेतोदिते देशे पर्णशय्यामकल्पयत्‌॥२२॥**

**तस्यां[^402] तस्य निषण्णस्य श्रान्तस्यापि यथा[^403]श्रुताम्‌।  
कथामाश्चर्यबहुलां तामेव ध्यायतश्चिरम्‌॥२३॥**

[^402]: # "'तस्य शय्यानिषण्णस्य' ख."

[^403]: # "'यथातथा' ख."

**पुनश्चिन्त[^404]यतः सेनां वैशम्पायनमेव च।  
पत्त्रलेखां च सा रात्रिर्गता प्रायेण जाग्रतः॥२४॥**

[^404]: # "'पुनःपुनर्महाश्वेताम्ः ख. "

**इन्द्रायुधस्तु[^405] बाष्पार्द्रां महाश्वेतामुखे दृशम्।  
सप्रत्यभिज्ञमश्वोऽपि भृशं[^406] मानुषवद्ददौ[^407]॥२५॥**

[^405]: # "'च वाहाग्र्यो' ख."

[^406]: # "'दृष्टिं' ख,"

[^407]: # "'दधौ' क."

**प्रयातायां तु यामिन्यां[^408] कृतप्राभातिकक्रिये।  
समाधेरुत्थितां बुद्ध्वा महाश्वेतां तदन्ति[^409]कम्‌॥२६॥**

[^408]: # "'शर्वर्या' ख."

[^409]: # "'तदन्तिके' ख."

**चन्द्रापीडे प्रविष्टेऽथ नातिदूरोदि[^410]ते रवौ ।  
सगन्धर्वकुमारासौ ततस्तरलिकाययौ॥२७॥**

[^410]: # "'दुरं गतेः ख."

**सा प्रविश्य महाश्वेतां प्रणिपत्योपविश्य च।  
सिद्धार्थासीति तत्पृष्टा वलित[^411]ग्रीवमब्रवीत्‌॥२८॥**

[^411]: # "'चलित–' ख. "

**त्वत्सन्देशं यं निश्म्योक्तं यत्तया भर्तृदारिके।  
तत्त्वं[^412] तच्छासनायातकेयुरकमुखाच्छृणु॥२९॥**

[^412]: # "'तच्च तच्छासनाकारिः' ख."

**अथ केयूरकः प्राह नानभिज्ञासि चेतसः।  
स्वस्य स्वमेव चेदं[^413] ते मन्मनः किं परीक्षसे॥३०॥**

[^413]: # "'चेतस्ते' ख."

**अथवा[^414] भूमिरस्याहमुपालम्भस्य भूयसः।  
तपोवननिवासिन्यां त्वयि[^415] स्वगृहवर्तिनी॥३१॥**

[^414]: # "'भूमिर्वैतादृशस्याहं' ख. "

[^415]: # " 'त्वय्यहङ्गृहदवासिनी' ख. "

**आकर्ण्य तव सन्देशवच[^416]स्तरलिकामुखात्‌।  
एष ते प्रतिसन्देशः प्रियसख्या निवेदितः॥३२॥**

[^416]: # " 'वाचः' क."

**इति श्रुत्वा महाश्वेता क्षणं[^417] निश्चललोचना।  
चिन्तयित्वा किमप्यन्तश्चन्द्रापीडमभाषत[^418]॥३३॥**

[^417]: # "'चिरं' ख."

[^418]: # "'अवोचत' ख."

**राजपुत्र तवाकारः[^419] सदाचारश्च पेशलः।  
तनोति सर्वभावेन[^420] निःस्पृहाणामपि स्पृहाम्‌॥३४॥**

[^419]: # "'तवोदारो समाचारश्च' ख."

[^420]: # "'भावेषु' क."

**कादम्बरीवचश्चेदं श्रुतं केयुरकात्त्वया।  
तत्र को वा प्रतीकारः स्वयं मद्गमनादृते॥३५॥**

**इतो न दुरे गन्धर्वराजधानी तदद्य ताम्‌।  
पश्यास्मदनुरोधेन श्वो गन्तासि स्ववाहिनीम्‌॥३६॥**

**उदात्तमधुरामेतां दर्शयन्त्या तवाकृतिम्‌[^421]।  
बहु गन्धर्वलोकस्य मयाप्युपकृतं भवेत्‌॥३७॥**

[^421]: # "'त्वदाकृतिम्‌' ख."

**चन्द्रापीडस्तु तां प्राह बहूक्त्वा भगवत्यलम्‌ ।  
अनतिक्रमणीयं मे मुनीनामपि शासनम्‌॥३८॥**

**प्रावर्तत ततो गन्तं स महाश्वेतया सह।  
पद्भ्यामेव नियुज्याश्वं[^422] प्रति केयूरकं दृशा॥३९॥**

[^422]: # " न्ययुङ्क्ताग्रे' क."

**केयूरकोऽग्रतो गत्वा कादम्बर्यै न्यवेदयत्‌।  
तस्यागमनमाकारं गुणांश्चाहितविस्मयान्‌॥४०॥**

**स[^423] गत्वा किञ्चिदध्वानं चन्द्रापीडः समासदत्‌।  
गन्धर्वराजनगरीमपराममरावतीम्‌॥४१॥**

[^423]: # " 'गत्वा कथञ्चिदध्वानं' ख."

**आसाद्य राजधानीं च तुङ्गस्फटिकतोरणाम्‌।  
सप्त कक्षा[^424] अतिक्रम्य कन्यान्तः पुरमभ्यगात्‌॥४२॥**

[^424]: # " 'कक्ष्या' क."

**केयूरकोपदिष्टेन वर्त्मनाथ प्रविश्य सः।  
भास्वन्मणिमय[^425]स्तम्भं श्रीमण्डपमलोकयत्‌॥४३॥**

[^425]: # " 'मरकत' ख."

**तन्मध्ये मणिपर्यङ्कमध्यासीनां महीयसा।  
रत्नाभरणचित्रेण स्त्रैणेन परिवारिताम्‌॥४४॥**

**देहप्रभावितानेन रत्नकान्त्यभिभाविना।  
प्रतप्त[^426]कनकोद्द्योतधौता विदधतीं दिशः॥४५॥**

[^426]: # "'उत्तप्तकनकद्योत–' ख."

**तटस्थमप्यार्द्रयतः सदाचा[^427]रमभिन्दतः।  
स्थितां नवस्य लावण्यसिन्धोरन्तस्तर[^428]ङ्गिणः॥४६॥**

[^427]: # "'स्वमाधारमसिञ्चतः' ख. "

[^428]: # "'रत्नतरङ्गिणीम्' ख."

**सर्वोपमे[^429]यभावानामुपमानत्वमागताम्‌।  
कादम्बरीं ददर्शासौ दूरविस्फारितेक्षणः॥ ४७॥(चक्क[^430]लकम )**

[^429]: # "'सर्वोपमान' क."

[^430]: # " 'चतुर्भिः कालापकम्‌' ख."

**परातिशयकोटिस्थ[^431]रूपलावण्यसम्पदम्‌।  
तस्यां निरूपयन्नन्तरचिन्त[^432]यदथ क्षणम्‌॥४८॥**

[^431]: # "'कोटिस्थां' ख."

[^432]: # "'चिन्तयामास' ख."

**सत्यं न कश्चिदाश्चर्यनिर्माणेष्ववधिर्विधेः ।  
ईदृशान्यपि रत्नानि स्रष्टं प्रभवति स्म[^433] यः॥४९॥**

[^433]: # "स्वयम्' ख."

**मोघं याति हरेः कालो लक्ष्मीं वक्षसि बिभ्रतः।  
देहार्धेन वहन्गौ[^434]रीं वञ्चितश्चन्द्रशेखरः॥५०॥ **

[^434]: # "'दधद्‌' ख."

**कृतमक्ष्णां सहस्रेण निर्निमेषेण वज्रिणः।  
अनन्यकर्मणा येन नित्यमेषा न दृश्यते॥५१॥**

**इत्यादिचिन्तयत्यस्मिन्नथ चित्ररथात्मजा।  
प्रत्युद्ययौ[^435] महाश्वेतामभ्युत्थाय ससम्भ्रमम्‌॥५२॥**

[^435]: # "प्रत्यागच्छत्' ख."

**महाश्वेता तु [^436]निःशङ्कमालिङ्ग्य स्नेहनिर्भरम।  
तां चन्द्रापीडनिर्देशपुरःसरमभाषत॥५३॥**

[^436]: # " 'अथ निःसङ्गं' क."

**अयं हि भारते वर्षे सार्वभौमस्य भूपतेः।  
तारापीडस्य तनयश्चन्द्रापीड इहागतः॥५४॥**

**अनेनेह प्रविष्टेन दृष्टेर्विषयवर्तिना।  
भवत्वखिलगन्धर्वकुटुम्बकमहोत्सवः॥५५॥**

**अथ कादम्बरी तस्मिन्पक्ष्मोत्क्षेपविकसिनीम्‌[^437]।  
आकर्णप्रसृतापाङ्गां चिरं दृशम[^438]पातयत्‌॥५६॥**

[^437]: # "'विलासिनीम्' क."

[^438]: # "'दृष्टिं' ख."

**द्वारेण विवृतेनाथ तेनैव निजमायया।  
मदनोऽन्तः प्रविश्यास्या बबन्ध हृदये पदम्‌॥५७॥**

**रत्नपीठोपविष्टे तु[^439] चन्द्रापीडे तदन्तिके।  
अध्या[^440]सातां कथामेकां हंसचित्ररथात्मजे॥५८॥**

[^439]: # " 'च' क."

[^440]: # " 'अध्यासिषातामेकत्र' ख."

** लज्जास्तिमितमासीनामथ[^441] कादम्बरीं बलात्‌।  
महाश्वेता गृहाचारं चन्द्रापीडे न्यधापय[^442]त्‌॥५९॥**

[^441]: # " 'साथ' क."

[^442]: # "'व्यदापयत' ख."

**शरीरनिर्विशेषास्याः कादम्बर्याः प्रिया सखी।  
मदलेखाभिधानासस्य चरणक्षालनं व्यधात्‌॥६०॥**

**कादम्बरी तु सस्वेदप्रकम्पिक[^443]रपल्लवा।  
ताम्बूलच्छद्मनात्मानं स्वयमस्मै न्यवेदयत्‌॥६१॥**

[^443]: # "'प्रकम्प' ख."

**तदङ्गुलितलस्पर्शमाधुर्यहृ[^444]तमानसा।  
नालक्षयत सा रत्नवलयं स्खलितं[^445] करात्‌॥६२॥**

[^444]: # "'विश्रान्तिहृतः' क."

[^445]: # "'प्रच्युतं' ख."

**चन्द्रापीडोऽपि तत्पाणिसंस्पर्श[^446]सुखसम्पदा।  
तदा त्रिभुवनाधीशमिवात्मानममन्यत॥६३॥**

[^446]: # "'तलसंस्पर्शमात्रया' ख."

**ततस्तूष्णी[^447]मुपासीने सर्वस्मिन्कन्यकाजने।  
शुकसारिकयोरासीत्कलहः[^448] केलिकारणम्‌॥६४॥**

[^447]: # "'तूष्णीमथ तदासीने' ख."

[^448]: # "'केलिः कलहकारणम्‌' ख."

**तत्प्रसङ्गे च सोत्प्रसं[^449] तयोः कलहवृद्धये।  
चन्द्रापीडोऽब्रवीद्येन सर्वा जहसु[^450]रङ्गनाः॥६५॥**

[^449]: # " 'सोत्प्राशं' क."

[^450]: # "तुतुषुः ख. "

**ताश्च दन्तप्रभाधौतदिङ्मुखा मधुरस्वराः।  
चन्द्रापीडगिरो जह्नुस्तरां[^451] कादम्बरीमनः॥६६॥**

[^451]: # "'तदा' ख."

**क्षणमात्रे[^452] व्यतीतेऽथ प्रविश्य प्राह कञ्चुकी।  
राजपुत्र्यौ युवां द्रष्टुं देवी निर्विघ्नमिच्छति॥६७॥**

[^452]: # "'क्षणं प्रविश्य तत्स्थानं मधुरं प्राह' ख."

**क्रीडापर्वतके कृत्वा सुतारमणिमन्दिरे।  
चन्द्रापीडं[^453] सुखासीनमायातं मातुरन्तिकम्‌[^454]॥६८॥**

[^453]: # "'चन्द्रापीडसुखावासमायाता' ख."

[^454]: # "'अन्तिके' ख."

**तत्राध्यासितपर्यङ्कं केयूरकपुरःसराः।  
मुख्याः कादम्बरीसख्यश्चन्द्रापीडं सिषेविरे॥६९॥**

**स्नानानुलेपनाहारक्रियाः स[^455] तदुपाहृतैः।  
चक्रेऽमर्त्यपु[^456]रप्राप्यैर्विभवैरुपपादिताः॥७०॥**

[^455]: # " 'च तदुपाहृताः' ख."

[^456]: # " 'शक्रपुरात्प्राप्तैः' ख."

**अथ मन्दप्रभे भानौ भेजे कादम्बरी निजम्।  
मदलेखाद्वितीयैव महाप्रासादमस्तकम्‌॥ ७१॥**

**चन्द्रापीडं च तत्रस्था क्रीडापर्वतवर्तिनम्‌।  
निरवर्णयदाकर्णप्रसृतापाङ्गया दृशा॥७२॥**

**चन्द्रापीडे तदा[^457] तस्याः कटाक्षास्तीक्ष्णकोटयः।  
समं निपेतुस्तस्यां[^458] च कुसुमायुधसायकाः॥७२॥**

[^457]: # "'कटाक्षाश्च तदीयाः' क."

[^458]: # "'अस्यां' क."

**कटाक्षैरुपनीतं च क्षिप्रं[^459] कृतसमागमैः।  
तल्लावण्यं पिबन्तीव मदलेखामुवाच सा॥७४॥**

[^459]: # "'तीव्रं' ख."

**न मया मदनो नाम कथास्वपि सखि[^460] श्रुतः।  
अद्य प्रत्यक्षतो दृष्टः स चन्द्रापीडविग्रहः॥७५॥**

[^460]: # "'स विश्रुतः' ख."

**न महाश्वेतया किञ्चिदुक्तं प्रकृतवस्तुनि।  
अथ[^461] चेहानयन्त्यामुं[^462] सर्वमुक्तमनक्षरम्‌॥७६॥**

[^461]: # "'अथवा' ख."

[^462]: # "'तु' ख."

**अपि नाम विकारो[^463] मे तदानीं लक्षितस्तया।  
किं वा न लक्षये[^464]देषा दृष्टमन्मथविक्रिया[^465]॥७७॥**

[^463]: # "'तदा नासौ प्रकारो' क."

[^464]: # "'लक्षयति' ख."

[^465]: # "'विभ्रमा' ख."

**अद्यापि निजसङ्कल्पविरुद्धां विकिया[^466]भिमाम।  
मनसः संहराम्येव स्थुलामपि यथातथा॥७८॥**

[^466]: # " 'प्रक्रियां' क."

**किं त्वसौ[^467] शेषहारोऽस्यास्तुल्यं मण्डनमाकृतेः।  
उरस्यनिच्छतोऽप्यस्य तूर्णमा[^468]मुच्यतां त्वया॥७९॥**

[^467]: # " 'पुनः' क."

[^468]: # " 'त्वरितं ग्रथ्यतां' क."

**मदलेखा गृहीत्वा तं हारं गत्वा तदन्तिके।  
तथोवाच यथा तेन[^469] प्रत्युक्ता सा न किञ्चन॥८०॥**

[^469]: # " 'अनेन' ख"

**आमुच्यत ततः कण्ठे हारोऽस्य मदलेखया।  
शरदिन्दुकरोदारप्रभापूरित[^470]दिङ्मुखः॥८१॥**

[^470]: # " 'प्लावित-' ख."

**चन्द्रापीडश्च हारेण रराज कृतमण्डनः।  
पूर्णेन्दुरिव नक्षत्रकलापपरिवारितः॥८२॥**

**कादम्बरी चिरं त्वेनमवलोक्य निशागमे।  
प्रासादशिखरात्तस्मान्मन्दम[^471]न्दमवातरत्‌॥८३॥**

[^471]: # "'खेदमन्दं' क."

**उत्थाय चन्द्रापीडोऽथ कृतसन्ध्याविधिस्ततः।  
अवालम्बत[^472] देहेन शय्यां चिन्तां च चेतसा॥८४॥**

[^472]: # "'युवालम्बत' क."

**किं स्युर्गन्धर्वकन्याया विलासाः सहजा इमे।  
उतेमं [^473]जनमुद्दिश्य बालां नर्तयति स्मरः॥८५॥**

[^473]: # "'अथेमं' ख."

**दृष्टिर्निपुणदूतीव यथा[^474] त्वस्या विचेष्टते।  
तथा[^475] सम्भावनाहीयं मम चेतसि वर्तते॥८६॥**

[^474]: # "'तथा तस्या व्यचेष्टत' क."

[^475]: # "'यथा सम्भावनान्यैवः' क."

**अवैमि[^476] पुष्पितस्तावत्किन्नरानुसृतिक्रमः।  
स कालपरिणामेन न जाने किं फलिष्यति॥८७॥**

[^476]: # "'दृश्यते पुष्पितस्तावक्तिमनङ्ग इति द्रुमः' ख."

**अथवा यौवनोन्मादकृत एष[^477] मम भ्रमः।  
क्व[^478] मनुष्यः क्व गन्धर्व इति किं तेन सङ्गतम्‌॥८८॥**

[^477]: # " 'एव' ख."

[^478]: # "'क्व मनुष्याः क्व गन्धर्वा बत किं केनः ख."

**आत्मसम्भावनां चेयमास्तामसदृशक्रमा।  
कालेन व्यज्यते सर्वं कथं च [^479]मम सैनिकाः॥८९॥**

[^479]: # "'तु' क."

**इति चिन्तयतस्तस्य निद्रा सुचिरसम्भृता।  
चकार मुकुलाकारे चक्षुषी[^480] आ निशात्ययात्‌॥९०॥**

[^480]: # "'अक्षिणी' ख."

**अपरेद्युरथोत्थाय कृतप्राभाति[^481]कक्रमः।  
कादम्बरीमहाश्वेते दृष्ट्वा लब्ध्वा[^482] चिरादिव॥९१॥**

[^481]: # "'सन्ध्याविधिक्रमः' ख."

[^482]: # "'लब्धे' ख."

**तदनुज्ञां [^483]तदादिष्टगन्धर्वपरिवारितः।  
स इन्द्रायुधमारुह्य प्रतस्थे पृतनां प्रति॥९२॥(युगलकम्‌ )**

[^483]: # "'तदनुज्ञासमादिष्ट' ख."

**अथ केनाप्युपायेन लब्ध[^484]मार्गमसौ निजम्‌।  
सन्निविष्टं सरस्तीरे[^485] स्कन्धावारं व्यलोकयत्‌॥९३॥**

[^484]: # "'लब्ध्वा' क."

[^485]: # "'नदीतीरे' ख."

**तत्र[^486] दृष्ट्वा नृपान्सर्वान्मन्त्रिपुत्रं रहस्यसौ।  
स्ववृत्तान्तविदं चक्रे पत्त्रलेखां विशेषतः॥९४॥**

[^486]: # " 'तदा' ख."

**निशि तस्यामतीतायामन्येद्युरूदिते रवौ।  
केयूरकस्तमागत्य[^487] हेमकूटादभाषत[^488]॥९५॥**

[^487]: # "'समागत्य' ख."

[^488]: # "'अथाब्रवीत्' ख."

**कादम्बरी न[^489] पट्वीति त्वामभीक्ष्णं दिदृक्षते।  
विस्मृतं शयनीये च शेषहारमुपाहरत्‌॥९६॥**

[^489]: # "'नरपतेः' ख."

**केयुर[^490]कवचः श्रुत्वा समेतः[^491] पत्त्रलेखया।  
चन्द्रापीडः पुनस्तूर्णं हेमकूटमथागमत्‌॥९७॥**

[^490]: # "'केयूरवचनं' क."

[^491]: # "'सङ्गतः' ख."

**अभिवन्द्य[^492] महाश्वेतां श्लाघितागमनस्तया।  
कादम्बरीमथापश्यत्पत्त्रलेखापुरःसरः॥९८॥**

[^492]: # " 'अभिनन्द्य' क."

**उद्याननलिनीतीरकल्पिते[^493] हिममन्दिरे।  
कदलीदलशय्यान्तर्निषण्णां म्लानविग्रहाम्‌॥९९॥**

[^493]: # "'पार्श्वकल्पिते' ख."

**तदालोकनमात्रेण मुहुराश्वासितेव सा।  
तत उत्थातुमिच्छन्ती शयानैवास्त तद्गिरा॥१००॥**

**ततः कुशलसम्प्रश्नसुखमात्रपवित्रिताः।  
नानानर्मविलासार्द्राः प्रावर्तन्त तयोर्गिरः॥१०१॥**

**तथा[^494] तत्र चिरं स्थित्वा तां तथावशवर्तिनीम[^495]।  
भङ्गीभणितिभिर्व्यक्तमावेद्य निज[^496]माशयम्‌॥१०२॥**

[^494]: # " 'तत्रतत्र' ख."

[^495]: # "'वर्तिनम्‌' खं."

[^496]: # "'निरगादयम्‌' क."

**तदभ्यर्थनयादिश्य पत्त्रलेखां तदन्तिके।  
केयुरकद्वितीयोऽसौ जगाम कटकं पुनः॥१०३॥**

**तत्रोज्ज[^497]यिन्याः सम्प्राप्तं पूर्वेद्यु[^498]र्लेखहारकम्‌।  
अवलोक्य तदानीं तं लेखं स्वयमवाचयत्‌॥१०४॥**

[^497]: # "'ततोज्जयिन्याः' ख."

[^498]: # "'अन्येद्युः' ख."

**पितुराज्ञां ततो ज्ञात्वा शीघ्रागमनशंसिनीम्‌।  
केयूरककरे न्यस्य तं लेखं कूणितेक्षणः॥१०५॥**

**तदर्थमथ सम्मन्त्रय सह मन्त्रिसुतेन सः।  
एवमेतदिति ध्यायन्प्र[^499]स्थानस्थैर्यमग्रहीत्॥१०६॥**

[^499]: # "'ध्यात्वा प्रस्थाने' क."

**सत्यं वच्मि भवत्प्रसादरसिकं चेतो मभैतत्पदं[^500]  
गन्तुं नेच्छति किन्तु शासनमिदं नोल्लङ्घनीयं गुरोः।  
मन्युः सम्प्रति सह्यतां पुनरिहागन्तास्मि कादम्बरीं  
सन्दिश्येति सबाष्पगद्गदमसौ केयूरकं प्राहिणोत्‌॥१०७॥**

[^500]: # "'परं' खं."

**वैशम्पायनमभ्यभाषत भवान्संवाहयन्वाहिनीं  
पश्चादेष्यति यामि सत्वरमहं स्वल्पैरितो वाजिभिः।  
निश्चित्येति तदा[^501] प्रयाणपटहे दत्ते प्रतस्थे ततो  
गत्वा किञ्चन चण्डिकाश्रमतटीम[^502]ध्यास्त तस्मिन्दिने॥ १०८॥**

[^501]: # "'तथा' क."

[^502]: # "'महीं' ख."

**तत्रोद्धतद्रविडधार्मिकरूपवेष-  
भाषातपश्च[^503]रणसिद्धिकथाविनोदैः।  
नीत्वा निशां सततमस्खलितप्रयाणैः  
स्वल्पै[^504]र्दिनैस्त्वरितमुज्जयिनीमवाप॥१०९॥**

[^503]: # "'तपःप्रशमसिद्ध' क."

[^504]: # "'अल्पैः' क."

इति श्रीविषश्चिद्वराग्रगण्याचार्यभट्टश्रीजयन्तसूनोरभिनन्दस्य कृतौ कादम्बरीकथासारे  
पश्चमः सर्गः

**अथ प्रविश्य नगरीमुपास्य पितरौ चिरम्‌।  
शुकनासं च सोऽध्यास्त कुमारभवनं पुनः॥१॥**

**तत्र कादम्बरीध्याननिरतैकाग्रचेतसः।  
बभूव रतये नास्य[^505] किञ्चित्स्थास्नु चरिष्णु च॥२॥**

[^505]: # " 'किञ्चिन्नास्य' ख."

**नाकरोत्प्रियमाहारं नाबोधि मधुरं रसम्‌।  
न चाद्रियत वासांसि न[^506] सिषेवे विलेपनम्‌॥३॥**

[^506]: # " 'न च स्पर्शविलेपनम्‌' ख."

**सार्थच्युत इवा[^507]ध्वन्यो यूथभ्रष्ट इव द्विपः।  
वागुरापाशसंरुद्ध[^508] इव सारङ्गशावकः॥४॥**

[^507]: # "'अधन्यो' क."

[^508]: # "'सम्बद्ध' ख."

**दिवा विभावरीं ध्यायन्निशि वाञ्छन्दिनाग[^509]मम्‌।  
उदीचीमेव सास्राक्षो वीक्षमाणोऽनिशं दिशम्‌॥५॥**

[^509]: # "'दिवागमम्‌' ख."

**केयूरका[^510]दिहायातं ज्ञात्वा मां किं भजेदसौ ।  
किं वदेत्पत्त्रलेखां वा सापि किं वोच्यते तया॥६॥**

[^510]: # "'केयूरकोदितं मां च ज्ञात्वा किं चिन्तयेदसौ' ख."

**इति नक्तन्दिनं ध्यायन्ननवा[^511]पितनिर्वृतिः।  
कालं कियन्तमप्येवं चन्द्रापीडोऽत्यवाहयत्‌॥७॥**

[^511]: # " 'अलब्धधृतिबन्धनः' ख."

**आकारः[^512] संवृतस्त्वीदृगस्य गूढात्मनोऽभवत्‌।  
यथा तां शुन्यतां तस्य[^513] पितरौ नाधिजग्मतुः॥८॥**

[^512]: # "'आकारसंवृतिस्तादृग्' ख."

[^513]: # " 'अस्य' ख."

**दिनेष्वथ व्यतीतेषु पत्त्रलेखा तदन्तिकम्‌।  
प्रापद्बलाहकाश्वीयनिर्वृत्तपथसौ[^514]ष्ठवा॥९॥**

[^514]: # " 'पथि' क-ख."

**चन्द्रापीडोऽपि[^515] तां दृष्ट्वा समुच्छ्व[^516]सितमानसः।  
कादम्बरीमिवामंस्त पस्पर्श च मुहुर्मुहुः॥१०॥**

[^515]: # "'अथ' ख."

[^516]: # "'समुल्लासित' ख."

**अथ[^517] विस्रब्धमप्राक्षीद्वद कादम्बरीकथाम्‌।  
किमस्माकं स्मरत्येषां रुषिते[^518]हागमेन किम्‌॥११॥**

[^517]: # "'ततो' ख."

[^518]: # " 'रुषितागमनात्कियत' ख. "

**साभ्यसूयमिवालोक्य चन्द्रापीडमुवाच सा।  
किमेत[^519]च्चिन्तया देव गुरूनाराधय स्थिरम्‌॥१२॥**

[^519]: # " 'किं तया' ख."

**असौ हि[^520] सुप्रसन्नेन स्वयं मकरकेतूना।  
त्वय्यर्पिता विधिवशाच्छ्रीपतौ श्रीरिवाब्धिना॥१३॥**

[^520]: # "'किं' ख."

**तां[^521] चाभिनवभावार्द्रां बालां तरुल[^522]तामिव।  
तीव्रे विरहदावाग्नौ क्षिप्तवानसि किं वृथा[^523]॥१४॥**

[^521]: # " 'मां' ख."

[^522]: # "'तनु' ख."

[^523]: # " 'कृते' ख."

**केयूरकमुखाद्बुद्ध्वागमनं देव तावकम्‌।  
वज्रा[^524]शनिहता शालिवल्लरीव व्यकम्पत॥१५॥**

[^524]: # "'वज्रसन्निहिता साल–' ख."

**लब्धस्थैर्यावदन्मां च पत्त्रलेखे किमीदृशम्।  
युक्तमेतस्य यन्मन्ये रक्षांस्यपि न कुर्वते ॥१६॥**

**ततः प्रभृति निःश्वासैर्वर्धमानैर्दिनैरिव।  
क्रान्ता[^525] ग्रीष्मनिशेवासावक्षीयत शनैः शनैः॥१७॥**

[^525]: # "'क्लान्ता मीम' ख."

**अलब्धकुसुमामोदाः प्रागभ्या[^526]सादुपागताः।  
दूरात्परिहरन्त्यस्याः कबरीमद्य षट्पदाः॥१८॥**

[^526]: # "'अभ्यासक्रमागताः' ख."

**रोदनारुणितापाङ्गे चिरं विस्मृतकज्जले।  
नेत्रे माधुर्य[^527]मुत्सृज्य गते तस्याः कषायताम्‌॥१९॥**

[^527]: # "'विसृष्टमाधुर्ये' ख."

**चिरं[^528] पत्त्रलतालेख्यं[^529] विहाय क्रियते नवम्‌।  
मण्डनं गण्डयोरश्रुकणमुक्तावली तया॥२०॥**

[^528]: # "'चित्र–' ख."

[^529]: # "'लेख्ये' ख."

**मृत्युः स्वभावरक्तश्च प्रवृद्धै[^530]रसमोष्मभिः।  
म्लानिमानीयते तस्या निःश्वासैरधरः खरैः[^531]॥२१॥**

[^530]: # "'प्रवृद्धरससोष्मभिः' ख."

[^531]: # "'खलैः' खं."

**स्तनौ च[^532] विरहे तस्या शोकान्नहि बहिष्कृतौ।  
दाहप्रतिक्रियाभ्य[^533]स्तैर्हारकर्पूरचन्दनैः ॥२२॥**

[^532]: # "'तु विरहेऽप्यस्याः' ख."

[^533]: # "'न्यस्तैः' ख."

**विरहज्वलनस्तस्या[^534] लावण्यरस[^535]निर्झरैः।  
वाडवश्चाब्धिना नैति शममित्यद्भुतद्वयम्॥२३॥**

[^534]: # " 'नास्याः' क."

[^535]: # "'जल' ख."

**मुहुः सरसि हंसीव मृगीवोपवने मुहुः।  
पुलिने चक्रवाकीव मुहुर्लुठति विह्वला॥२४॥**

**मुहुर्भवन्तमालिङ्ग्यः पुरः[^536] सङ्कल्पकल्पितम्‌।  
शून्ये भुजलते वीक्ष्य विलक्षा पतति क्षितौ॥२५॥**

[^536]: # "'सङ्कल्पविहितं पुनः' ख."

**मा गादयमुपालब्धः सङ्कल्पादपि दूरतः।  
इत्युपालभते न त्वां भीतेव[^537] मनसापि सा॥२६॥**

[^537]: # " 'भीतेन' क."

**त्वदाकारमयं ध्यानं[^538] त्वद्गोत्रप्रवणा गिरः।  
त्वद्गात्र[^539]कीलिता दृष्टिस्तस्याः सम्प्रति वर्तते॥२७॥**

[^538]: # " 'चित्तं त्वन्नामस्मरणाः' ख."

[^539]: # "'त्वन्मार्गकल्पिता' ख."

**कियती[^540] वर्ण्यते तस्याः परेण विरहव्यथा।  
स्वसंवेदनगम्या हि सा तयैवानुभूयते॥२८॥**

[^540]: # "'कियद्वा' ख."

**किमप्याख्यातुकामेव बहुशः स्फुरिताधरा।  
हृदयस्थेन केनापि गुरुणेव निषिध्यते॥२९॥**

**भूमौ विवेष्ट[^541]मानाङ्गी भुजङ्गीव कदाप्य[^542]सौ।  
दिशो वीक्ष्य मदुत्सङ्गे निधाय मुखमब्रवीत्‌॥३०॥**

[^541]: # "'विचेष्ट' ख."

[^542]: # " 'एकदा पुरः' ख."

**उत्तिष्ठा[^543]नय तं गत्वा जीवन्तीं मां यदीच्छसि।  
अहं गन्तुम[^544]शक्ता हि सुदूरमिदमन्तरम्‌॥३१॥**

[^543]: # " 'उत्तिष्ठ यदि जीवन्तीं मामिच्छसि तमानय' ख."

[^544]: # "'नेतुमशक्या हि सुदुरमिव मन्दिरम्' ख."

**स्थितायां त्वयि जीवामि त्वत्स्पर्शेन यथा तथा।  
गतायां तु चिरं कालं न[^545] सहन्ते ममासवः॥३२॥**

[^545]: # " 'क्षमन्ते मम मासवः' ख."

**इति श्रुत्वा च [^546]दृष्ट्वा च समायातास्म्यतः परम्‌।  
देवः प्रमाणमित्युक्त्वा रुदती विरराम सा॥३३॥**

[^546]: # "'तु तां दृष्ट्वा' ख."

**चन्द्रापीडस्तदाकर्ण्य गत[^547]स्मृतिरिवाभवत्‌।  
लब्धस्मृति[^548]श्चिरं दध्यौ गमने हेतुमात्मनः॥३४॥**

[^547]: # "'जात' ख."

[^548]: # "'स्थितिः' ख."

**दिनेष्वथ[^549] व्यतीतेषु वक्तुं कादम्बरीदशाम्‌।  
आगात्केयूरकोऽप्याशु दीर्घाध्वश्रान्तवाहनः॥३५॥**

[^549]: # "'अन्येषु जातेषु' ख."

**स चोज्जयिन्या विभवं प्रविशन्वीक्ष्य विस्मितः।  
स्थाने ननु मनो लग्नं स्वामिन्या इत्यमन्यत॥ ३६॥**

**प्रविश्य ददृशे तेन चन्द्रापीडः कृशा[^550]कृतिः।  
चक्रवाक इवैकाकी प्रियाविरहविह्वलः॥३७॥**

[^550]: # " 'यथाक्रमम्‌' ख."

**दधत्या पार्श्चवर्तिन्या चामरं पत्त्रलेखया।  
संवीज्यमानः पर्यङ्क[^551]निषण्णग्लानविग्रहः॥३८॥**

[^551]: # " 'पर्यङ्के निषण्णः क्षाम' ख."

**प्रणिपत्योपविष्टोऽसौ गाढं शिरसि पस्पृशे।  
कुर्वता कुशलप्रश्नं चन्द्रापीडेन सादरम्‌॥३९॥**

**ततः कादम्बरीवार्तां पृष्ठः केयूरकोऽमुना।  
पत्त्रलेखामुखं वीक्ष्य सङ्क्षिप्तमिदमब्रवीत्‌॥४०॥**

**उक्तोऽसि नानया किञ्चिदुक्तश्चेत्किमिह स्थितः[^552]।  
उक्तश्चेह स्थितश्चासि कृतमागमनेन मे॥४१॥**

[^552]: # "'आस्थितः' ख."

**तथापि वद सन्देशमिति तेन तु चोदितः[^553]।  
पुनः केयूरकः प्राह[^554] नहि विज्ञप्यसे[^555] मृषा॥४२॥**

[^553]: # "'नोदितः' ख."

[^554]: # "'अथाह' ख."

[^555]: # "'विज्ञायसे' ख."

**कथं जीवत्सु युष्मासु विरहो हन्ति मामिति।  
तस्याः साकूतया दृष्ट्या विसृष्टोऽस्मि न तद्गिरा॥४३॥**

**किमत्र बहुनोक्तेन तथास्या विरहव्यथा।  
अद्यापि जीवतीत्यत्र[^556] यथा दोलायते मनः॥४४॥**

[^556]: # "'एवं' ख."

**तत्कथाभिस्तथा स्थित्वा तस्मिन्नथ दिने चिरम्‌।  
चन्द्रापीडः सुखावासं केयूरकमकारयत्‌॥४५॥**

**न राक्षसो न चाण्डा[^557]लो न किरातश्च यद्यहम्।  
अच्छोदसरसस्तीरं प्राप्तं मां वेत्थ सर्वथा॥४६॥**

[^557]: # "'मातङ्गो ख."

**अन्येद्युरिति सन्दिश्य सहितं पत्त्रलेखया।  
केयूरकमसौ रत्नैः[^558] प्राहिणोत्कृतमण्डनम्‌॥४७॥**

[^558]: # " 'अग्रे' ख."

**तयोस्तु गतयोरास्त सुतरां खिन्नमानसः।  
गमनाय[^559] धिया किञ्चिन्निमित्तमनुसन्दधत्‌॥४८॥**

[^559]: # " 'तांस्तानभिलषन्बुद्ध्या हेतुन्गमनसाधकान्' ख."

**शुश्रावाथ नियुक्तेभ्यः स्कन्धावारमुपागतम्‌।  
योजनद्वयमात्रं[^560] च पुनः कृतनिवेशनम्‌॥४९॥**

[^560]: # "'मात्रेण पुरः' ख. "

**वैशम्पायनतो लप्स्ये किञ्चित्कारणमित्यसौ ।  
प्रत्युद्गन्तु[^561]मगादेनं कटकं पितुराज्ञया॥५०॥**

[^561]: # "'प्रत्युद्गतं' ख."

**तत्र राजकमालोक्य बलाहकनिवेदितम्‌।  
क्व वैशम्पायनावास इति सेनान्यमभ्यघात्‌॥५१॥**

**बलाहकस्त्वघोदृष्टिरुल्लिखन्मेदिनीं मुहुः।  
न वैशम्पायनोऽत्रेति[^562] सन्नकण्ठमभाषत॥५२॥**

[^562]: # "'अस्ति' ख."

**ततोऽवतीर्य तुरगादपविश्य तरोरधः।  
हा वयस्येति विलपन्प्रावर्तत स रोदितुम्‌॥५३॥**

**बलाहको निपत्याथ चन्द्रापीडस्य पादयोः।  
देव भैवं कृथाः सोऽस्ति[^563] जीवन्नेवेत्यवोचत॥५४॥**

[^563]: # "'शोचं जीकन्नस्तीत्यभाषत' ख."

**चन्द्रापीडस्तु तं प्राह नैव[^564] श्रद्धीयते मया।  
न [^565]वैशम्पायनो जीवन्मुहूर्तं मां विमुञ्चति॥५५॥**

[^564]: # "'नेन्द' ख."

[^565]: # "'स वैशम्पायनो जीवन् त्यक्त्वालम्बत मामिति ख."

**इति सम्भ्रान्त[^566]चित्तं तं पुनराह बलाहकः।  
श्रूयतां देव तद्वृत्तं[^567] यथासाविह नागतः॥५६॥**

[^566]: # " 'उपप्लुतचित्तं' ख."

[^567]: # " 'यद्वृत्तं तत्र प्रचलिते त्वयि' खं."

**देव[^568] तादृशमादिश्य प्रस्थितं त्वामनुव्रजन्‌।  
ततः प्रतिनिवृत्तोऽसौ तत्रैवा[^569]स्त यथास्थिति॥ ५७॥**

[^568]: # "'अस्मानेव समादिश्य' ख."

[^569]: # " 'आस्त तत्रैव' ख."

**अपरेद्युरथोत्थाय विधाय[^570] प्रकृताः क्रियाः।  
विनोदयितुमात्मानं स[^571] चचार वनस्थलीः॥५८॥**

[^570]: # "'विहायः' क."

[^571]: # "'व्यचरत्स' ख."

**तत्र सप्रत्य[^572]भिज्ञानं स्थानं स्थानं निरूपयन्‌।  
किञ्चिन्नष्टमिवान्वेष्टुमैच्छदुत्त्रस्त[^573]लोचनः॥५९॥**

[^572]: # " 'सप्रत्यभिज्ञः सन्स्थानान्यनुरूपयन्' ख."

[^573]: # "'ऐक्षद्वित्रस्त' ख."

**ततोऽस्ताभिमुखे भानावस्माभिः सोऽभ्यधीयत।  
स्थीयतामद्य[^574] शिष्टास्तु श्वो द्रष्टाऽसि वनस्थलीः॥६०॥**

[^574]: # "'अवशिष्टास्तु ता द्रक्ष्यामो' ख."

**अथासौ[^575] विचरन्नेव साभ्यसूयमिवावदत्‌।  
यात यूयं न गन्तव्यं यावज्जीवमितो मया॥६१॥**

[^575]: # " 'तदासौ नाग्रहीद्वाक्यं विचरन्नेव चाब्रवीत्' ख."

**स बहूनि दिनान्यास्त तत्रैव[^576] विचरन्वने।  
न[^577] च केनाप्युपायेन शक्तश्चलयितुं ततः॥६२॥**

[^576]: # " 'तथैव' ख."

[^577]: # " 'न सोऽस्त्युपायो येनासौ शक्य आनयितुं ततः' ख."

**कृतप्रायोपवेशेन पादयोः पततानिशम्[^578]।  
इयता राजचक्रेण नार्थितो[^579]ऽप्यत्यजद्ग्रहम्॥६३॥**

[^578]: # "'भृशम्‌' ख."

[^579]: # "'नाबुद्ध प्रार्थितोऽपि सः' ख."

**ततस्तत्र चिरं स्थित्वा कार्यशेषमपश्य[^580]ता।  
आयातमिह सैन्येन त्वं प्रमाणमतः[^581] परम्॥६४॥**

[^580]: # "'अपश्यत' क."

[^581]: # "'ततः' ख."

**इति श्रुत्वा निपतितः शोकविस्मयसङ्कटे।  
चन्द्रापीडस्तथैवास्त तरुमूले कृतास[^582]नः॥६५॥**

[^582]: # "'वृताननः' क."

**अथ वार्तां विदित्वेमां[^583] सामात्या[^584]न्तःपुरो नृपः।  
अनपेक्षितवृत्तान्त[^585]स्तमेवोद्देशमागमत्‌॥६६॥**

[^583]: # " 'एनां' ख."

[^584]: # "स्व-पुस्तकेऽस्मिञ्श्लोके द्वितीयचदुर्थपादयोर्व्यत्ययः."

[^585]: # "'विस्तारः' ख."

**तत्र क्षणमिव स्थित्वा स्वयं पृष्ट्वा[^586] बलाहकम्।  
अधिक्षिपन्निव प्राह चन्द्रापीडं क्षितीश्वरः॥६७॥**

[^586]: # " 'दृष्ट्वा' क."

**नूनमुत्पादितो मन्युर[^587]विनीत त्वयास्य सः।  
येनान्तःस्फुरता सोऽद्य त्वां विहाय वनं श्रितः[^588]॥६८॥**

[^587]: # "'त्वया कोऽप्यस्य तादृशः' ख."

[^588]: # "'गतः' ख."

**वियुक्तौ न पुरा दृष्टौ युवां क्षणमपि क्वचित्‌।  
किं जातमधुना येन त्वं गृहे स तपोवने॥६९॥**

**विना नहि निमित्तेन शक्या सोढं गरीयसी।  
मातापितृसुहृद्बन्धुदेशवित्तवियोगिता॥७०॥**

**इत्युक्तवति भूपाले शुकनासस्ततः[^589] परम्‌।  
अन्तः स्फुरितशोकोऽपि स्तम्भितक्षोभमभ्यधात्‌॥७१॥**

[^589]: # "'त्वनन्तरम्‌' ख."

**किमेतदविचार्यैव वस्तुतत्त्वमतर्कयत्‌।  
शङ्कानुमानमात्रेण कृतधी[^590]रपि भाषसे॥७२॥**

[^590]: # "'हतधीरिवः' ख."

**अलातवृष्टिरिन्दोर्वा क्षीराद्वा कण्टको[^591]द्गमः।  
तस्य मन्युः कुमाराद्वा कस्य[^592] सम्भावनास्पदम्‌॥७३॥**

[^591]: # "'कण्टको मुहुः' क."

[^592]: # "'तव' ख."

**पापः स एव दुर्जातः सुचिरं पाठितः शुकः।  
यत्रामृत[^593]रसोऽप्युप्तः फलितः कालकूटवत्‌॥७४॥**

[^593]: # "यथा तरुः सुधासिक्तः' ख."

**स त्वं कुमार सर्वस्याः पृथिव्या एकजीवितम्।  
दुरात्मनो न तस्यार्थे शोकमुद्वोढुमर्हसि॥७५॥**

**तस्मिन्विरतवाक्ये तु पतित्वा पादयोः पितुः।  
साश्रुकण्ठमथोवाच चन्द्रापीडः कृताञ्जलिः॥७६॥**

**तात[^594] शङ्कितदोषस्य प्रायश्चित्तमिदं मम।  
गत्वा तमानयामीहः[^595] त्वमनुज्ञातुमर्हसि॥७७॥**

[^594]: # "'दोषप्रायश्चित्तमिदं मम साधनमेव तव्‌' ख."

[^595]: # "'इति तदनु-' खं."

**तमागतमिहालोक्य पृष्ट्वा शपथ[^596]पूर्वकम।  
ततो गुणं वा दोषं वा क्रमेण[^597] मम वेत्स्यसि॥७८॥**

[^596]: # "'शिरसि चुम्बितम्‌' ख."

[^597]: # "'मम तातोऽवभोत्स्यते' ख."

**तत्रस्थं तमना[^598]नीय मयैतत्स्वगृहं पुनः।  
न[^599] प्रवेष्टव्यमित्येष गृह्यतां[^600] मम निश्चयः॥७९॥**

[^598]: # " 'च तमानीय व्यपेताश्वं गृहं' ख."

[^599]: # "'सम्प्रवेष्टव्यं' ख."

[^600]: # "'गृहीतोमेति' ख."

**तथोक्तवति तस्मिंस्तु[^601] राजा बाष्पाकुलेक्षणः।  
अपश्य[^602]न्मन्त्रिणो वक्त्रं स च तत्र बहूक्तवान्‌॥८०॥**

[^601]: # "'च' ख."

[^602]: # "'सोऽपश्यत्' ख."

**सर्वोपायप्रयोगेण पित्रा तेन च मन्त्रिणा।  
चन्द्रापीडो निरुद्धोऽपि नाचलन्निश्चयात्ततः॥८१॥**

**सिञ्चन्ती बाष्पधाराभिर्मौलिमस्य मनोरमा।  
करुणं विलपन्ती च चिरमेनमबोधयत्‌॥८२॥**

**त्वयि स्थिते निजोत्सङ्गस्थि[^603]तं पश्यामि पुत्रकम्‌[^604]।  
नूनमन्धीभवाम्येव त्वयि पुत्र[^605] गते पुनः॥८३॥**

[^603]: # "'उत्सङ्गे स्थितिं' ख."

[^604]: # "'पुत्रक' ख."

[^605]: # "'वत्स' ख."

**इत्थं विलासवत्या च तया च करुणोक्तिभिः ।  
चिरमार्द्रीकृतोऽप्येष न जहाति स्म निश्चयम्‌॥८४॥**

**ततोऽश्रुभरसंरुद्धगद्गदा[^606]क्षरया गिरा।  
यथा जानासि पुत्रेति कथमप्यव[^607]दन्नृपः॥८५॥**

[^606]: # "'दुर्ग्रहाक्षरया' ख."

[^607]: # "'अब्रवीत्' ख."

**दत्तानुज्ञस्ततः पित्रा प्रणम्य सकलान्गुरून्।  
श्रुत्वा च[^608] यत्नसंरुद्धबाष्पधारास्तदाशिषः॥८६॥**

[^608]: # "'प्रयत्न' ख."

**चन्द्रापीडः पथि[^609] श्रान्तान्विनिवर्त्य यथा तथा॥  
पार्थिवान्परिवारेण प्रातिष्ठत लघीयसा॥८७॥**

[^609]: # "'परिश्रान्तान्संस्थाप्य च यथातथम्' ख."

**किं तन्मयाभ्यधायीति दूयमानमनाः शुचा।  
सेनया[^610] सह राजापि विवेश नगरीं शनैः॥८८॥**

[^610]: # "'शुष्यन्कृच्छ्रेण राजा तु विवेश नगरीं पुनः' ख."

**बलाहकस्तु सेनानीर्वार्यमाणोऽपि भूभुजा।**  
**स्वगृहानप्रविश्यैव स चन्द्रापीडमन्वगात्‌॥८९॥**

**चन्द्रापीडस्ततो गच्छन्दयितादर्शनोत्सुकः।  
अमन्यत तृणायैव[^611] मध्यं कैलासविन्ध्ययोः॥९०॥**

[^611]: # "उल्लङ्घितवत्” खं, "

**नाजीगणद्द्रुमानद्रीन्न नदीर्न वनस्थलीः।  
न रात्रिं नापि[^612] दिवसं धावन्निन्द्रायुधेन सः॥९१॥**

[^612]: # " “न दिनं वापि ख"

**ततः प्रावर्तत प्रावृट् पीनोन्नतपयोधरा।  
कान्ता नवरसार्द्रेव प्रत्युद्गन्तुमिवागता[^613]॥९२॥**

[^613]: # "“आगतम्‌” ख,"

**तामप्यगणयित्वासौ व्रजन्कतिपयैर्दिनैः।  
अच्छोदसरसस्तीरमवा[^614]पाकलुषाम्भसः॥९३॥**

[^614]: # " “आपतन्द्भूषिताम्भसः ख."

**अपनीतसमायोगः परिवर्त्य च वाससी।  
महाश्वेतामथ द्रष्टुं स जगाम तदाश्रमम्‌॥९४॥**

**तत्रापश्यन्महाश्वेतां तदालरोकनलज्जिताम्‌।  
वदन्तीं[^615] निजदौरत्म्यं प्रलापकरुणस्वनाम्‌[^616]॥९५॥**

[^615]: # " “रुदती निजदौरात्म्यात् ख."

[^616]: # " “स्वरम्‌ ख"

**तत्कारणापरिज्ञानमोहनिष्पन्दपक्ष्मणा।  
निरीक्ष्य चक्षुषा पृष्ठा चन्द्रापीडेन साब्रवीत्॥९६॥**

**केयूरकमुखाद्बुद्ध्वागतं तां पितुराज्ञया।  
दिनानि कतिचित्स्थित्वा कादम्बर्या[^617] निकेतने॥९७॥**

[^617]: # "कादम्बर्य्यनुरोधतः “ख,"

**तत एत्य यथान्यायमासीनात्र[^618] तपोवने।  
विप्रं युवानमद्राक्षं विचिन्वन्तं वनस्थलीः॥९८॥(युगलकम् )**

[^618]: # "अस्मि ख."

**सप्रत्यभिज्ञमालोक्य मामसौ मन्मथोन्मुखः[^619]।  
तीव्रमुद्वेजयामास नर्मवक्रोक्तिचापलैः॥९९॥**

[^619]: # "'मन्मथोत्सुकः' ख"

**ततो निर्भर्त्स्यमानोऽपि वार्यमाणोऽपि यत्नतः।  
स बहूनि दिनान्यास्त तथा मामुपपीडयन्‌॥१००॥**

**एकदा तु शयानां मामागत्य निशि निस्त्रपः।  
कृताञ्जलिरनङ्गार्तो ययाचे रतिदक्षिणाम्‌॥१०१॥**

**ततो धिगिति सक्रोधमभिधाय कमण्डलोः।  
जल्मुद्धृत्य शप्तोऽसौ मयानुध्याय देवताः॥१०२॥**

**शुकवन्मुखरागेण वक्तुमेवासि[^620] शिक्षितः।  
गच्छ दूरं दुराचार शुक एव भविष्यसि॥१०३॥**

[^620]: # "' एषोऽपि' ख."

**शापेन तेन महता मदनाग्निना वा  
जन्मान्तरोपचितदुष्कृतकर्मणा वा।**

**चित्रेण वा भगवतश्चरितेन धातुः  
प्राणानसावुदसृजन्निपपात्त चोर्व्याम्‌॥१०४॥**

**आक्रन्दतः परिजनात्सशुचो[^621]ऽपरेद्यु-  
र्ज्ञातो मया स शुकनाससुतः सखा ते।**

[^621]: # "अथ सोपरेद्युः ख."

**तत्पश्य मां प्रियवयसस्यवियोगहेतु- \`  
मस्तत्रपामकरुणां प्रखलामभव्याम्‌[^622]॥१०५॥**

[^622]: # "अभाग्यां ख."

**इति श्रुत्वा भिन्नः प्रसभमिव शूलाग्रशिखया  
कदा त्वां पश्येयं कमलमुखि कादम्बरि पुनः।**

**वदन्नेवं[^623] शोकस्फुटितहृदयस्तत्क्षणमसू-  
ञ्जहच्चन्द्रापीडस्तरुरिव पपात क्षितितले॥१०६॥**

[^623]: # "क्वचिन्नैवं ख, "

**ततो भूग्नग्रीव॑[^624] मुकुकितसितापाङ्गनयनं  
प्रभां तन्वीं दीनामिव विदधतं दिक्षु शनकैः।**

[^624]: # "भग्नग्रीवं क"

**क्षितौ विस्रस्ताङ्गं पतितममुमालोक्य विवशा  
महाश्वेता सद्यः कथमपि न याति स्म दलशः॥१०७॥**

**द्रष्टुं दीनामिव वसुमतीमक्षमस्तेन शन्यां  
कुर्वन्नुद्य[^625]द्विरहकरुणाक्रन्दिनश्चक्रवाकान्‌।**

[^625]: # "'उद्यन्' क."

**अब्रह्मण्यध्वनिमिव करैरुर्ध्वशाखैर्विमुञ्च-  
न्नस्तं तिग्मद्युतिरपिं ततः क्षामधामां जगाम॥१०८॥**

**इति श्रीविपश्चिद्वराग्रगण्याचार्यश्रीभट्टजयन्तसूनोरभिनन्दस्य कतौ कादम्बरीकथासारे  
षष्ठः सर्गः।  
————————————**



**तसिन्नवसरे बाला कादम्बर्यपि वेधसा।  
तत्रैवानीयतानेकमायावैचित्र्यकारिणा॥१॥**

**असौ हि पत्रलेखाया सुखात्केयूरकस्य च।  
असंशयं[^626] समायातं चन्द्रापीडमबुध्यत[^627]॥२॥**

[^626]: # "मुक्तासंश्यमायान्तं ख, "

[^627]: # "व्यबुध्यतः ख,"

**ततः सकरुविस्रम्भयोग्यं विगतयन्त्रणम्‌।  
महाश्वेताश्रमं[^628] प्राप्तं प्रत्युद्गन्तुमियेष तम्‌॥३॥**

[^628]: # "आश्रमोपान्त' ख,"

**वैशम्पायन विध्वंसवृत्तान्तसमनन्तरम्‌।  
चन्द्रापीडमनःपीडां वयस्यविरहयोद्भवाम्‌॥४॥**

**शङ्कमाना महाश्वेता तत्समाश्वास्कारणम्‌।  
कादम्बरीं परिज्ञाय[^629] तदागमनसत्वरा॥५॥**

[^629]: # "“समर्थ्याय तदायनः ख. "

**तस्यास्तरलिकां[^630] गेहं शीघ्रमागम्यतामिति।  
सन्दिश्य प्राहिणोत्सा च[^631] तूर्णं गत्वा तथावदत्‌॥६॥(तिलकम्)**

[^630]: # "सखी ख."

[^631]: # "अथः ख,"

**जातोत्साहा महाश्वेतासन्देशेन गरीयसीम[^632]्।  
गमनोस्कलिकां दध्रे बुद्धौ कादम्बरी ततः॥७॥**

[^632]: # "गरीयसा ख,"

**ताम्बूलपटवासादिसंविधान[^633]विदग्धया।  
हृदयप्रख्यया सख्या सहिता मदलेखया॥८॥**

[^633]: # "'संविदानविधिज्ञयाः ख."

**पत्रलेखां पुरस्कृत्य प्रियचर्चा[^634]विचक्षणाम्‌।  
केयूरकं[^635] च तत्रैव दिने तां भुवमाययौ॥९॥**

[^634]: # "चर्या ख."

[^635]: # "केयूरकोऽपि' ख."

**आयान्त्याश्च तदा तस्या दुर्निमित्तपरम्परा।  
प्रादुर्बभूव बालाया हृदयोत्कम्पकारिणी॥१०॥**

**महाश्वेताश्रमोपान्तमापदस्तं गते रवौ।  
उदिते च निशानाथे सन्तप्तकरुणाह्रदे॥११॥**

**आसाद्य च तमुद्देशं करुणं रोदनध्वनिम्‌।  
वनान्तकन्दरोद्भूतप्रतिशब्दमथाशृणोत्‌॥१२॥**

**मन्ये सैव महाश्वेतावर्त्मनीयमुपस्थिता।  
इत्यसौ शङ्कमानेव विवेश द्रुतमाश्रमम्‌॥१३॥**

**ततः प्रविश्य सापश्यत्प्राणनाथं तथास्थितम्‌।  
महाश्वेतां च[^636] तामार्तनिनादं रुदतीं भृशम्‌॥१४॥**

[^636]: # "तु” ख,"

**तामुद्वीक्ष्य[^637] महाश्वेतां विकसच्छोकविह्वला।  
नितान्तताडितोरस्कं[^638] रुदती मोहमागमत्‌॥१५॥**

[^637]: # "महाश्वेतां तु सा वीक्ष्य विकसच्छोकविह्वलाम्"

[^638]: # " “उरस्काः ख, "

**पत्रलेखापि तद्रूपचन्द्रापीडविलोकिनी।  
मुमोच[^639] सहसा प्रज्ञां निपतन्ती शिलातले॥१६॥**

[^639]: # " “मुमुचे सञ्ज्ञया सद्यः ख"

**कादम्बर्या तु[^640] धैर्येण पृष्टा तरलिका तदा।  
आचचक्षे यथावृत्तं सर्वं तन्मरणक्रमम्‌[^641]॥१७॥**

[^640]: # "अथ ख,"

[^641]: # "तं ख."

**तदाकण्ये विहायैव वेक्लव्यं स्त्रीजनोचितम्‌।  
लब्धसञ्ज्ञां[^642] महाश्वेतां मदलेखां च सावदत[^643]्॥१८॥**

[^642]: # " 'लब्धसञ्ज्ञा” ख."

[^643]: # "साब्रवीत् ख"

**सन्ततस्थूलधारेण चिरात्प्रभृति रोहता।  
खिन्नाहमधुनानेन[^644] रोदनेन गरीयसा॥१९॥**

[^644]: # "दग्धरोदनेनः. ख, "

**यथायोग्येन गच्छामि नहि शक्नोमि रोदितुम्‌।  
सखि[^645] मे दर्शितः पन्था नाथेनैव महात्मना॥२०॥**

[^645]: # "'स च मे दर्शितोऽनेनः खं,"

**अयं सुहृद्वियोगेन नारोदीद्विलाप वा।  
प्राणानेव प्रियानौज्झीत्स पन्थाः क्व गतो मम॥२१॥**

**युक्तं सखि मदाश्वेते तव प्राणितुमीदृश[^646]म्।  
जीविता देवताभिस्त्वं पुनसतत्सङ्गमाशया॥२२॥**

[^646]: # " “जीवितुं ख."

**अहं सर्वप्रकारं[^647] तु विधिभग्नमनोरथा।  
करोमि दुर्भगैः प्राणैः; कि कल्याणबहिष्कृता॥२३॥**

[^647]: # "“प्रकारेणः ख,"

**मदलेखे[^648] तथा कुर्यास्तातोऽम्बा च कुले स्थितौ।  
मद्वियोगोपतपेन यथा न धृतिमुज्झतः॥२४॥**

[^648]: # "मद्वियोगोपतापेन तातो वासगृहे चिरम्‌। मदलेखे तथा कुर्या यथा न धृतिमुज्झति ख,"

**स्वयमेवासितव्यं[^649] तु मद्वासभवने[^650] त्वया।  
यथा मणिगवाक्षेषु शेरते नोर्णनाभयः॥२७॥**

[^649]: # " अधिशय्यते ख."

[^650]: # "भवनं ख,"

**कुमारस्तत्र चित्रस्थो मयैव लिखितः स्वयम्‌।  
रक्षणीयो यथा नास्तमेति[^651] चित्रस्थितोऽप्यसौ॥२६॥**

[^651]: # "नान्तर्धत्ते सङ्खि ततो. वप्यसौ"

**शयनीयशिरोभागे[^652] रत्नभित्तिनिवेशितः।  
कामदेवपटः क्वापि निक्षिप्तव्यो वृथार्चितः॥२७॥**

[^652]: # "“भागरत्न-ख."

**क्रीडापर्वतको देयः प्रशान्ताय तपस्विने।  
तत्रोषितं कुमारेण नहि सोऽन्य शोभते॥२८॥**

**दीर्घिकाराममध्ये मे पद्धिनीखण्डभण्डिता[^653]।  
तत्रास्ते कृतवास्तव्यं राजहंसकदम्बकम्‌॥२९॥**

[^653]: # "मण्डितं ख."

**धारायन्त्रै प्रपूर्णा च क्रीडावापी तदन्तिके।  
स्नेहान्मया स्वयं सिक्तमस्ति लीलावनं नवम्‌॥३०॥**

**तस्मिन्कृतनवोद्वाहा बकुले भालती[^654] मया।  
मुक्तः कङ्कणको नास्या मोचनीयस्त्यैव सः॥२१॥**

[^654]: # " 'मालसीलता” ख"

**मया पुत्रीकृतश्चासावशोकतरुबालकः।  
वर्धनीयः स्वहस्ताम्बुसेकैः स्तन्यपयोनिभैः॥३२॥**

**गृहे मुहुरपश्यन्ती मामसौ चपलेक्षणा[^655]।  
नूनं हरिणिका चास्ते[^656] प्रापयेस्तां[^657] च काननम्‌॥३३॥**

[^655]: # "चट्वलेक्षणा” ख. "

[^656]: # "यास्ते ख."

[^657]: # "'नानयेः” क."

**शुकः स परिहासो मे कालिन्दी सा च शारिका।  
यथा न कलहायेते तथा कुर्याः प्रतिक्षणम्‌॥३४॥**

**नित्यमेव मदुत्सङ्गलालिता नकुलाङ्गना।  
स्वाङ्के शाययितव्यासौ यथा याति न दूरतः॥३५॥**

**किं[^658] कार्यमथवा गेहे चिन्तया दीर्घयानया।  
सम्पादय चितामाशु प्रज्वलज्जातवेदसम्‌[^659]॥३६॥**

[^658]: # " कृतं मे गृहकारुण्यचिन्तया ख."

[^659]: # " वेदसः ख."

**केयूरक चिरं कि मामतिदीनाम[^660]वेक्षसे।  
उत्तिष्ठ चिनु काष्ठानि समाहर हुताशनम्‌॥३७॥**

[^660]: # " 'दीनां दीनमिवेक्षसे' ख,"

**लावण्याम्बुमयो देहः क्रियते कथमग्निसात्‌।  
कुमारममुमादाय[^661] पतामि वरमम्भसि॥३८॥**

[^661]: # "कुमारस्य प्रसादाय ख."

**कुमारे मित्रदःखेन तत्क्षणोज्झितजीविते।  
इयन्तमपि[^662] जीवन्ती कालं लज्जेऽहमात्मना[^663]॥३९॥**

[^662]: # " 'इमतीमपि जीवन्ती ख."

[^663]: # " “आत्मनः ख."

**द्रष्टव्यसि कदेत्येवमभिधायैव मां गतः।  
सेयं चिरयतीत्येवमधुनाभिदधीत सः॥४०॥**

**इति कादम्बरीं[^664] सद्यो मरणव्यवसायिनीम्‌[^665]।  
शिशिरज्योतिरालोक्य[^666] कम्पमान इवाभवत्‌[^667]॥४१॥**

[^664]: # " 'कादम्बरी' ख, "

[^665]: # " व्यवसायिनी' ख."

[^666]: # " “अप्याशु' ख. "

[^667]: # "उदभूत् ख,"

**येन दुग्धाब्धिकल्लोलप्लाव्यमान इवाभवत्‌।  
प्रवृत्तदन्तवीणश्च[^250] सकल कन्यकाजनः॥४२॥**

**अथ दिव्याकृतिः कोऽपि पुरूषो गगनस्थितः[^669]।  
गर्जद्घनघटाधीरगम्भीरध्वनिरभ्यधात्‌॥४३॥**

[^669]: # " गगने ख."

**त्वं हि वत्से महाश्वेते पूर्वमाश्वासिता मया।  
स्थितो[^670] मदन्तिकेऽद्यापि पुण्डरीकस्तव प्रियः॥४४॥**

[^670]: # "मदन्तिके स्थितः ख,"

**चन्द्रापीडस्त्वयं कञ्चित्कालं निद्रायमाणवत्‌[^671]।  
अत्रैवाविचलत्कान्तिरास्तां कादम्बरीप्रियः॥४५॥**

[^671]: # "सुप्त इव स्फुटम्‌ ख."

**अनेन हि पुनर्लब्धनिरपायस्थिरायुषा।  
गाढकण्ठग्रहानन्दः कादम्बर्याः करिष्यते॥४६॥**

**भवत्योरयमेवास्तु चित्तसम्प्रत्ययो[^672] दृढः।  
नस्प्रक्ष्यति विकारो[^673] यदेनं भूतस्वभावजः॥४७॥**

[^672]: # " “चित्तेः ख,"

[^673]: # "विकारोऽयं यदेन भूतभावनः ख."

**इत्युक्त्वा ज्योतिषा तेन सहसैव तिरोदधे।  
सविस्मयस्तदा चाभूदखिलः कन्यकाजनः॥४८॥**

**पत्रलेखा च[^674] तेनैव ज्योतिषो वचनेन सा[^675]।  
लब्धसञ्ज्ञा[^676] समुत्थाय गत्वा च[^677] त्वरितैः पदेः॥४९॥**

[^674]: # " 'तु' ख."

[^675]: # "“च ख."

[^676]: # "'लब्ध्चा चैतन्यमुत्थायः ख, "

[^677]: # "द्रुततरैः ख."

**तमिन्द्रायुधनामानमश्वमादाय[^678] केसरात्‌।  
तेनैव सहिता[^679]गाधमविशन्मध्यमम्भसः॥५०॥**

[^678]: # "आरुह्म केसरैः ख"

[^679]: # "सहसागाधं प्रविष्टा ख"

**शङ्कितेव च[^680] तां वेगयातां तरलिकान्वगात्‌।  
सा ददर्श पयोमध्यादुद्गच्छन्तं कपिञ्चलम्‌॥५१॥**

[^680]: # "अथ ख."

**प्रसृत्य व्याचचक्षे[^681] ह यथालोकितमद्भूतम्‌।  
तेनापि विस्मयस्तासां कन्यानां[^682] ववृधेतराम्॥५२॥**

[^681]: # " च ख,"

[^682]: # "अवर्धततरां ततः ,"

**अथ बिभ्रज्जटास्तन्वीः स्रुताम्बु[^683]कणदन्तुराः।  
जलार्द्रवल्कल[^684]स्ताभिरदृश्यत कपिञ्जलः॥५३॥**

[^683]: # " च्युताम्बुकणविप्रुषः ख,"

[^684]: # " स्नानर्द्र ख, "

**स्वयमेवोपसृत्यासौ[^685] महाश्वेतामभाषत।  
राजपुत्र्यपि नामायं प्रत्यभिज्ञायते जनः॥५४॥**

[^685]: # "उपविश्याथ ख."

**अहं कपिञ्जलस्तस्य वयस्यो दयितस्य ते।  
चिरागमनदौरात्म्यादुपाकभ्योऽस्मि न[^686] त्वया॥५५॥**

[^686]: # " “च ` ख, "

**कादम्बर्या त्वया चैताः[^687] श्रुताश्चन्द्रमसो गिरः।  
मदीयमपि वृत्तान्तमधुना शृणुतं युवाम्‌॥५६॥**

[^687]: # " इमाः. ख,"

**अप्युत्क्षिप्य तथा[^688] नीते वयस्ये तेन तत्क्षणम्‌।  
अहं[^689] गगनमुत्प्लुत्य[^690] तमेवानुसरन्नगाम्‌[^691]॥५७॥**

[^688]: # " ततो” ख"

[^689]: # " ततो ख."

[^690]: # "'उत्पत्य ख."

[^691]: # "अहं ख."

**ततोऽसौ दूरमुत्प्लुत्य[^692] प्रविश्य शशिमण्डलम।  
पश्चादेव[^693] प्रविष्टं मामवलोक्येदमब्रवीत्‌॥५८॥**

[^692]: # "'उत्सृज्य ख"

[^693]: # " अनन्तरं ख."

**कपिञ्जल मृगाङ्क मां विद्धि सोऽहं तवामुना[^694]।  
वयस्येन वृथा शप्तो हेमकूटगते[^695] त्वयि॥५९॥**

[^694]: # " तवात्मजः ख. "

[^695]: # "हेमकूटं ख"

**वोढव्या लोकयात्रा हि लोकपालतया मया।  
उदितोऽहं निजे काले ततः[^696] प्रविकिरन्करान्॥६०॥**

[^696]: # "यथाकालमतः ख."

**अयं मदंशुसंस्पर्श[^697]प्रवृद्धाधिकमन्मथः।  
कण्ठोपान्तगलप्राणो[^698] रुक्षं मां वीक्ष्य शप्तवान्‌॥६१॥**

[^697]: # " सम्पन्न कः,"

[^698]: # " चलत्प्राणो ख,"

**यथाद्य[^699] विरहार्तोऽहमेत्य[^700] व्यापादितस्त्वया।  
मरिष्यसि तथैव त्वमिन्दो जन्मनि जन्मनि॥६२॥**

[^699]: # " यथैव ख."

[^700]: # "इह ख."

**परिशप्तो मयाप्येष त्वमप्येवं मरिष्यसि।  
मद्वदे[^701]वेत्यनागस्कशापतापोदितक्रुधा॥६३॥**

[^701]: # "मद्वदेवेत्य--इवाभवत्‌” इति स्व-पुस्तके त्रुटितम्‌, "

**ततः प्रतनुमन्योर्मे परिताप इवाभवत्‌।  
जामातायं[^702] महाश्वेतासम्बन्धादिति[^703] पश्यतः॥६४॥**

[^702]: # "'तस्मतैषः ख. "

[^703]: # "“भवति स्फुटम्‌ ख"

**कुर्यादपि[^704] प्रतीकारं श्चेतकेतुर्महातपाः।  
शरीरमस्य[^705] रक्षामि सामृतैरहमंशुभिः॥६५॥**

[^704]: # "अत्र ख,"

[^705]: # "देहमेतस्य' क"

**इत्युत्क्षिप्य मयेहायमानीतस्त्व॑ च सत्वरम्‌।  
गत्वा निवेदयेमं तु वृत्तान्तं श्वेतकेतवे॥६६॥**

**इति चन्द्रमसो वाक्यं श्रुताहं दिवमुत्पतन्‌।  
त्वरित्प्रसूतः[^706] कञ्चिद्वैमानिकमलङ्घ्यम्‌॥६७॥**

[^706]: # "त्वरितं प्रस्थितः ख,"

**स च मामशपत्कोपादश्वेनेव तपोबलात्‌।  
त्वयाहं लङ्घितस्तसादश्व एव भविष्यसि॥६८॥**

**स मया प्रणिपत्योक्तः सुहृच्छोका[^707]न्वचेतसः।  
प्रमादान्न तपोदर्पान्मम तत्संहर क्रुधम्‌॥६९॥**

[^707]: # " “बन्धुशोकार्तचेतसा ख."

**स त्वभाषत दत्तोऽयं शापो भवति नान्यथा।  
तारापीडस्य पुत्रार्थमुज्जयिन्यां तपस्यतः॥७०॥**

**राज्ञो[^708] यास्यति पुत्रत्वं चन्द्रमाः सनिदर्शनम्।  
पुण्डरीकस्तदीयस्य शुकनासस्य मन्त्रिणः॥७१॥**

[^708]: # "स तु यास्यति पुत्रत्वं चन्द्रमास्तन्निदर्शनं ख."

**तारापीडसुतस्येन्दोर्गन्ता[^709] वाहनतां भवान्‌।  
तस्यावसाने स्नात्वैव[^710] शापस्यान्तो भविष्यति॥७२॥**

[^709]: # "सुतस्य त्वं, गत्वा वाहनतां ब्रज ख,"

[^710]: # " स्नात्वा तु शान्तपापो ख."

**तदाकर्ण्यार्णवे तूर्णमपतं शापमूर्च्छितः।  
ततोऽश्वीभूत एवाहमुदतिष्ठमयोनिजः॥७३॥**

**स्मृतिस्तुरगभावेऽपि[^711] न मे नष्टा कदाचन।  
ममैवायमिहानीतः किन्नरानुसृतिक्रमात्॥७४॥**

[^711]: # "“भावेन न.मेः भ्रष्टा क, "

**वीप्सान्वितश्च शापोऽयं[^712] परस्परमभूतयोः।  
द्वितीयजन्मयोगेन तयोर्भाव्यमतो[^713] ध्रुवम्‌॥७५॥**

[^712]: # "कोपोऽयं क."

[^713]: # " भवितव्यः ख,"

**तद्गच्छामि यथावृत्तमाख्यातुं श्वेतकेतवे।  
द्वितीयजन्मग्रहणं क्व तयोश्चेति वेदितुम्‌॥७६॥**

**कपिञ्जलकथातस्तं[^714] वैशम्पायनविग्रहम्‌॥  
पुण्डरीकं परिज्ञाय महाश्वेतान्वत्तप्यत॥७७॥**

[^714]: # "'कथाख्यात॑' ख. "

**द्वितीयेऽप्यहमेवास्य जाता जन्मनि मृत्यवे।  
शप्तो मदनुरक्तोऽसौ मयेति विललाप सा॥७८॥**

**कपिञ्चलस्तदा[^715] त्वेवं रुदतीं तामवारयत्‌।  
अब्रवीच्चावधिः पूर्णः प्रायो दुःखस्य वर्तते॥७९॥**

[^715]: # "तदेत्येवं 'ख,"

**अल्पेनैव[^716] हि कालेन पुण्डरीकस्य सस्मितम्‌[^717]।  
अक्षमालां पुनः पाणिपल्लवे विनिधास्यसि॥८०॥**

[^716]: # " कालेनानतिदीर्घेण ख."

[^717]: # " 'सम्मतां ख."

**कादम्बर्यपि माधुर्यमुग्धस्मितसुधाञ्चितैः।  
प्रमोदयिष्यते[^718] सान्द्रैश्चन्द्रापीडस्य भाषितैः॥८१॥**

[^718]: # " 'प्रसादमेष्यते” ख"

**कादम्बरी तमप्राक्षीद्भगवन्सा[^719] क्व वर्तते।  
पत्रलेखेति स त्वाह मया नाज्ञायि तद्गतिः॥८२॥**

[^719]: # "क्व नु वर्तते, ख."

**इत्युक्त्वोदपतद्व्योम रोमकूपाम्बुविप्रुषः।  
आच्छोदबिन्द[^720]ुसन्दिग्धाः किरन्नथ कपिञ्जलः॥८३॥**

[^720]: # " नैशोदबिन्दु ख."

**तस्मिन्गते महाश्वेताकादम्बर्यौ तथैव ताम्‌।  
सशोकविस्मयोत्कम्पे कथया निन्यतुर्निशाम्‌॥८४॥**

**महाश्वेता[^721]कथावृत्ते तत्राप्यधिकनिश्चया।  
कादम्बर्याः स्थिरं चित्ते[^722] सा प्रत्ययमजीजनत्‌॥८५॥**

[^721]: # " यथावृते ख."

[^722]: # "तत्र सप्रत्ययं ख"

**अपरेद्युरथ स्फीतस्फटिकोपलवेदिकाम्‌।  
हृद्यामारोप्य कर्पूरतुषारषणदन्तुरैः॥८४॥**

**अभिषिच्य जलैः स्वच्छैरनुलिप्य च चन्दनैः।  
आनर्च चन्द्रापीडस्य देहं कादम्बरी चिरम्‌॥८७॥**

**यदि सत्योऽथमाश्वासः[^723] सर्वं स्वस्थमथान्यथा।  
क्व गतो मे तदा मृत्युरित्यन्तार्निश्चिकाय सा॥८८॥**

[^723]: # "महाश्वासः सर्व स्वच्छं' ख"

**जगाद च महाश्वेतामिदानीं ननु[^724] न त्रपे।  
सखीति व्याहरन्ती[^725] त्वां दुःखसब्रह्मचारिणीसम्॥८९॥**

[^724]: # " “अगतत्रपा' ख."

[^725]: # "व्यहरन्ती' क"

**विधिना विदधानेन समानसुखदुःखताम्।  
आवयोश्चिरसंरूढं सखीप्रेम[^726] न दूषितम्‌॥९०॥**

[^726]: # "सखि प्रेम” ख, "

**ततः प्रभृति सा मूर्तिमर्चयन्त्यास्त[^727] तापसी।  
चन्द्रापीडस्य चन्द्रार्धचूडस्येव दिवानिशस्‌॥९१॥**

[^727]: # "अथः ख"

**दिवसेष्वथ गच्छत्सु निर्विकारशरीरताम्।  
चन्द्रापीडस्य पश्यन्ती स्थिरामाशां[^728] बबन्ध सा॥९२॥**

[^728]: # "“स्थिरः ख."

**ततः सेनान्यमाहूय बलाहकमभाषत।  
विदितस्तव सर्वोऽयं वृत्तान्तो बहुविस्मयः॥९३॥**

**देहकान्त्यविनाशेन जातः सम्प्रत्ययश्च नः।  
अर्हस्युज्जयिनीं गन्तुं प्रभोरा[^729]श्वासकारणात्‌॥९४॥**

[^729]: # "प्रभावाश्वासः क. "

**सोऽब्रवीद्देबि तत्रान्यं[^730] प्रज्ञातं प्रहिणोम्यहम्।  
युवराजमिहोत्सृज्य न गन्तुं स्वयमुत्सहे[^731]॥९५॥**

[^730]: # "तत्राहं प्रज्ञानं क."

[^731]: # "मम साम्प्रतम्‌"

**ततस्त्वरितको नाम राज्ञो निकट[^732]सेवकः।  
प्रहितस्तेन लेखश्च स्वहस्तलिखिताक्षरः॥९६॥**

[^732]: # "निकटवर्तिनः ख."

**स गत्वोज्जयिनीं प्राप्य सर्वं राज्ञे न्यवेदयत्‌।  
राजा सान्तःपुरामात्यस्तदश्रौषीद्यथाक्रमम्‌॥९७॥**

**ततस्तनयहृद्भङ्गश्रुतिव्यामूढचेतनः।  
ऊत्तरस्मिंश्च[^733] वृत्तान्ते न दृढप्रत्ययोऽभवत्‌॥९८॥**

[^733]: # "“उक्ते तस्मिन्सः ख"

**शुकनासस्तु तं प्राह कर्म वैचित्र्यवेदिनः।  
श्रुतेतिहासशास्त्रस्य देव कोऽत्र तव भ्रमः॥९९॥**

**देवतानामृषीणां च मर्त्यदेहपरिग्रहः।  
श्रूयते हि पुराणेषु स्मृतावपि च पठ्यते॥१००॥**

**चित्राश्चातिशयाः[^734] पुंसाममानुषजनोचिताः।  
किमप्यलौकिकं बीजमनुमापयितुं क्षमाः॥१०१॥**

[^734]: # "'चित्राख्यातिशयाः ख,"

**कुमारश्च[^735] गुणैरिन्दोरतिदूर॑ न भिद्यते।  
सा कान्तिराकृति शक्तिः क्व टृष्टा[^736] चान्यमानुषे॥१०२॥**

[^735]: # "कुमारस्य गुणैरिन्दोरपि भेदो न विद्यते ख"

[^736]: # "“दृष्टान्यत्र पूरुषे ख,"

**स्वप्ने देवीमुखे दृष्टचन्द्रमास्तु विशंस्त्वया[^737]।  
[मया](http://# " मयापि पुण्डरीकाङ्को ब्राह्मणः ख,") च पुण्डरीकाङ्का ब्राह्मणी समदृश्यत॥१०३॥**

[^737]: # " 'तदा त्वया ख,"

**शाप[^738] एव तयोरासीदावयोस्तु परो वरः।  
देवताः पुत्रतां यान्ति यद्गृहे[^739] मर्त्यधर्मिणाम्‌॥१०४॥**

[^738]: # " 'अत एव तयोरीदृगावयोश्च परे ख"

[^739]: # " यद्देहे ख."

**कपिञ्जलः[^740] पुनस्तादृगविलुप्तगतिर्दिंवि।  
आचक्षीत कथं[^741] मिथ्या तुरगो नु क्व तादृशः॥१०५॥**

[^740]: # " 'कपिञ्जलमुनिस्तावद्‌ ख."

[^741]: # " महातेजा कथं मिथ्या तु ख."

**इदं क्वान्यत्र दृष्टं वा जीवितेनोज्झितोऽपि यः[^742]।  
पञ्चभूतात्मको[^743] देहः स यदेति न विक्रियाम्‌॥१०६॥**

[^742]: # " यत् ख."

[^743]: # "“याति पञ्चात्मकोदेहश्चिरादपि ख"

**ब्रूयात्त्वरितको मिथ्या लिखेन्मिथ्या बलाहकः।  
सुपरीक्षितभृत्यस्य[^744] नेयं शङ्का तवोचिता॥१०७॥**

[^744]: # " “मूलस्य ख."

**विदधतभसमोल्लसत्प्र[^745]मोदं जगदचिरादवलोकयिष्यसि त्वम्‌।  
शशिनगिव विधुन्तुदेन मुक्तं व्यपगतशापकदर्थनं कुमारम्‌॥१०८॥**

[^745]: # "अकृशोल्लसत्” ख."

**इति मन्त्रिवचांसि शृण्वतोऽपि स्वसुतस्नेहविसंस्थुलस[^746]्य राज्ञः।  
न धृतिं[^747] प्रतिपद्यते स चेतो गमने च व्यवसायमाललम्बे॥१०९॥**

[^746]: # "'विसंष्ठुलस्य ख "

[^747]: # " वृति क."

**अथ सह महिषीभिर्बाष्पपूर्णेक्षणाभि-  
र्निजसचिवसमेतोऽगच्छद[^748]च्छोदतीरम्।**

[^748]: # "प्रापत्, ख. "

**शिथिलितसकलास्थः[^749] पार्थिवस्तं च यान्तं  
जनपदपुरलोकः सर्व एवान्वगच्छत्‌[^750]॥११०॥**

[^749]: # "सकलार्थ पार्थिवं तं ख"

[^750]: # "“अन्वयासीत्‌ ख ।"

**तस्मिन्गतेऽथ विजनापणवीथिकान्ता  
कान्तारभूरिव तदोज्जयिनी बभूव।**

**याते दिवं सकललोकमनोभिरामे  
रामे यथा हतसमृद्धिरभूदयोध्या॥१११॥**

**गत्वा नृपः सुतमवेक्ष्य चिरं रुदित्वा  
तत्तद्विलप्य सह तत्र विलासवत्या।**

**उत्सृज्य राज्यविभवं यतिवद्व्यधत्त  
तत्रैव वल्कलधरः स्थितिमद्य तावत्‌[^751]॥११२॥**

[^751]: # "यावत्‌” ख"

**इत्युक्त्वा कृतजृम्भिकः क्षणमिव स्थित्वाचचक्षे पुन[^752]-  
र्जाबालिर्भगवानहो बत[^753] रसाद्दीर्घा कथा वर्णिता।**

[^752]: # "'मुनिः' क."

[^753]: # " 'बहुरसात्” ख."

**तद्योऽस[^754]ौ शुकनाससूनुरनया शप्तो महाश्वेतया  
वैशम्पायन एष सोऽद्य शुकतां तेनागतः कर्मणा॥११३॥**

[^754]: # "तथ्योऽसौ ख."

**अपि चाविनयेन[^755] कोपितः शुकनासो[^756]ऽस्य पिता यदब्रवीत्‌।  
शुक एव बृथा स पाठितः फलितं तच्च[^757] वचोऽद्य शापवत्‌। ११४॥**

[^755]: # "“वाः ख. "

[^756]: # "अपि क."

[^757]: # "तद्वचनं चः ख,"

**श्रुत्वैतां तु कथां तपोधनगणैः पृष्टः पुनर्विस्मितैः  
को हेतुर्भगवन्यदस्य भदनस्तावान्मुनेरप्यभूत्‌[^758]।**

[^758]: # "मनसः ख."

**जाबालिः पुनराह नन्वभिहितं स्त्रीवीर्यतः केवला-  
दुत्पन्नोऽयमतो रजस्वलतया नारीस्वभावोऽभवत्‌॥११५॥**

**विरतवचसि तस्मिन्नेतदुक्त्वा[^759] महर्षौ  
रजनिरपि समाप्तिं सा कथावत्प्रपेदे।**

[^759]: # "'एवमुक्त्वा' ख,"

**तदनु मुनिजनेनाकारि[^760] पूर्वाह्णसन्ध्या-  
समयसमुचितेषु ध्यानकार्येषु सिद्धिः॥११६॥**

[^760]: # "अधायि ख."

**दरति श्रीविपश्चिद्वराग्रगण्याचार्यभट्टजयन्तसूनोरभिनन्दस्य कृतौ कादम्बरीकथासारे**

**सप्तमः सर्गः।**  
**———————————————**



**अष्टमः सर्गः।**

**अथ मां[^761] श्रुतजाबालिप्रोक्ताखिल[^762]कथाक्रमम्।  
अस्पृशद्बालभावेऽपि प्राग्जन्मचरितस्मृतिः[^763]॥१॥**

[^761]: # "संश्रुत क"

[^762]: # "“गीताखिल” स, "

[^763]: # ". 'पूर्वजन्मस्मृतिस्तदा ख."

**श्वेतकेतुसुतो[^764] भूत्वा शुकनाससुतोऽभवम्।[^765]  
स वैशम्पायनोऽद्याहं जातः पुत्रः पतत्रिणः॥२॥**

[^764]: # "श्वेतकेतोः ख,"

[^765]: # "शुकनासात्मजो” ख,।"

**मम[^766] जातस्मृतेर्व्यक्ता प्रावर्तत सरस्वती।  
स्वयं[^767] प्राग्जन्मविद्याश्च हृदयं प्रतिपेदिरे॥३॥**

[^766]: # "उपजातः ख"

[^767]: # " मम ख."

**यथैव चान्ये संस्काराः प्राक्तना मामुपागमन्‌।  
महाश्वेतानुरागोऽपि तथैव[^768] समुपागमत्‌॥४॥**

[^768]: # "तेभ्यः पूर्वतरस्तथाः ख,"

**अथाभ्यधिक[^769]हारीतस्नेहसंवर्धितः क्रमात्‌।  
गगनाक्रमणे शक्तिमवापं जातपक्षतिः[^770]॥५॥**

[^769]: # "ततोऽप्यधिकः ख."

[^770]: # " धुतपक्षतिः' क"

**दिवसेष्वथ गच्छत्सु शनैरारूढ[^771]यौवनः।  
गन्तुमैच्छं[^772] महाश्वेताश्रमं[^773] तद्दर्शनोत्सुकः॥६॥**

[^771]: # " “सुपारूढ ख, "

[^772]: # "'ऐच्छन् ख."

[^773]: # " मुखदर्शनदोदहदी” ख,"

**अचिन्तयं[^774] च जन्मेदं विहगस्यापि यातु[^775] मे।  
महाश्वेताश्रमोपान्ते[^776] तरुकोटरशायिनः॥७॥**

[^774]: # "अकल्पयं ख."

[^775]: # "जातु ख."

[^776]: # "उपान्ततरु- ख"

**एकदा त्वथ जाबालेराश्रमं समुपागमत्‌[^777]।  
ताताज्ञया सुहृत्स्नेहपराधीनः कपिञ्चलः॥८॥**

[^777]: # "समुपाययो ख."

**स च[^778] पप्रच्छ हारीतं क्व वैश्यम्पायनः शुकः।  
उत्पन्नप्रत्यभिज्ञोऽहमयमस्मीत्यबादिषम्‌॥९॥**

[^778]: # "वै ख."

**ततश्चिरमरोदीत्स स्वबक्षसि निधाय माम्‌।  
वयस्यपुण्डरीकस्य धिगियं[^779] वर्तते दशा॥१०॥**

[^779]: # " यदियं ख."

**मयापि[^780] रुदतोक्तोऽसौ किं रोदिषि कपिञ्जल।  
त्वयापि[^781] मत्कृते दृष्टा सखे कष्टतरा[^782] दशा॥११॥**

[^780]: # "` मया रुदित्वाप्युक्तो' ख."

[^781]: # "तवापि ख."

[^782]: # " 'कष्टतरी' ख"

**क्व हेषा क्व जपाभ्यासः क्व पर्याणं क्व वल्कलम्‌।  
क्वोपवीतं क्व वा वल्गा क्व मुनिः क्व तुरङ्गमः॥१२॥**

**ततः[^783] क्षणमिव स्थित्वा तातसन्देशमब्रवीत्।  
कपिञ्जलः समासेन पुनर्गमनसत्वरः॥१३॥**

[^783]: # "“इत्थं ख,"

**तातस्त्वामादिदेशैव॑[^784] वत्स दोषस्तवात्र कः।  
रजोविकृतयो युक्ताः शुद्धस्त्रीबीजजन्मनाम्‌[^785]॥१४॥**

[^784]: # "आदिशत्येवं' ख."

[^785]: # "वीर्य ख."

**अल्पायुष[^786]ं विदित्वापि जाते[^787] त्वयि न यन्मया।  
कृता[^788] प्रतिक्रिया तत्र तदालस्यं हि मामकम्‌॥१५॥**

[^786]: # "'अनायुषं ख."

[^787]: # "याते ख."

[^788]: # "तव प्रतिक्रियाकारि' ख"

**अद्यायुष्कामयागस्तु प्रस्तुतः स समाप्यते।  
न यावद्भवता तावन्न गन्तव्यमितः क्वचित्‌॥१६॥**

**अहं त्वदन्तिके सथातुमिच्छ्न्नपि निवारितः।  
तातेन[^789] तत्र कर्मैकदेशलेशाङ्गतां[^790] गतः॥१७॥**

[^789]: # "ततो च ख."

[^790]: # "'लेशी मतामतः' ख,"

**इत्याख्याय परिष्वज्य क्रियाभ्यावर्तनेन[^791] माम्‌।  
क्रमेणामन्त्र्य स मुनीन्प्रतस्थे[^792] दिवमुत्यतन्॥१८॥**

[^791]: # " 'क्रियात्वभ्मर्थ्य तत्र माम्‌ ख."

[^792]: # " मुनिः ख."

**गृहीततातादेशोऽपि काले गच्छत्यहं बलात्‌।  
नीतो बारितवामेन कामेन वशमात्मनः॥१९॥**

**अनुभूय चिरं तत्र चित्राः स्मरबिभीषिकाः।  
अकार्षं[^793] धियमुत्क्रान्तगुर्वाज्ञो[^794] गमनं प्रति॥२०॥**

[^793]: # "अवोधिषमतिक्रान्तः ख,"

[^794]: # "निष्क्रान्त ख."

**एकदा त्वाश्रमं शून्यं वीक्ष्य निर्गततापसम्।  
एकाकी सहसोड्डीय प्रस्थितोऽस्म्युत्तरां दिशम्‌॥२१॥**

**अदूरोड्डीन एवाहमनभ्यस्ताध्वलङ्घनः।  
क्रान्तस्तृषा[^795] च क्लान्त्या च जलं क्वचिदतर्कयम्॥२२॥**

[^795]: # "“अतस्तृषावान्” ख,"

**ततः क्वापि पयः पीत्वा किञ्चिदादश्य चञ्चुना।  
विश्रान्तयेऽविशं[^796]ं श्रान्तस्तरोः कस्यापि कोटरे॥२३॥**

[^796]: # " 'बीतातपे परिश्रान्तः ख"

**क्षणं तत्र निषण्णस्य तदा मे निद्रया मनः।  
गाढया पाशपङ्क्त्या च शरीरकमगृह्यत[^797]॥२४॥**

[^797]: # "शरीरं समगृह्यत ख"

**अथ सुप्तोत्थितोऽपश्यमग्रतो नातिदूरगम्‌।  
कृतार्थमिव मातङ्गयुवानं[^798] मम बन्धनात्‌॥२५॥**

[^798]: # " मातङ्गं ख,"

**आकृत्या कर्मणा[^799] चास्य समर्थ्यपिशितार्थिताम्।  
प्रार्थनापेशलं किञ्चिदात्ममोक्षार्थमभ्यधाम्‌॥२६॥**

[^799]: # "“कर्मणो वास्य, ख."

**भद्र सौम्याकृतिस्त्वं हि न मांसार्थीव दृश्यसे।  
नहि मांसार्थिनो युक्तं निद्राच्छेदप्रतीक्षणम्‌।॥२७॥**

**तदुन्मुञ्चतु[^800] मां भद्र सु्खं यामि यथेप्सितम्‌[^801]।  
मया बद्धेन कोऽर्थस्ते का[^802] विमुक्तेन वा क्षतिः॥२८॥**

[^800]: # "ततो विमुञ्च ख."

[^801]: # "“यथेहितम्' ख "

[^802]: # "किं विसृष्टेन ख."

**सोऽब्रवीन्न स्वतन्त्रेण गृहीतोऽसि[^803] मया शुक।  
प्रथितस्त्वं हि जाबालेराश्रमे सुचिरं स्थितः॥२९॥**

[^803]: # " त्वं धृतोऽसि"

**स त्वं गतो मम स्वामिदुहितुः कर्णगोचरम्‌।  
तन्नियोगाद्गृहीतोऽसि कथमुन्मुच्यसे[^804] मया॥३०॥**

[^804]: # "स कथं मुच्यसे ख."

**पुनराख्या[^805]ममुं सौम्य युवासि स्मरवेदनाम्‌।  
जानासि न प्रियोत्कण्ठाकृष्टं मां रोद्धुमर्हसि॥३१॥**

[^805]: # " 'आख्यमिं ख."

**सोऽभाषत न भृत्येन तरुणेनापि शासनम्‌।  
स्वामिनोऽलङ्घनीयं तत्कथं त्वां त्यक्तुमुत्सहे॥३२॥**

**अवोचं पुनरप्येनं शापाज्जाति[^806]स्मरो मुनिः।  
शुकीभूतोऽस्मि मां मुञ्च परं पुण्यमवाप्स्यसि॥३३॥**

[^806]: # " साधो ख."

**स त्वाह नाह[^807]ं जानामि मुनिं गन्धर्वमेव वा।  
राक्षसं वा पिशाचं वा नीत एवासि पक्वणम्‌[^808]॥३४॥**

[^807]: # "त्वामहं तु ख."

[^808]: # "तत्क्षणं खं "

**इत्युक्त्वादाय पाणौ मामनैषीत्पक्वणं बलात्‌।  
प्रावेशयच्च[^809] चण्डालकुमारीवेश्म कश्मलम्॥३५॥**

[^809]: # "प्राविशश्च स ख."

**दृष्ट्वा तदार्द्रगोमांसवसा[^810]सृक्पङ्कपिच्छलम्।  
प्रोच्छलत्क्रिमि[^811] मत्प्राणैरुत्क्रान्तमिव दुर्भगैः॥३६॥**

[^810]: # "गोमांसं ख."

[^811]: # " 'प्राणवत्प्राणै रक्षान्तमिव दुर्भगं ख."

**अथ मातङ्गकन्यायै स मामस्यै न्यवेदयत्‌।  
स्वामिन्ययमिहानीतः स वैशम्पायनः शुकः॥३७॥**

**साकूतमिव मां वीक्ष्य स्नेहबाष्प[^812]ार्द्रया दृशा।  
क्षणं वक्षसि धृत्वैषा न्यक्षिपद्बद्ध[^813]पञ्जरे॥३८॥**

[^812]: # " बद्धार्द्रया” ख. "

[^813]: # "“प्राक्षिपद्वज्र' ख"

**ततोऽतिमहता चक्रे[^814] शोकेन हृदि मे पदम्‌।  
मुनिर्भूत्वाद्य चण्डाल्या[^815] अहं क्रीडाशुकः स्थितः॥३९॥**

[^814]: # "दध्रे ख."

[^815]: # "“चाण्डाल्या सोऽहं ` लीलाशुकः कृतः” ख. "

**व्यक्तां विदधता वाचं जातिं स्मरयता निजाम्‌।  
अहो जाबालिना दिव्यदृष्टिनापकृतं मम॥४०॥**

**अहो बत महत्कष्टमविषह्यमुपस्थितम्।  
ब्राह्मणेन मया नेयः कालश्चण्डालवेश्मनि॥४१॥**

**भवत्वनशनेनैव त्यजामि[^816] हतजीवितम्‌।  
ताड्यमानोऽपि नाश्नामि न वाचं विसृजामि च॥४२॥**

[^816]: # " त्यजामीह तु” ख."

**एवमुद्वेजितैषा मां हन्याद्वा यदि वा[^817] त्यजेत्।  
भवत्युभयथाप्येष[^818] पन्थाः श्रेयस्करो मम॥४३॥**

[^817]: # " 'वोत्सृजेत्” ख,"

[^818]: # " “एवं' ख."

**इत्युपारूढसङ्कल्पमुपाहृतफलोदुका[^819]।  
एषा चण्डालकन्या मामाख्यन्मधुरया गिरा॥४४॥**

[^819]: # " फलोच्चया' खं,"

**फलान्यशान पानीयमुपयुङ्क्ष्व किमीक्षसे।  
भग्नमौनव्रतो[^820]ऽप्यद्य किं मौनमवरुम्बसे॥४५॥**

[^820]: # "'भण मौनतरतोऽस्यद्य' क."

**मुनिर्जातिस्मरोऽसि[^821] त्वं यदि कोऽपि तथापि ते।  
तिर्यग्जात्युचिता[^822] प्राणबृत्तिरेवाद्य सम्मता[^823]॥४६॥**

[^821]: # "'वा त्वं ख."

[^822]: # " 'उचिता- चारवृर्तिः ख. "

[^823]: # "शास्त्रिद्या क"

**न[^824] तिर्यङ्गधिकारं हि शास्त्रं शास्त्रविदो विदुः।  
तिरश्चां क्व च संस्काराश्चौडोपनयनादयः[^825]॥४७॥**

[^824]: # "नातिर्यगधिकारी हि शस्त्रे ख."

[^825]: # " “चौल-” ख, "

**भक्ष्याभक्ष्याविचारादि यदि चाद्य[^826] चिकीर्षसि।  
तत्किमुच्छिष्टशौचादि नाश्रमेऽश्रितं त्वया॥४८॥**

[^826]: # "वाद्य ख."

**आपत्काले[^827] द्विजातीनामप्यनाश्यं न विद्यते।  
हस्तादस्मा[^828]कमादत्त विश्वामित्रः श्वजाघनीम्॥४९॥**

[^827]: # "आपत्सु च' खं."

[^828]: # "अस्माभिर्दत्तमादत्ते . विश्वामित्रः स्म जीवितं ख."

**कृतद्वित्रो[^829]पवासोऽपि जिहासुरिव जीवितम्‌।  
लक्ष्यसे वत्स मामैवं कृथा[^830] भुङ्क्ष्व फलोदकम्‌॥५०॥**

[^829]: # "'द्वित्र्युपवासो हि ख."

[^830]: # " कृत्वा पिब ख."

**इत्यचण्डालसदृशैर्वाक्यैरस्त्रीजनोचितैः।  
तस्या विस्मयमानोऽहं चिरमेवमचिन्तयम्‌॥५१॥**

**नूनमेषा न चण्डाली[^831] मद्वत्केनापि कर्मणा।  
भन्ये[^832] गतिमिमां प्राप्ता मदर्थेनैव[^833] वा पुनः॥५२॥**

[^831]: # " “चाण्डाली ख."

[^832]: # " मध्यां ख."

[^833]: # " मदर्थे तापिता ख."

**बिभर्त्युरसि मातेव पश्यति स्निग्धया दृशा।  
वक्ति चैवमतोऽमुष्याः करोमि वचनं वरम्‌॥५३॥**

**इति मत्वासृजं वाचमभव॑ भोजनोन्मुखः।  
अभोजयच्च मामेपा जननीव दिने दिने॥५४॥**

**तत्त्वमस्याः परिज्ञातुमन्वहं प्रार्थनां व्यधाम्।  
न त्वेषा प्रार्थ्यमानापि निजं रूपमदीदृशत्[^834]॥५५॥**

[^834]: # " तत्त्वं ख."

**नित्यमाशङ्क[^835]मानोऽहमेतां[^836] कामपि देवताम्‌।  
पोष्यमाणोऽनया कालमनैषं तत्र पक्वणे[^837]॥५६॥**

[^835]: # " आशङ्कमानो ख. "

[^836]: # "“एनां ख,"

[^837]: # " पञ्जरे ख."

**एकदा च[^838] त्रियामान्ते परित्यक्तोऽस्मि निद्रया।  
अद्राक्षं पक्वणं सर्वमार्यप्रायजनावृतम्‌॥५७॥**

[^838]: # " 'तु” ख."

**आत्मानं च नवोदारहेमपञ्चरवर्तिनम्‌।  
कन्यकामीदृशीं[^839] चैतां दृष्टा देवेन यादृशी॥५८॥(युगलकम् )**

[^839]: # "तादृशी ख."

**तत्कोऽहं[^840] किमहं तत्र रक्षितश्चिरमेनया।  
कस्य हेतोरिहानीत इति राजन्न वेद्म्यहम्‌॥५९॥**

[^840]: # "ततः केयं कि तथात्र चिरमस्तोहदमेतया' ख,"

**इत्याकर्ण्य कथामेतां वैशम्पायनवर्णिताम्[^841]।  
स राजा शूद्रकः शय्यागृहा[^842]दङ्गनमागमत्‌॥६०॥**

[^841]: # "निर्मिताम्‌ ख"

[^842]: # "सद्यो गृहादङ्गनं ख."

**ससम्भ्रमं च निर्गत्य तूर्णमानाययत्पुनः।  
आस्थानावसरे दृष्टां[^843] तां मातङ्गकरुमारिकाम्॥६१॥**

[^843]: # "प्रष्टुं ख."

**अथ सा प्रविश्य विततेन तेजसा ददतीव दिक्षु शरदिन्दुचन्द्रिकाम्‌।  
जिततारवेणु[^844]रवभिन्नवल्लकीध्वनिना स्वरेण गदितुं प्रचक्रमे॥६२॥**

[^844]: # " तत्रतार ख"

**हे रोहिणीरमण सर्वकलानिकेत  
तारापते रजनिनाथ सुधानिधान।**  
**कादम्बरीरसगुणेषु निबद्धभाव[^845]-  
मात्मानमर्पितशरीरमनुस्मरेन्दो॥६३॥**

[^845]: # " भार ख. "

**मां च बिद्धि शशकेतन श्रियं साहमस्य जननी दुरात्मनः।  
पूर्वचेष्टितमवैत्वतो[^846] भवानित्यमुं तव समीपमानयम्‌॥६४॥**

[^846]: # "अवैति' कः"

**अयमपि गुरुशासनां[^847] विलङ्घ्य स्मरपरतन्त्रमनाः प्रयातुकामः।  
अविनयपरिशङ्किना तदानीमविकलयोगदृशा व्यलोकि पित्रा॥६५॥**

[^847]: # " 'शासनं' ख "

**तस्याज्ञयैवमनुभावयितुं विपाक-  
मासन्नमेव गुरुशासनलङ्घनस्य[^848]।**  
**अत्रावतीर्य[^849] विपिने जनसङ्गभीत्या  
मातङ्गभङ्गिमवलम्ब्य तथा स्थितास्मि॥६६॥**

[^848]: # "लङ्घनोत्थ्यं' ख,"

[^849]: # " सद्योऽबतीर्यं ख."

**गच्छन्तमेनं विनयस्य पाशैस्तीव्राणि दुःखान्यनुभावयन्ती।  
शापावसानं युवयोः क्रतोश्च समाप्तिमासं चिरमीक्षमाणा॥६७॥**

**मुनेरवभृथावधिः पितुरमुष्य सिद्धः क्रतुः  
कृतस्त्वमपि[^850] संस्मृतस्वतनुरद्य तच्छासनात्‌।**  
**तदुज्झितमली[^851]मसेदृशशरीरयोगौ युवां  
निजेन वपुषाधुनानुभवतां प्रियासङ्गमम्‌॥६८॥**

[^850]: # "'त्वमसि सम्भृत' क."

[^851]: # " 'तदुद्धृत' ख,"

**इत्युक्त्वोच्चैर्गमनरभसप्रश्लथीभूतबन्ध-  
स्रग्विस्रस्तैः कुसुमविदलैर[^852]र्चयित्वेव पृथ्वीम्।**  
**दिक्चक्रं च प्रचलरशनानूपुरोस्थैर्निनादै-  
रामन्त्र्येव[^853] द्रुतमुदपतद्द्यामसौ दिव्यकान्तिः॥६९॥**

[^852]: # "वदलैः ख. "

[^853]: # " एवं” ख."

**स्वतनुकान्तितिरस्कृतचन्दिकां गगनमुत्पतितामवलोक्य ताम्‌।  
सुचिरमूर्ध्वनिवेशितलोचनः कमपि विस्मयमाप जनोऽखिलः॥७०॥**

**अथास्य राज्ञो हृदि शूद्रकस्य निरस्तधैर्यादि[^854]गुणप्रबन्धः।  
कादम्बरीसंस्मरणानुबन्धलब्धावकाशो मदनो जजृम्भे॥७१॥**

[^854]: # "धैर्यद्विगुण' ख."

**न राजलक्ष्मीं बहु मन्यते स्म[^855] नोर्वीं न कीर्तिं न सरस्वतीं च।  
स[^856] तद्रसक्षीब इवाललम्बे कादम्बरीमेव नितान्तमन्तः॥७२॥**

[^855]: # " स ख. "

[^856]: # " “अमुष्य तत्केवलमाललम्वे ख,"

**पराङ्मुखः स्पर्शरसेष्बजस्रं नादत्त कर्णौ मधुरेऽपि स ध्वनौ।  
जगाम निर्लक्ष्यनिविष्टलोचनः क्रमेण सर्वेन्द्रियवृत्तिशुन्यताम्॥७३॥**

**न गुरोरुपदेशमग्नहीद्बुबुधे वस्तु ततो न मूढधीः।  
यतते स्म न तस्य सिद्धये न परिस्पन्दितुमप्यपारयत्‌॥७४॥**

**भूताभिभूत इव मूढ इव क्रमेण  
निःसञ्ज्ञतां विरहवेदनयोपनीतः[^857]।**  
**भुक्त्वा[^858] मयोभवरुजं सुचिरं स राजा  
प्राणान्मुमोच सहसैव विहङ्गमोऽपि॥७५॥**

[^857]: # "'उपनीतां' ख."

[^858]: # " भुक्त्वा मनोभवरूजः ख."

**जीवात्मक॑ चान्द्रमसं च तेजः कृतास्पदं तस्य[^859] तनौ यदासीत्‌।  
तदेव सङ्क्रान्तमथान्यदेहे[^860] चित्रा हि विश्वप्रकृतेर्बिकाराः॥७६॥**

[^859]: # "तच्च ख,"

[^860]: # "अथापरत्र' ख,"

**अत्रान्तरे चित्ररथात्मजायास्तनुं प्रियस्यान्वहमर्चयन्त्याः।  
तथैव[^861] कालः सुमहानगच्छदच्छोदतीरे तपसि स्थितायाः॥७७॥**

[^861]: # " तत्रैव ख."

**अथ मधुसमये विजृम्भिते सा मदनमहोत्सववासरे सरागम्‌।  
मलयजघनसारहारपुष्पैस्तनुमतिमात्रमलञ्चकार भर्तुः॥७८॥**

**तं वीक्ष्य तापसदृशापि[^862] तदा जगाम  
कामस्य गोचरमसौ मधुजृम्भितस्य।**  
**एकाकिनी सपदि दिक्षु निधाय चक्षुः  
कान्तं स्वपन्तमिव निर्भरमालिलिङ्ग॥७०॥**

[^862]: # " 'दृशो” ख,"

**चन्द्रापीडं सा च जग्राह गाढं कण्ठस्थाने जीवितं च प्रपेदे।  
तेनापूर्वां[^863] सा समुल्लासलक्ष्मीमिन्दुस्पृष्टा सिन्धुवेलेव भेजे॥८०॥**

[^863]: # "अपूर्व ख."

**शिलाशय्यां त्यक्त्वा तदनु चिरसुप्तोत्थित इव  
क्षणं स्थित्वा जृम्भालसवदनमुन्मील्य[^864] नयने।**  
**ततश्चन्द्रापीडः प्रहतमुरजध्वानपटुना  
स्वरेण स्वेनैव स्मितमधुरमाह प्रियतमाम्‌॥८१॥**

[^864]: # " वदनपङ्केरुहरुचिः। बभाषे पीयुषद्रवनिवहसं- स्पर्धिभिरयं वचोभिस्तामुञ्जैःः सपदि दरमानन्दविवशः॥ख"

**त्वगभृत[^865]प्रभवेऽप्सरसां कुले[^866] सुतनु जन्म समर्थितवत्यसि।  
यदमुना परिरम्भलवेन[^867] ते सपदि जीचितमापदयं जनः॥८२॥**

[^865]: # " तदमृतं क."

[^866]: # "“कुलेष्वतनु” ख,"

[^867]: # "बलेन ख."

**मामन्वबीभवदिमामियतीमवस्थां  
शापः[^868] कलङ्क इव कान्तिहरो द्वितीयः।**  
**त्वद्धिप्रयोगक्रुत एव बभूव ताप-  
स्तन्वङ्गि शूद्रकतनोरपि मृत्यवे मे॥८३॥**

[^868]: # " शापः कृशोदरि स कान्तिहरो ख."

**संरक्षिता तनुरियं तु तव प्रियेति  
शापः कृशोदरि विराममवाप[^869] पापः।**  
**अद्यप्रभृत्यखिलमेव हि चन्द्रलोकं  
शाधि स्वपादकृतमुद्रमिमं[^870] च लोकम्‌॥८४॥**

[^869]: # "'चिरं समवाप ख."

[^870]: # "अमुं ख."

**सख्याः प्रियं पितृतपोभिरवाप्तदिव्य-  
जीवं कपिञ्चलकरार्पितपाणिपद्मम्‌।**  
**मन्मण्डलादवत्तरन्तमखण्डकान्ति-  
मापाण्डुगण्डमवलोकय पुण्डरीकम्‌॥८५॥**

**कौतूहलात्सह मयैव गतेन्दुलोक-  
मालोकयिष्यसि कदाचन पत्रलेखाम्।**

**कापि प्रिये सहचरी मम सा ततस्त्या  
मा तेऽद्यभूत्त[^871]दनवाप्तिकृतो विषादः॥८६॥**

[^871]: # "तेडप्यभूत् ख,"

**श्रुत्वेति तस्य गिरमुत्सुकतां गतासौ  
रोमाञ्चकञ्चुकमकृत्रिममुद्वहन्ती।**  
**कादम्बरी चरणयोर्निपपात पत्यु[^872]-  
रानन्दबाष्पकणशीकरितेक्षणश्रीः॥८७॥**

[^872]: # "भर्तुः ख."

**उत्थाय त्वरितोपसृत्य च महाश्वेतां परिष्वज्य सा  
प्रत्यक्षीकृतपुण्डरीकपदवीं कृत्वा च लब्धोत्सवाम्‌।**  
**दिष्ट्या बर्धयितुं न्ययुङ्क्त रभसेनाहूय केयूरकं  
तारापीडतपोवने परिजनं[^873] पित्रोर्गृहे चापरम्‌[^874]॥८८॥**

[^873]: # "परिगतं ख."

[^874]: # " 'वापरम्” ख"

**तारापीडः कृशपरिणतो[^875]ऽप्याययौ तत्र तूर्णं  
सानन्दान्तःपुरपरिजनामात्यवर्गानुयातः।**  
**तस्मिन्काले[^876] सखिसहचरः पुण्डरीकोऽपि नाका-  
त्किञ्चिल्ल[^877]ज्जावनतवदनो वल्लभां स्वामुपेतः॥८९॥**

[^875]: # "परिणति कृशोः ख "

[^876]: # "'अस्मिन्‌ः क"

[^877]: # "लज्जावनमितमुखो वल्लभाभ्यासमभ्यैत्” ख"

**चन्द्रापीडः स्वामपि जानन्नथ मूर्तिं  
तारापीडं पुत्रवदेव प्रणनाम।**  
**तद्वतं[^878] च प्रश्रयशीलं शुकनासं  
दत्तानन्दोऽबन्दत सर्वाश्च जनित्रीः[^879]॥९०॥**

[^878]: # "मूर्ध्ना तद्वत् ख."

[^879]: # "सवित्री ख."

**माया धातुरियं भवेद्भगवतः कि श्वेतकेतोरिदं  
सामर्थ्यं तपसामथेन्दुमहसामत्रेन्द्रजालायितम्।**  
**वैशम्पायन इत्यवेदि यदसावेकोऽपि तद्बन्धुभि[^880]-  
र्यच्चासौ[^881] स तु पुण्डरीक इति ते तस्मिन्भवे संस्थिताः॥९१॥**

[^880]: # " यद्बन्धुभिः ख,"

[^881]: # "यच्चामंसत' ख"

**स पुण्डरीको मुनिरप्यपश्यदात्मानमन्तः शुकनासपुत्रम्।  
प्रणम्य भूमौ[^882] पितरं क्रमेण मनोरमाद्या[^883] जननीर्ववन्दे॥९२॥**

[^882]: # "भूपं ख."

[^883]: # "मनो- रमाख्यां जननीं"

**अथ विलासबती समनोरमा स्तनयुगस्रुतदुग्धकणावलिः[^884]।  
सुत[^885]वपुःपरिरम्भमहोत्सवं फलितसर्वमनोरथमन्वभूत्‌॥९३॥**

[^884]: # "स्नुतपयोधरबिन्दुकणाबली ख"

[^885]: # "सुतबधु” ख."

**तारापीडस्तु पादानतम[^886]नवरतप्रस्नुतानन्दबाष्प-  
श्चन्द्रापीडं भुजाभ्यां कथमपि[^887] सहसोत्थाप्य कण्ठे गृहीत्वा।**  
**शापेनात्रावतीर्णो यदि तदपि भवानुत्तमो लोकपालः  
प्रख्याप्येति प्रतीपं प्रणततनुरसावस्य पादौ ववन्दे॥९४॥\`**

[^886]: # " 'पादावनतमपि मुहुः ख."

[^887]: # " कथमिव क."

**तस्मै निवेद्य नृपचक्रमनिच्छतेऽपि[^888]  
यत्नात्कमोपनतया[^889] सह राजलक्ष्म्या।**  
**नोज्झाञ्चका[^890]र चिरसञ्चितमाश्रमं स-  
त्तत्प्रार्थि[^29]तोऽपि महिषीसहितश्चतुर्थम्‌॥९५॥**

[^888]: # "'तिजचक्रं क, "

[^889]: # "क्रमोपचितया ख."

[^890]: # "अङ्गीचकार करसम्भृतं” ख."

**आजग्मुस्तत्र बल्गत्तुरगखुरपुटोद्धृतधूलीकडारै-  
श्चूडाप्रत्युप्त[^892]भास्वन्मणिरुचिरुचिरैः शोभमानाः शिरोभिः।**  
**चन्द्रापीडेन्दुसन्दर्शनरभ[^893]ससमुत्फुल्ललोलेक्षणाब्जा  
निर्व्याजौत्सुक्यसज्जीकृतनिरतिशयोपायनाः पार्थिवेन्द्रांः॥९६॥**

[^892]: # "चूडारत्नप्रवालोज्जवलं ख"

[^893]: # "रसरभसोत्फुल्ल' ख"

**सङ्कल्पाधिकलाभहर्षरभसोज्ज[^894]म्भाविलोलीभव-  
द्गन्धर्वप्रमदाकदम्बककृतक्रीडा[^895]करास्फालनाम्‌।**  
**सार्धं[^896] चित्ररथस्ततो मदिरया हंसश्च गौर्या सह  
द्रष्टुं नृत्तमयी[^897]मिव व्यवसितामाजग्मतुर्मन्दिनीम्॥९७॥**

[^894]: # " रभसौ जृम्भा ख."

[^895]: # "पदक्रीडा ख."

[^896]: # " सर्व” ख"

[^897]: # " नृत्तमयातिभव्यभवनां' ख,"

**ताभ्यामेत्य निरस्त तापसतनुच्छायां दुहित्रोः कृते  
भास्वत्कौतुकमङ्गले नववधूवेषे[^898] मनोहारिणि।**  
**निर्वृत्ते च समन्त्रहोम[^899]सुभगारम्भे विवाहोत्सवे  
तत्रान्योन्यसमागमप्रमुदिताः सर्वे मुदं[^900] लेभिरे॥९८॥**

[^898]: # " 'बृहद्वेपे' ख"

[^899]: # "सुमन्त्र ख."

[^900]: # " समं रेजिरे ख."

**इति विततविलासः पुण्डरीकेण साकं[^901]  
दिवि भुवि च विचित्रोद्यानलीलाविहारः।**  
**अभिमतफलसम्पद्वर्धमानाभिनन्दः  
श्रियमभजत चन्द्रापीडमूर्तिर्मृगाङ्कः॥९९॥**

[^901]: # ". 'सार्धं ख,"

**इति श्रीसकलविपश्चिच्चक्रवर्तिश्रीभट्टजयन्तसूनोरभिनन्दस्य कृतौ कादम्बरीकथासारे**  
**अष्टमः सर्गः।**
