


सलिनथ: रसमन्जरि

र्मञ्ज्, १
यद्गण्डमण्डलगलन्मधुवारि बिन्दुः पानालसाति निभृतां ललितालिमाला /
यद्गुञ्जितेन विनिहन्ति नवेन्द्रनील शङ्कां स वो गणपतिः शिवमातनोतु // र्मञ्ज्_१.१ //
इन्दीवरी भवति यच्च चरणारविन्दद्वन्द्वे पुरन्दरपुरःसरदेवतानाम् /
वन्दारुता कलयतां सुकिरीटकोटिः श्री शारदा भवतु सा भवपारदाय // र्मञ्ज्_१.२ //
हे भस्माङ्गविरक्तिरूपगुणदं तं प्रेरणादं शिवं गङ्गाभूषितशेखरं स्मरहरं शक्तिस्वरूपं प्रभो /
त्वामीशं करुणार्णवं शरणदं विद्यानिधिं निर्गुणं सूतेन्द्रं गिरिजापतिं शशिधरं माङ्गल्यदेवं नमः /
श्रीवैद्यनाथतनयः सुनयः सुशीलः श्रीशालिनाथ इति विश्रुतनामधेयः /
तेनावलोक्य विधिवद् विविधप्रबन्धान् आरम्भते सुकृतिना रसमञ्जरीयम् // र्मञ्ज्_१.३ //
सन्मधुव्रतं वृन्दानां सततं चित्तहारिणी /
अनेकरसपूर्णेयं क्रियते रसमञ्जरी // र्मञ्ज्_१.४ //
हरति सकलरोगान्मूर्छितो यो नराणां वितरति किल बद्धः खेचरत्वं जवेन /
सकलसुरमुनीन्द्रैर् वन्दितं शम्भुबीजं स जयति भवसिन्धुः पारदः पारदो'यम् // र्मञ्ज्_१.५ //
शुष्केन्धनमहाराशिं यद्वद्दहति पावकः /
तद्वद्दहति सूतो'यं रोगान् दोषत्रयोद्भवान् // र्मञ्ज्_१.६ //
तेजो मृगाङ्कमौले सोढुं यन्नैव तेजसां पुञ्जैः /
अजरामरतां वितरति कल्पतरुं च रसेश्वरं वन्दे // र्मञ्ज्_१.७ //
यो न वेत्ति कृपाराशिं रसहरिहरात्मकम् /
वृथा चिकित्सां कुरुते स वैद्यो हास्यतां व्रजेत् // र्मञ्ज्_१.८ //
गुरुसेवां विना कर्म यः कुर्यान्मूढचेतसः /
स याति निष्फलत्वं हि स्वप्नलब्धधनं यथा // र्मञ्ज्_१.९ //
विद्यां गृहीतुमिच्छन्ति चौर्यच्छद्मबलादिना /
न तेषां सिध्यते किञ्चिन्मणिमन्त्रौषधादिकम् // र्मञ्ज्_१.१० //
मन्त्रसिद्धो महावीरो निश्चलः शिववत्सलः /
देवीभक्तः सदा धीरो देवतायागतत्परः // र्मञ्ज्_१.११ //
सर्वशास्त्रार्थतत्त्वज्ञः कुशलो रसकर्मणि /
एतल्लक्षणसंयुक्तो रसविद्यागुरुर्भवेत् // र्मञ्ज्_१.१२ //
शिष्यो निजगुरोर्भक्तः सत्यवक्ता दृढव्रतः /
निरालस्यः स्वधर्मज्ञो देव्याराधनतत्परः // र्मञ्ज्_१.१३ //
<पारदनामानि>
शिवबीजं सूतराजः पारदश्च रसेन्द्रकः /
एतानि रसनामानि तथान्यानि शिवे यथा // र्मञ्ज्_१.१४ //
अन्तःसुनीलो बहिरुज्ज्वलो वा मध्याह्णसूर्यप्रतिमप्रकाशः /
शस्तो'थ धूम्रः परिपाण्डुरश्च चित्रो न योज्यो रसकर्मसिद्धये // र्मञ्ज्_१.१५ //
दोषमुक्तो यदा सूतस्तदा मृत्युरुजापहः /
साक्षादमृतम् एवैष दोषयुक्तो रसो विषम् // र्मञ्ज्_१.१६ //
<पारददोषाः>
नागो वङ्गो'ग्निचापल्यम् असह्यत्वं विषं गिरिः /
मला ह्येते च विज्ञेया दोषाः पारदसंस्थिताः // र्मञ्ज्_१.१७ //
जाड्यं कुष्ठं महादाहं वीर्यनाशं च मूर्च्छनाम् /
मृत्युं स्फोटं रोगपुञ्जं कुर्वन्त्येते क्रमान्नृणाम् // र्मञ्ज्_१.१८ //
<पारदशोधन>
अथातः सम्प्रवक्ष्यामि पारदस्य च शोधनम् /
रसो ग्राह्यः सुनक्षत्रे पलानां शतमात्रकम् // र्मञ्ज्_१.१९ //
पञ्चाशत् पञ्चविंशद्वा दश पञ्चैकमेव वा /
पलाद् ऊनं न कर्तव्यं रससंस्कारमुत्तमम् // र्मञ्ज्_१.२० //
<पारदशोधन (१)>
पलत्रयं चित्रकसर्षपाणां कुमारीकन्याबृहतीकषायैः /
दिनत्रयं मर्दितसूतकस्तु विमुच्यते पञ्चमलादिदोषैः // र्मञ्ज्_१.२१ //
<पारदशोधन (२)>
इष्टिकारजनीचूर्णैः षोडशांशं रसस्य च /
मर्दयेत्तं तथा खल्वे जम्बीरोत्थद्रवैर्दिनम् // र्मञ्ज्_१.२२ //
<सामान्यदोषशोधनम्>
काञ्जिकैः क्षालयेत्सूतं नागदोषं विमुञ्चति /
विशालाङ्कोलचूर्णेन वङ्गदोषं विमुञ्चति // र्मञ्ज्_१.२३ //
राजवृक्षो मलं हन्ति पावको हन्ति पावकम् /
चापल्यं कृष्णधत्तूरस् त्रिफला विषनाशिनी // र्मञ्ज्_१.२४ //
कटुत्रयं गिरिं हन्ति असह्यत्वं त्रिकण्टकः /
प्रतिदोषं पलांशेन तत्र सूतं सकाञ्जिकम् // र्मञ्ज्_१.२५ //
सर्वदोषविनिर्मुक्तः सप्तकञ्चुकवर्जितः /
जायते शुद्धसूतो'यं योजयेत् सर्वकर्मसु // र्मञ्ज्_१.२६ //
<पारदशोधनम् (२)>
सुवस्त्रगालितं खल्वे सूतं क्षिप्त्वा विमर्दयेत् /
उद्धृत्य चारनालेन मृद्भाण्डे क्षालयेत् सुधीः // र्मञ्ज्_१.२७ //
रसस्य दशमांशं तु गन्धं दत्त्वा विमर्दयेत् /
जम्बीरोत्थद्रवैर् यामं पात्यं पातनयन्त्रके // र्मञ्ज्_१.२८ //
पुनर् मर्द्यं पुनः पात्यं सप्तवारं विशुद्धये /
युक्तं सर्वस्य सूतस्य तप्तखल्वे विमर्दनम् // र्मञ्ज्_१.२९ //
<तप्तखल्व>
अजाशकृत्तुषाग्निं तु भूगर्भे त्रितयं क्षिपेत् /
तस्योपरिस्थितं खल्वं तप्तखल्वमिदं स्मृतम् // र्मञ्ज्_१.३० //
<पारदशोधनम्>
कुमार्याश्च निशाचूर्णैर् दिनं सूतं विमर्दयेत् /
पातयेत्पातनायन्त्रे सम्यक् शुद्धो भवेद्रसः // र्मञ्ज्_१.३१ //
<पारदशोधन>
श्रीखण्डदेवदारुश्च काकतुण्डी जयाद्रवैः /
कर्कोटीमुसलीकन्याद्रवं दत्त्वा विमर्दयेत् // र्मञ्ज्_१.३२ //
दिनैकं मर्दयेत्पश्चाच्छुद्धं च विनियोजयेत् /
<पारदशोधन>
अथवा हिङ्गुलात् सूतं ग्राहयेत् तन्निगद्यते // र्मञ्ज्_१.३३ //
जम्बीरनिम्बुनीरेण मर्दितं हिङ्गुलं दिनम् /
ऊर्ध्वपातनयन्त्रेण ग्राह्यः स्यान्निर्मलो रसः // र्मञ्ज्_१.३४ //
कञ्चुकैर्नागवङ्गाद्यैर् निर्मुक्तो रसकर्मणि /
विना कर्माष्टकेनैव सूतो'यं सर्वकर्मकृत् // र्मञ्ज्_१.३५ //
सर्वसिद्धमतम् एतद् ईरितं सूतशुद्धिकरम् अद्भुतं परम् /
अल्पकर्मविधिभूरिसिद्धिदं देहलोहकरणे हि शस्यते // र्मञ्ज्_१.३६ //
संस्कारहीनं खलु सूतराजं यः सेवते तस्य करोति बाधाम् /
देहस्य नाशं विविधं च कुष्ठं कष्टं च रोगाञ्जनयेन्नराणाम् // र्मञ्ज्_१.३७ //


र्मञ्ज्, २
अथातः सम्प्रवक्ष्यामि रसजारणमुत्तमम् /
अथाजीर्णम् अबीजं च सूतकं यस्तु घातयेत् // र्मञ्ज्_२.१ //
ब्रह्महा स दुराचारी मम द्रोही महेश्वरि /
तस्मात्सर्वप्रयत्नेन जारितं मारयेद्रसम् // र्मञ्ज्_२.२ //
<पारदबलिजारणं>
प्रक्षिप्य तोयं मृत्कुण्डे तस्योपरि शरावकम् /
सञ्चूर्णमेखलायुक्तं स्थापयेत्तस्य चान्तरे // र्मञ्ज्_२.३ //
रसं क्षिप्त्वा गन्धकस्य रजस्तस्योपरि क्षिपेत् /
समं भागं ततो दद्याच्छरावेण पिधापयेत् // र्मञ्ज्_२.४ //
लघ्वीयसीं भस्ममुद्रां ततः कुर्याद् भिषग्वरः /
आरण्योपलकैः सम्यक् चतुर्भिः पुटमाचरेत् // र्मञ्ज्_२.५ //
एवं पुनः पुनर् गन्धं दत्त्वा दत्त्वा भिषग्वरः /
सम्यक् कुर्वीत सूतस्य देवि षड्गुणजारणम् // र्मञ्ज्_२.६ //
<पारदसुवर्णजारणं>
रसः स्यात् पटुशिग्रुतुत्थैः सराजिकैर्व्योषणकैस् त्रिरात्रम् /
पिष्टस्ततः स्विन्नतनु सुवर्णमुख्यानयं खादति सर्वधातून् // र्मञ्ज्_२.७ //
<पारदजारण>
अथवा बिडयोगेन शिखिपित्तेन लेपितम् /
चरेत् सुवर्णं रसराट् तप्तखल्वे यथासुखम् // र्मञ्ज्_२.८ //
निर्दग्धशङ्खचूर्णं च रविक्षीरेण सम्प्लुतम् /
पुटितं शतशो देवि प्रशस्तं जारणं विदुः // र्मञ्ज्_२.९ //
स्वर्णाभ्रसर्वलोहानि यथेष्टानि च जारयेत् /
अनेकविधिना सूतं चतुःषष्ट्यंशकादिना /
द्वात्रिंशत् षोडशांशेन जारयेत् कनकं बुधः // र्मञ्ज्_२.१० //
<रसमारणं>
द्विपलं शुद्धसूतं च सूतार्धं शुद्धगन्धकम् /
कन्यानीरेण सम्मर्द्य दिनमेकं निरन्तरम् // र्मञ्ज्_२.११ //
रुद्ध्वा तद्भूधरे यन्त्रे दिनैकं मारयेत् पुटान् /
<रसमारण (२)>
भुजङ्गवल्लीनीरेण मर्दितं पारदं दृढम् // र्मञ्ज्_२.१२ //
कर्कोटीकन्दमृन्मूषासम्पुटस्थं पुटे गजे /
भस्म तद्योगवाहि स्यात्सर्वकर्मसु योजयेत् // र्मञ्ज्_२.१३ //
<रसमारण (३)>
श्वेताङ्कोलजटावारि मर्द्यः सूतो दिनत्रयम् /
पुटितश्चान्धमूषायां सूतो भस्मत्वमाप्नुयात् // र्मञ्ज्_२.१४ //
प्रत्यहं रक्तिकापञ्च भक्षयेन्मधु सर्पिषा /
को वा तस्य गुणान् वक्तुं भुवि शक्नोति मानवः // र्मञ्ज्_२.१५ //
<रससिन्दूरनिर्माण (१)>
पलमत्र रसं शुद्धं तावन्मात्रं सुगन्धकम् /
विधिवत् कज्जलीं कृत्वा न्यग्रोधाङ्कुरवारिभिः // र्मञ्ज्_२.१६ //
भावनात्रितयं दत्त्वा स्थालीमध्ये निधापयेत् /
विरच्य कवचीयन्त्रं वालुकाभिः प्रपूरयेत् // र्मञ्ज्_२.१७ //
जायते रससिन्दूरं तरुणारुणसन्निभम् /
अनुपानविशेषेण करोति विविधान् गुणान् // र्मञ्ज्_२.१८ //
<रससिन्दूर (२)>
गन्धकेन समः सूतो निर्गुण्डीरसमर्दितः /
पाचितो वालुकायन्त्रे रक्तं भस्म प्रजायते // र्मञ्ज्_२.१९ //
<रससिन्दूर>
सूतार्द्धं गन्धकं शुद्धं माक्षिकोद्भूतसत्त्वकम् /
गन्धतुल्यं विमर्द्याथ दिनं निर्गुण्डिकाद्रवैः // र्मञ्ज्_२.२० //
स्थापयेद्वालुकायन्त्रे काचकूप्यां विपाचयेत् /
अन्धमूषागतं वाथ वालुकायन्त्रके दिनम् // र्मञ्ज्_२.२१ //
पक्वं सञ्जायते भस्म दाडिमीकुसुमोपमम् /
<रससिन्दूर>
पृथक् समं समं कृत्वा पारदं गन्धकं तथा // र्मञ्ज्_२.२२ //
नवसारं धूमसारं स्फटिकीं याममात्रके /
निम्बुनीरेण सम्मर्द्य काचकुप्यां विपाचयेत् // र्मञ्ज्_२.२३ //
मुखे पाषाणवटिकां दत्त्वा मुद्रां प्रलेपयेत् /
सप्तभिर् मृत्तिकावस्त्रैः पृथक् संशोष्य वेष्टयेत् // र्मञ्ज्_२.२४ //
सच्छिद्रायां मृदः स्थाल्यां कूपिकां सन्निवेशयेत् /
पूरयेत् सिकतापुरैर् आ गलं मतिमान् भिषक् // र्मञ्ज्_२.२५ //
निवेश्य चुल्ल्यां दहनं मन्दमध्यखरं क्रमात् /
प्रज्वाल्य द्वादशं यामं स्वाङ्गशीतलम् उद्धरेत् // र्मञ्ज्_२.२६ //
स्फोटयित्वा पुनः स्थालीमूर्ध्वगं गन्धकं त्यजेत् /
अधस्थं रससिन्दूरं सर्वकर्मसु योजयेत् // र्मञ्ज्_२.२७ //
<रससिन्दूर>

<वज्रमारण (२)>
कांस्यपात्रे तु भेकस्य मूत्रे वज्रं सुभावयेत् /
त्रिः सप्तकृत्वः सन्तप्तं वज्रमेव मृतं भवेत् // र्मञ्ज्_३.२७ //
<वज्रमारण (३)>
त्रिः सप्तकृत्वः सन्तप्तं खरमूत्रेण सेचयेत् /
मत्कुणैस्तालकं पिष्ट्वा तद्गोले कुलिशं क्षिपेत् // र्मञ्ज्_३.२८ //
प्रध्मातं वाजिमूत्रेण सिक्तं पूर्वक्रमेण वै /
भस्मीभवति तद्भुक्तं वज्रवत्कुरुते तनुम् // र्मञ्ज्_३.२९ //
आयुष्यं सौख्यजनकं बलदं रूपदं तथा /
रोगघ्नं मृत्युहरणं वज्रभस्म भवत्यलम् // र्मञ्ज्_३.३० //
<वैक्रान्त>
अष्टास्रश् चाष्टफलकः षट्कोणो मसृणो गुरुः /
शुद्धमिश्रितवर्णैश्च युक्तो वैक्रान्त उच्यते // र्मञ्ज्_३.३१ //

श्यामलः कृष्णवर्णश्च कर्बुरश्चाष्टधा हि सः // र्मञ्ज्_३.३२ //
<वैक्रान्तशोधनम्>
वैक्रान्तं वज्रवच्छुद्धं ध्मातं तं हयमूत्रके /
हितं तद्भस्म संयोज्यं वज्रस्थाने विचक्षणैः // र्मञ्ज्_३.३३ //

<वैक्रान्तसत्त्वपातनम्>
वैक्रान्तं वज्रकन्दे च पेषयेद् वज्रवारिणा /
माहिषं नवनीतं च सक्षौद्रं पिण्डितं ततः // र्मञ्ज्_३.३५ //
शोषयित्वा धमेत् सत्त्वं इन्द्रगोपसमं भवेत् /

चतुर्थं च वरं ज्ञेयं न चाग्नौ विकृतिं व्रजेत् // र्मञ्ज्_३.३७ //
तस्माद्वज्राभ्रकं ग्राह्यं व्याधिवार्धक्यमृत्युजित् /
अशुद्धाभ्रं निहन्त्यायुर्वर्धयेन्मारुतं कफम् // र्मञ्ज्_३.३८ //
अहतं छेदयेद्गात्रं मन्दाग्निकृमिवर्धनम् /
अतः शुद्धाभ्रकं ग्राह्यं मन्दाग्निकृमिनाशनम् // र्मञ्ज्_३.३९ //

पादांशशालिसंयुक्तम् अभ्रं बद्ध्वाथ कम्बले /
त्रिरात्रं स्थापयेन्नीरे तत् क्लिन्नं मर्दयेद् दृढम् // र्मञ्ज्_३.४० //
कम्बलाद्गलितं श्लक्ष्णं वालुकारहितं च यत् /
तद्धान्याभ्रमिति प्रोक्तं सद्भिर्देहस्य सिद्धये // र्मञ्ज्_३.४१ //
<धान्याभ्रककरण>
धमेद्वज्राभ्रकं वह्णौ ततः क्षीरेण सेचयेत् /
भिन्नपत्रं तु तत्कृत्वा मेघनादाम्लयोर्द्रवैः /
भावयेदष्टयामं तद्धान्याभ्रं कारयेत् सुधीः // र्मञ्ज्_३.४२ //
<धान्याभ्र (३)>
अथवा बदरीक्वाथे ध्मातमभ्रं विनिक्षिपेत् /
मर्दितं पाणिना शुष्कं धान्याभ्राद् अतिरिच्यते // र्मञ्ज्_३.४३ //
<अभ्रकभस्म>
धान्याभ्रकस्य भागैकं द्वौ भागौ टङ्कणस्य च // र्मञ्ज्_३.४४ //
पिष्ट्वा तदन्धमूषायां रुद्ध्वा तीव्राग्निना पचेत् /
स्वभावशीतलं चूर्णं सर्वरोगेषु योजयेत् // र्मञ्ज्_३.४५ //
<अभ्रमारण (१)>
धान्याभ्रकं दृढं मर्द्यम् अर्कक्षीरे दिनावधि /
वेष्टयेद् भानुपत्रैश्च चक्राकारं तु कारयेत् // र्मञ्ज्_३.४६ //
कुञ्जराख्ये पुटे पाच्यं सप्तवारं पुनः पुनः /
ततो वटजटाक्वाथैस्तद्वद्देयं पुटत्रयम् // र्मञ्ज्_३.४७ //
म्रियते नाम सन्देहः सर्वरोगेषु योजयेत् /

धान्याभ्रकं समादाय मुस्तक्वाथैः पुटत्रये // र्मञ्ज्_३.४८ //
तद्वत्पुनर्नवानीरैः कासमर्दरसैस् तथा /
नागवल्लीरसैर्युक्तं सूर्यक्षीरैः पृथक्पृथक् // र्मञ्ज्_३.४९ //
दिने दिने मर्दयित्वा क्वाथैर्वटजटोद्भवैः /
दत्त्वा पुटत्रयं पश्चात् त्रिपुटं मुशलीजलैः // र्मञ्ज्_३.५० //
त्रिर् गोक्षुरकषायेण त्रिः पुटेद्वानरीरसैः /
मोचाकन्दरसैः पाच्यं त्रिवारं कोकिलाक्षकैः // र्मञ्ज्_३.५१ //
रसैः पुटेत्ततो धेनुः क्षारोदकं पुटं मुहुः /
दध्ना घृतेन मधुना स्वच्छया सितया तथा // र्मञ्ज्_३.५२ //
एकमेकं पुटं दद्याद् अभ्रस्यैवं मृतिर्भवेत् /
सर्वरोगहरं व्योम जायते योगवाहकम् // र्मञ्ज्_३.५३ //
कामिनीमददर्पघ्नं शस्तं पुंस्त्वोपवाहिनाम् /
वृष्यमायुष्करं शुक्रवृद्धिसन्तानकारकम् // र्मञ्ज्_३.५४ //
<अभ्रमारण (३)>
दुग्धत्रयं कुमार्यम्बु गङ्गापत्रं नृमूत्रकम् /

तालं गन्धं समं पश्चान्मर्दयेन्निम्बुकद्रवैः /
द्वित्रैः पुटैः भवेद्भस्म योज्यमेव रसादिषु // र्मञ्ज्_५.२२ //
<सिल्वेर्:: मेद्. प्रोपेर्तिएस्>
शीतं कषायं मधुरमम्लं वातप्रकोपजित् /
दीपनं बलकृत् स्निग्धं गाढाजीर्णविनाशनम् /
आयुष्यं दीर्घरोगघ्नं रजतं लेखनं परम् // र्मञ्ज्_५.२३ //
<ताम्र>
न विषं विषमित्याहुस्ताम्रं तु विषमुच्यते /
<चोप्पेर्:: ८ दोषस्>
एको दोषो विषे ताम्रे त्वष्टौ दोषाः प्रकीर्तिताः // र्मञ्ज्_५.२४ //
भ्रमो मूर्च्छा विदाहश्च स्वेदक्लेदनवान्तयः /
अरुचिश्चित्तसन्ताप एते दोषा विषोपमाः // र्मञ्ज्_५.२५ //
तस्माद्विशुद्धं सङ्ग्राह्यं ताम्रं रोगप्रशान्तये /
<चोप्पेर्:: शोधन>
लवणैर् वज्रदुग्धेन ताम्रपत्रं विलेपयेत् // र्मञ्ज्_५.२६ //
अग्नौ सन्ताप्य निर्गुण्डीरसैः सिक्तं च सप्तधा /
स्नुह्यर्कस्वरसे'प्येवं शुल्बशुद्धिर्भविष्यति // र्मञ्ज्_५.२७ //
<चोप्पेर्:: शोधन>
गोमूत्रेण पचेद्यामं ताम्रपत्रं दृढाग्निना /
शुध्यते नात्र सन्देहो मारणं वाप्यथोच्यते // र्मञ्ज्_५.२८ //
<ताम्रभस्म>
सूतमेकं द्विधा गन्धं यामं कन्यां विमर्दयेत् /
द्वयोस्तुल्यं ताम्रपत्रं स्थाल्या गर्भं निरोधयेत् // र्मञ्ज्_५.२९ //
सम्यङ् मृल्लवणैः सार्द्धं पार्श्वे भस्म निधाय च /
चतुर्यामं पचेच्चुल्ल्यां पात्रपृष्ठे सगोमये // र्मञ्ज्_५.३० //
जलं पुनः पुनर्देयं स्वाङ्गशीतं विचूर्णयेत् /
म्रियते नात्र सन्देहः सर्वयोगेषु योजयेत् // र्मञ्ज्_५.३१ //
<ताम्रभस्म>
चतुर्थांशेन सूतेन ताम्रपत्राणि लेपयेत् /
अम्लपिष्टं द्विगुणितमधोर्ध्वं दापयेद् बलिम् // र्मञ्ज्_५.३२ //
चाङ्गेरीकल्कगर्भे तद्भाण्डे यामं पचेद्दृढम् /
भस्मीभूतं ताम्रपत्रं सर्वयोगेषु योजयेत् // र्मञ्ज्_५.३३ //
<ताम्रभस्म>
जम्बीररससम्पिष्टं रसगन्धकलेपितम् /
शुल्बपत्रं शरावस्थं त्रिपुटैर्याति पञ्चताम् // र्मञ्ज्_५.३४ //
<ताम्रभस्मगुणाः>
ताम्रं तिक्ताम्लमधुरं कषायं शीतलं परम् // र्मञ्ज्_५.३५ //
कफपित्तक्षयं धातुकुष्ठघ्नं च रसायनम् /
नाशयेच्छूलम् अर्शांसि वृक्षमिन्द्राशनिर्यथा // र्मञ्ज्_५.३६ //
<राजरीति, घोषशोधन, मारण>
राजरीतिस्तथा घोषं ताम्रवन्मारयेत् पृथक् /
ताम्रवच्छोधनं तेषां ताम्रवद् गुणकारकम् // र्मञ्ज्_५.३७ //
<नागवङ्गशोधनम्>
नागवङ्गौ च गलितौ रविदुग्धेन सेचयेत् /
त्रिवारं शुद्धिमायाति सच्छिद्रे हण्डिकान्तरे // र्मञ्ज्_५.३८ //
<लेअद्:: मारण (१)>
त्रिभिः कुम्भपुटैर्नागो वासास्वरसमर्दितः /
सा शिला भस्मतामेति तद्रजः सर्वमेहहृत् // र्मञ्ज्_५.३९ //
<नागभस्म>
भूभुजङ्गम् अगस्तिं च पिष्ट्वा पात्रं विलेपयेत् /
तद्रसं विद्रुते नागे वासापामार्गसम्भवम् // र्मञ्ज्_५.४० //
क्षारं विमिश्रयेत्तत्र चतुर्थांशं गुरूक्तितः /
प्रहरं पाचयेच्चुल्ल्यां वासादर्व्या च घट्टिता // र्मञ्ज्_५.४१ //
तत उद्धृत्य तच्चूर्णं वासानीरे विमर्दयेत् /
पुटेत् पुनः समुद्धृत्य तेनैव परिमर्दयेत् // र्मञ्ज्_५.४२ //
एवं सप्त पुटं नागं सिन्दूरं जायते ध्रुवम् /
<नागभस्मगुणाः>
तारस्थो रञ्जनो नागो वातपित्तकफापहः // र्मञ्ज्_५.४३ //
ग्रहणीकुष्ठमेहार्शःप्राणशोषविषापहः /
<तिन्:: मारण (१)>
आभीरं शोधयेदादौ द्रावयेद्धण्डिकान्तरे // र्मञ्ज्_५.४४ //
अपामार्गचतुर्थांशं चूर्णितं मेलयेत्ततः /
स्थूलाग्रया लोहदर्व्या शनैस्तद् अवचालयेत् // र्मञ्ज्_५.४५ //
यावद्भस्मत्वमायाति तावन्मर्द्यं च पूर्ववत् /
तत एकीकृतं सर्वं भवेदङ्गारवर्णकम् // र्मञ्ज्_५.४६ //
नूतनेन शरावेण रोधयेदन्तरे भिषक् /
पश्चात्तीव्राग्निना पक्वं वङ्गभस्म भवेद्ध्रुवम् // र्मञ्ज्_५.४७ //
<तिन्:: मारण (२)>
वङ्गं सतालमर्कस्य पिष्ट्वा दुग्धेन तं पुटेत् /
शुष्काश्वत्थभवैर्वल्कैः सप्तधा भस्मतां व्रजेत् // र्मञ्ज्_५.४८ //
<वङ्गभस्मगुणाः>
वङ्गं तिक्तोष्णकं रूक्षमीषद्वातप्रकोपनम् /
मेहश्लेष्मामयघ्नं च कृमिघ्नं मोहनाशनम् // र्मञ्ज्_५.४९ //
<इरोन्:: शोधन:: अद्रि>
त्रिफलादष्टगुणे तोये त्रिफला षोडशं पलम् /
तत्क्वाथे पादशेषे तु लोहस्य पलपञ्चकम् // र्मञ्ज्_५.५० //
कृत्वा पत्राणि तप्तानि सप्तवारं निषेचयेत् /
एवं प्रलीयते दोषो गिरिजो लोहसम्भवः // र्मञ्ज्_५.५१ //
<लोहभस्म>
शुद्धस्य सूतराजस्य भागो भागद्वयं बलेः /
द्वयोः समं सारचूर्णं मर्दयेत् कन्यकाम्बुना // र्मञ्ज्_५.५२ //
यामद्वयं ततो गोलं स्थापयेत्ताम्रभाजने /
आच्छाद्यैरण्डजैः पत्रैरुष्णो यामद्वयं भवेत् // र्मञ्ज्_५.५३ //
त्रिदिनं धान्यराशिस्थं तं ततो मर्दयेद् दृढम् /
रजस्तद्वस्त्रगलितं नीरे तरति हंसवत् // र्मञ्ज्_५.५४ //
तीक्ष्णं मुण्डं कान्तलोहं निरुत्थं जायते मृतम् /
त्रिफलामधुसंयुक्तम् एतत्सेव्यं रसायनम् // र्मञ्ज्_५.५५ //
<लोहभस्म>
द्वादशांशेन दरदं तीक्ष्णचूर्णस्य मेलयेत् /
कन्यानीरेण सम्मर्द्य यामयुग्मं तु तत्पुटेत् /
पुटेदेवं लोहचूर्णं सप्तधा मरणं व्रजेत् // र्मञ्ज्_५.५६ //
<लोहभस्म>
काकोदुम्बरिकानीरे लोहपत्राणि सेचयेत् /
तप्ततप्तानि षड्वारं कुट्टयेत्तद् उदूखले // र्मञ्ज्_५.५७ //
तत्पञ्चमांशं दरदं क्षिप्त्वा सर्वं विमर्दयेत् /
कुमारीनीरतस् तीक्ष्णं पुटे गजपुटे तथा // र्मञ्ज्_५.५८ //
त्रिवारं त्रिफलाक्वाथैस्तत्सङ्ख्याकैरतन्द्रितः /
एवं चतुर्दशपुटैर्लोहं वारितरं भवेत् // र्मञ्ज्_५.५९ //
<लोहभस्म>
तिन्दूफलस्य मज्जायां खड्गं लिप्त्वातपे खरे /
धारयेत् कांस्यपात्रेण दिनैकेन पुटत्यलम् // र्मञ्ज्_५.६० //
लेपं पुनः पुनः कुर्याद् दिनान्ते तत्प्रपेषयेत् /
त्रिफलाक्वाथसंयुक्तं दिनैकेन मृतिर्भवेत् // र्मञ्ज्_५.६१ //
<लोहभस्म>
लोहं पत्रमतीव तप्तमसकृत्क्वाथे क्षिपेत्त्रैफले चूर्णीभूतमतो भवेत् त्रिफलजे क्वाथे पचेत् गोजले /
मत्स्याक्षीत्रिफलारसेन पुटयेद्यावन्निरुत्थं भवेत् पश्चादाज्यमधुप्लुतं सुपुटितं भस्म भवेद् आयसम् // र्मञ्ज्_५.६२ //
<इरोन्:: छेच्किन्ग् थे वारितर स्तते>
सर्वमेतन्मृतं लोहं ध्मातव्यं मृतपञ्चके /
यद्येव स्यान्निरुत्थानं सत्यं वारितरं भवेत् // र्मञ्ज्_५.६३ //
<लोहनिरुत्थभस्म>
गन्धकं तुत्थकं लोहं तुल्यं खल्वे विमर्दयेत् /
दिनैकं कन्यकाद्रावै रुद्ध्वा गजपुटे पचेत् /
इत्येवं सर्वलोहानां कर्तव्येत्थं निरुत्थितिः // र्मञ्ज्_५.६४ //
<लोहभस्मगुणाः>
कृष्णायसो'थ शूलार्शःकुष्ठपाण्डुत्वमेहनुत् /
वयःस्थं गुरु चक्षुष्यं सरं मेदोगदापहम् // र्मञ्ज्_५.६५ //
आयुःप्रदाता बलवीर्यकर्ता रोगस्य हर्ता मदनस्य कर्ता /
अयःसमानं नहि किञ्चिद् अन्यद् रसायनं श्रेष्ठतमं हि जन्तो // र्मञ्ज्_५.६६ //
कूष्माण्डं तिलतैलं च माषान्नं राजकं तथा /
मद्यमम्लरसं चैव त्यजेल्लोहस्य सेवकः // र्मञ्ज्_५.६७ //
<मुण्ड>
ये गुणा मारिते मुण्डे ते गुणा मुण्डकिट्टके /
तस्मात् सर्वत्र मण्डूरं रोगशान्त्यै नियोजयेत् // र्मञ्ज्_५.६८ //
शतोत्थमुत्तमं किट्टं मध्यमाशीतिवार्षिकम् /
अधमं षष्टिवर्षीयं ततो हीनं विषोपमम् // र्मञ्ज्_५.६९ //
दग्धाक्षकाष्ठैर् मलम् आयसं तु गोमूत्रनिर्वापितमष्टवारान् /
विचूर्ण्य लीढं मधुनाचिरेण नृणां क्षयं पाण्डुगदं निहन्ति // र्मञ्ज्_५.७० //
किट्टाद्दशगुणं मुण्डं मुण्डात्तीक्ष्णं शताधिकम् /
तीक्ष्णाल्लक्षगुणं कान्तं भक्षणात् कुरुते गुणम् // र्मञ्ज्_५.७१ //


र्मञ्ज्, ६
क्षीराब्धेरुत्थितं देवं पीतवस्त्रं चतुर्भुजम् /
वन्दे धन्वन्तरिं नित्यं नानागदनिषूदनम् // र्मञ्ज्_६.१ //
यथा गुरुमुखं श्रुत्वा सानुभूतं च तद्रसम् /
स रसः प्रोच्यते ह्यत्र व्याधिनाशनहेतवे // र्मञ्ज्_६.२ //
मुक्त्वैकं रसवैद्यं च लाभपूजायशस्विनम् /
तृणकाष्ठौषधैर् वैद्यः को लभेत वराटिकाम् // र्मञ्ज्_६.३ //
यस्य रोगस्य यो योगो मुनिभिः परिकीर्तितः /
तत्तद्योगसमायुक्तं भिषक् सूतं प्रयोजयेत् // र्मञ्ज्_६.४ //
मात्राधिकं न सेवेत रसं वा विषम् औषधम् /
त्र्याधिबद्धं च कोष्ठं च वीक्ष्य मात्रां प्रयोजयेत् // र्मञ्ज्_६.५ //
<रत्नगर्भपटोली>
रसं वज्रं हेम तारं नागं लोहं च ताम्रकम् /
तुल्यांशमारिते योज्यं मुक्तामाक्षिकविद्रुमम् // र्मञ्ज्_६.६ //
शङ्खं च तुल्यतुल्यांशं सप्ताहं चित्रकद्रवैः /
मर्दयित्वा विचूर्ण्याथ तेनापूर्य वराटिकाम् // र्मञ्ज्_६.७ //
टङ्कणं रविदुग्धेन पिष्ट्वा मूषां च बन्धयेत् /
मृद्भाण्डे च निरुध्याथ सम्यग्गजपुटे पचेत् // र्मञ्ज्_६.८ //
स्वाङ्गशीतं समुद्धृत्य सूक्ष्मचूर्णानि कारयेत् /
आदाय चूर्णयेत्सर्वं निर्गुण्ड्याः सप्तभावनाः // र्मञ्ज्_६.९ //
आर्द्रकस्य रसैः सप्त चित्रकस्यैकविंशतिः /
द्रवैर्भाव्यं ततः शोष्यं देयं गुञ्जाचतुष्टयम् // र्मञ्ज्_६.१० //
क्षयरोगं निहन्त्याशु साध्यासाध्यं न संशयः /
योजयेत् पिप्पलीक्षौद्रैः सघृतैर्मरिचेन च // र्मञ्ज्_६.११ //
महारोगाष्टके कासे ज्वरे श्वासे'तिसारके /
पोटलीरत्नगर्भो'यं योगवाहेषु योजयेत् // र्मञ्ज्_६.१२ //
<मृगाङ्करसः>
स्याद्रसेन समं हेम मौक्तिकं द्विगुणं भवेत् /
गन्धकं च समं तेन रसपादस्तु टङ्कणम् // र्मञ्ज्_६.१३ //
सर्वं तद्गोलकं कृत्वा काञ्जिकेनावशोषयेत् /
भाण्डे लवणपूर्णे च पचेद्यामचतुष्टयम् // र्मञ्ज्_६.१४ //

सूतं गन्धकचित्रकं त्रिकटुकं मुस्ता विषं त्रैफलम् एतेभ्यो द्विगुणां गुडेन गुटिकां गुञ्जाप्रमाणां हरेत् /
कुष्ठाष्टादश वायुशूलमुदरं शोषप्रमेहादिकं रोगानीककरीन्द्रदर्पदलने ख्यातो हि पञ्चाननः // र्मञ्ज्_६.८१ //
<मृतसञ्जीवनीरसः>
म्लेच्छस्य भागाश्चत्वारो जैपालस्य त्रयो मताः /
द्वौ भागौ टङ्कणस्यैव भागैकममृतस्य च // र्मञ्ज्_६.८२ //
तत्सर्वं मर्दयेत् सूक्ष्मं शुष्कं यामं भिषग्वरः /
शृङ्गवेराम्बुना देयो व्योषचित्रकसैन्धवैः /
गुञ्जाद्वयमितस्तापं हरत्येष विनिश्चयः // र्मञ्ज्_६.८३ //
घनसारेण युक्तेन चन्दनेन विलेपयेत् // र्मञ्ज्_६.८४ //
विदध्यात्कांस्यपात्रेण जीवयेद्रोगिणं भिषक् /
शाल्यन्नं तक्रसहितं भोजयेद् बिल्वसंयुतम् // र्मञ्ज्_६.८५ //
सन्निपाते महाघोरे त्रिदोषे विषमज्वरे /
आमवाते वातशूले गुल्मे प्लीह्नि जलोदरे // र्मञ्ज्_६.८६ //
शीतपूर्वे दाहपूर्वे विषमे सततज्वरे /
अग्निमान्द्ये च वाते च प्रयोज्यो'यं रसेश्वरः // र्मञ्ज्_६.८७ //
मृतसञ्जीवनं नाम ख्यातो'यं रससागरे // र्मञ्ज्_६.८८ //
<रविसुन्दररसः>
द्विभागतालेन हतं च ताम्रं रसं च गन्धं च विषं समं स्यात् /
जयपालबीजं द्विगुणं च दद्यात् त्रिसप्तवारेण दिवाकरांशौ // र्मञ्ज्_६.८९ //
विमर्द्य निम्बस्वरसेन चूर्णं गुञ्जैकमानं सितया समेतम् /
ज्वराङ्कुशो'यं रविसुन्दराख्यो ज्वरान्निहन्त्यष्टविधान् समग्रान् // र्मञ्ज्_६.९० //
<सन्निपातभैरवरसः>
ताम्रगन्धरसश्वेतस्पन्दामरिचपूतनाः /
समीनपित्तजैपालास्तुल्या एकत्र मर्दिताः // र्मञ्ज्_६.९१ //
गुञ्जाचतुष्टयं चास्या नवज्वरहरः परः /
ज्वराङ्कुशः सन्निपातभैरवो'यं प्रकाशितः // र्मञ्ज्_६.९२ //
<भस्मेश्वररसः>
भस्मषोडशनिष्कं स्यादारण्योपलकोद्भवम् /
निष्कत्रयं च मरिचं विषं निष्कं च चूर्णयेत् // र्मञ्ज्_६.९३ //
अयं भस्मेश्वरो नाम सन्निपातनिकृन्तनः /
पञ्चगुञ्जामितो भक्षेदार्द्रकस्य रसेन च // र्मञ्ज्_६.९४ //
<प्रतापलङ्केश्वररसः>
अपामार्गस्य मूलस्य चूर्णं चित्रकमूलजैः /
वल्कलैर्मर्दयित्वा च रसं वस्त्रेण गालयेत् // र्मञ्ज्_६.९५ //
तेन सूतसमं गन्धमभ्रकं दरदं विषम् /
टङ्कणं तालकं चैव मर्दयेद् दिनसप्तकम् // र्मञ्ज्_६.९६ //
त्रिदिनं मुशलीकन्दैर्भावयेद् घर्मरक्षितम् /
मूषां च गोस्तनाकारामापूर्य परिढक्कयेत् // र्मञ्ज्_६.९७ //
सप्तभिर् मृत्तिकावस्त्रैर् वेष्टयित्वा पुटेल्लघु /
रसतुल्यं लोहवङ्गौ रजतं ताम्रकं तथा // र्मञ्ज्_६.९८ //
मधूकसारजलदौ रेणुका गुग्गुलुः शिला /
चव्यकं च समांशं स्याद्भागार्द्धं शोधितं विषम् // र्मञ्ज्_६.९९ //
तत्सर्वं मर्दयेत्खल्वे भावयेद्विषनीरतः /
आतपे सप्तधा तीव्रे मर्दयेद् घटिकाद्वयम् // र्मञ्ज्_६.१०० //
कटुत्रयकषायेण कनकस्य रसेन च /
फलत्रयकषायेण मुनिपुष्परसेन च // र्मञ्ज्_६.१०१ //
समुद्रफलनीरेण विजयावारिणा तथा /
चित्रकस्य कषायेण ज्वालामुख्या रसेन च // र्मञ्ज्_६.१०२ //
प्रत्येकं सप्तधा भाव्यं तद्वत्पिष्टं च भावयेत् /
सर्वस्य समभागेन विषेण परिधूपयेत् // र्मञ्ज्_६.१०३ //
दिनं विमर्दयित्वाथ रक्षयेत्कूपिकान्तरे /
गुञ्जैकं वह्णिनीरेण शृङ्गवेररसेन वा // र्मञ्ज्_६.१०४ //
प्रदद्याद्रोगिणे तीव्रमोहविस्मृतिशान्तये /
शस्त्रेण तालुमाहत्य मर्दयेदार्द्रनीरतः // र्मञ्ज्_६.१०५ //
नोद्घटन्ते यदा दन्तास्तदा कुर्यादमुं विधिम् /
सेचयेन्मन्त्रयित्वाथ वारां कुम्भशतैर् मुहुः // र्मञ्ज्_६.१०६ //
भोजनेच्छा यदा तस्य जायते रोगिणस्तदा /
दध्योदनं सितायुक्तं दद्यात्तक्रं सजीरकम् // र्मञ्ज्_६.१०७ //
पाने पानं सितायुक्तं यदीच्छति तदा ददेत् /
एवं कृते न शान्तिः स्यात्तापस्य रसजस्य च // र्मञ्ज्_६.१०८ //
सचन्द्रचन्दनरसोल्लेपनं कुरु शीतलम् /
तूलिकामल्लिकाजातीपुन्नागबकुलावृताम् // र्मञ्ज्_६.१०९ //
विधाय शय्यां तत्रस्थं लेपयेच्चन्दनैर् मुहुः /
हावभावविलासोक्तिकटाक्षचञ्चलेक्षणैः // र्मञ्ज्_६.११० //
पीनोत्तुङ्गकुचोत्पीडैः कामिनीपरिरम्भणैः /
रम्यवीणानिनादाद्यैर् गायनैः श्रवणामृतैः // र्मञ्ज्_६.१११ //
पुण्यश्लोकपुराणानां कथासम्भाषणैः शुभैः /
एभिः प्रकारैस्तापस्य जायते शमनं परम् // र्मञ्ज्_६.११२ //
वर्जयेन्मैथुनं तावद्यावन्नो बलवान् भवेत् /
दद्याद्वातादिरोगेषु सिन्धुगुग्गुलवह्णिभिः // र्मञ्ज्_६.११३ //
दद्यात्कणामाक्षिकाभ्यां कामलाक्षयपाण्डुषु /
तत्तद्रोगानुपानेन सर्वरोगेषु योजयेत् // र्मञ्ज्_६.११४ //
अयं प्रतापलङ्केशः सन्निपातनिकृन्तनः // र्मञ्ज्_६.११५ //
<महोदधिरसः>
सूतकं गन्धकं लोहं विषं चापि वराटकम् /
ताम्रकं वङ्गभस्माथ अभ्रकं च समांशकम् // र्मञ्ज्_६.११६ //
त्रिकटु पत्रमुस्तं च विडङ्गं नागकेशरम् /
रेणुकामलकं चैव पिप्पलीमूलमेव च // र्मञ्ज्_६.११७ //
एषां च द्विगुणं भागं मर्दयित्वा प्रयत्नतः /
भावना तत्र दातव्या गजपिप्पलिकाम्बुना // र्मञ्ज्_६.११८ //
मात्रा चणकमाना तु वटिकेयं प्रकीर्तिता /
श्वासं हन्ति तथा कासम् अर्शांसि च भगन्दरम् // र्मञ्ज्_६.११९ //
हृच्छूलं पार्श्वशूलं च कर्णरोगं कपालिकम् /
हरेत्सङ्ग्रहणीरोगमष्टौ च जाठराणि च /
प्रमेहविंशतिं चैव अश्मरीं च चतुर्विधाम् // र्मञ्ज्_६.१२० //
न चान्नपाने परिहारमस्ति न शीतवाताध्वनि मैथुने च /
यथेष्टचेष्टाभिरतः प्रयोगे नरो भवेत्काञ्चनराशिगौरः // र्मञ्ज्_६.१२१ //
<मदोदधिवटी>
एकैकं विषसूतं च जाती टङ्कं द्विकं द्विकम् /
कृष्णात्रिकं विश्वषट्कं दग्धं कपर्दिकाद्विकम् // र्मञ्ज्_६.१२२ //
देवपुष्पं बाणमितं सर्वं सम्मर्द्य यत्नतः /
महोदध्याख्यवटिका नष्टस्याग्नेश्च दीपनी // र्मञ्ज्_६.१२३ //
<उन्मत्तरसः>
रसं च गन्धकं चैव धत्तूरफलजैर् द्रवैः /
मर्दयेद् दिनमेकं च तुल्यं त्रिकटुकं क्षिपेत् // र्मञ्ज्_६.१२४ //
उन्मत्ताख्यरसो नाम सन्निपातनिकृन्तनः /
कस्तेन न कृतो धर्मः कां च पूजां न सो'र्हति /
सन्निपातार्णवे मग्नं यो'भ्युद्धरति देहिनम् // र्मञ्ज्_६.१२५ //
<सञ्ज्ञाकरणरसः>
वचा रसोनकटुकं सैन्धवं बृहतीफलम् /
रुद्राक्षं मधुसारं च फलं सामुद्रकामृतम् // र्मञ्ज्_६.१२६ //
समभागानि चैतानि ह्यर्कक्षीरेण भावयेत् /
भावयेन्मत्स्यपित्तेन त्रिवारं चूर्णयेत्ततः // र्मञ्ज्_६.१२७ //
धमनं कथितं श्रेष्ठं सन्निपाते सुदारुणे /
कफोल्वणे'तिवाते च अपस्मारे हलीमके // र्मञ्ज्_६.१२८ //
शिरोरोगे कर्णरोगे नेत्ररोगे विधानतः /
दापयेद्घ्राणछिद्राभ्यां सञ्ज्ञाकरणम् उत्तमम् // र्मञ्ज्_६.१२९ //
<चन्द्रशेखररसः>
शुद्धं सूतं समं गन्धं मरिचं टङ्कणं तथा // र्मञ्ज्_६.१३० //
चतुस्तुल्या सिता योज्या मत्स्यपित्तेन भावयेत् /
त्रिदिनं मर्दयेत्तेन रसो'यं चन्द्रशेखरः // र्मञ्ज्_६.१३१ //
द्विगुञ्जमार्द्रकद्रावैर्देयं शीतोदकं पुनः /
तक्रभक्तं च वृन्ताकं पथ्यं तत्र निधापयेत् // र्मञ्ज्_६.१३२ //
<कनकसुन्दररसः>
हिङ्गुलं मरिचं गन्धं पिप्पलीं टङ्कणं विषम् // र्मञ्ज्_६.१३३ //
कनकस्य च बीजानि समांशं विजयारसैः /
मर्दयेद्याममात्रं तु चणमात्रा वटी कृता // र्मञ्ज्_६.१३४ //
भक्षणाद्ग्रहणीं हन्याद्रसः कनकसुन्दरः /
अग्निमान्द्यं ज्वरं तीव्रमतिसारं च नाशयेत् // र्मञ्ज्_६.१३५ //
ग्रहणीदोषिणां तक्रं दीपनं ग्राहिलाघवम् /
पथ्यं मधुरपाकित्वान्न च पित्तप्रकोपनम् // र्मञ्ज्_६.१३६ //
<राववाणरसः>
सूतकं गन्धकं चैव शाणं शाणं च गृह्यते // र्मञ्ज्_६.१३७ //
दरदं टङ्कणं चैव मरिचं च विषं तथा /
चत्वार औषधयः सर्वे द्विद्विटङ्कं च कथ्यते // र्मञ्ज्_६.१३८ //
जैपालबीजं संयोज्यं टङ्कं च दिक्प्रमाणतः /
तिन्तिडीरससम्मर्द्यं गुञ्जामात्रवटी कृता // र्मञ्ज्_६.१३९ //
तुलसीपत्रसंयुक्ता सर्वे च विषमज्वराः /
ऐकाहिकं द्व्याहिकं च त्र्याहिकं च चतुर्थकम् // र्मञ्ज्_६.१४० //
शीतदाह्यादिकं सर्वं नाशयति च वेगतः /
पथ्यं दुग्धौदनं देयं दधिभक्तं च भोजनम् // र्मञ्ज्_६.१४१ //
रामवाणरसो नाम सर्वरोगप्रणाशकः // र्मञ्ज्_६.१४२ //
<चन्द्रप्रभावटी>
मृतं सूतं मृतं ताम्रं मृतं स्वर्णं समं समम् /
तुल्यं च खादिरं सारं तथा मोचरसं क्षिपेत् // र्मञ्ज्_६.१४३ //
द्रवैः शाल्मलिमूलोत्थैर् मर्दयेत् प्रहरद्वयम् /
चणमात्रां वटीं भक्षेन्निष्कैकं जीरकैः सह /
त्रिदोषोत्थमतीसारं सज्वरं नाशयेद्ध्रुवम् // र्मञ्ज्_६.१४४ //
<चित्राम्बररसः>
शुद्धसूतं मृतं चाभ्रं गन्धकं मर्दयेत्समम् /
लोहपात्रे घृताभ्यक्ते यामं मृद्वग्निना पचेत् // र्मञ्ज्_६.१४५ //
चालयेल्लोहदण्डेन ह्यवतार्य विभावयेत् /
त्रिदिनं जीरकैः क्वाथैर् मासैकं भक्षयेन्नरः // र्मञ्ज्_६.१४६ //
रसश् चित्राम्बरो नाम ग्रहणीं रक्तसंयुताम् /
शमयेदनुपानेन आमशूलं प्रवाहिकाम् // र्मञ्ज्_६.१४७ //
<ग्रहणीकपाटरसः>
तारमौक्तिकहेमायः सारश् चैकैकभागिकाः /
द्विभागो गन्धकः सूतस्त्रिभागो मर्दयेद्दिनम् // र्मञ्ज्_६.१४८ //
कपित्थस्वरसैर्गाढं मृगशृङ्गे ततः क्षिपेत् /
पुटेन्मध्यपुटेनैव तत उद्धृत्य मर्दयेत् // र्मञ्ज्_६.१४९ //
बलारसैः सप्तधैवम् अपामार्गरसैस् त्रिधा /
लोध्रप्रतिविषामुस्ताधातकीन्द्रयवामृताः // र्मञ्ज्_६.१५० //
प्रत्येकमेषां स्वरसैर्भावना स्यात्त्रिधा त्रिधा /
माषमात्ररसो देयो मधुना मरिचैः सह // र्मञ्ज्_६.१५१ //
हन्यात्सर्वानतीसारान्ग्रहणीं पञ्चधापि च /
कपाटो ग्रहणीनाम रसो'यं चाग्निदीपनः // र्मञ्ज्_६.१५२ //
<ग्रहणीकपाटरसः>
मुक्तासुवर्णं रसगन्धटङ्कणं घनं कपर्दो'मृततुल्यभागम् /
सर्वैः समं शङ्खकचूर्णयुक्तं खल्वे च भाव्यो'तिविषाद्रवेण // र्मञ्ज्_६.१५३ //
गोलं च कृत्वा मृत्कर्पटस्थं संरुध्य भाण्डे हि पचेद् दिनार्धम् /
सुस्वाङ्गशीतो रस एष भाव्यो धत्तूरवह्णिमुशलीद्रवैश्च // र्मञ्ज्_६.१५४ //
लोहस्य पात्रे परिपाचितश्च सिद्धो भवेत् सङ्ग्रहणीकपाटः /
वातोत्तरायां मरिचाज्ययुक्तः पित्तोत्तरायां मधुपिप्पलीभिः // र्मञ्ज्_६.१५५ //
श्लेष्मोत्तरायां विजयारसेन कटुत्रयेणापि युतो ग्रहण्याम् // र्मञ्ज्_६.१५६ //
क्षये ज्वरे'प्यर्शसि विड्विकारे सामातिसारे'रुचिपीनसे च /
मोहे च कृच्छ्रे गतधातुवृद्धौ गुञ्जाद्वयं चापि महामयघ्नम् // र्मञ्ज्_६.१५७ //
<वज्रकपाटरसः>
मृतसूताभ्रकं गन्धं यवक्षारं सटङ्कणम् // र्मञ्ज्_६.१५८ //
अग्निमन्थं वचां कुर्यात् सूततुल्याम् इमां सुधीः /
ततो जयन्तीजम्बीरभृङ्गद्रावैर् विमर्दयेत् // र्मञ्ज्_६.१५९ //
त्रिवासरं ततो गोलं कृत्वा संशोष्य धारयेत् /
लोहपात्रे च लवणं अथोपरि निधापयेत् // र्मञ्ज्_६.१६० //
अधोवह्निं शनैः कुर्याद् यामार्धं च तद् उद्धरेत् /
रसतुल्यामतिविषां दद्यान्मोचरसं तथा // र्मञ्ज्_६.१६१ //
कपित्थविजयाद्रावैर्भावयेत् सप्तधा भिषक् /
धातकीन्द्रयवमुस्तालोध्रबिल्वगुडूचिकाः // र्मञ्ज्_६.१६२ //
एतद्रसैर्भावयित्वा वारैकं च विशोषयेत् /
रसवज्रं कपाटाख्यं माषैकं मधुना लिहेत् // र्मञ्ज्_६.१६३ //
वह्निः शुण्ठी विडङ्गापि बिल्वं च लवणं समम् /
पिबेद् उष्णाम्बुना चानु सर्वजां ग्रहणीं जयेत् // र्मञ्ज्_६.१६४ //
<विजयभैरवरसः>
सूतकं गन्धकं लोहं विषं चित्रकमभ्रकम् /
विडङ्गं रेणुका मुस्ता एला केशरपत्रकम् /
फलत्रयं त्रिकटुकं शुल्बभस्म तथैव च // र्मञ्ज्_६.१६५ //
एतानि समभागानि द्विगुणो दीयते गुडः /
कासे श्वासे क्षये गुल्मे प्रमेहे विषमज्वरे // र्मञ्ज्_६.१६६ //
सूतायां ग्रहणीमान्द्ये शूले पाण्ड्वामये तथा /
हस्तपादादिरोगेषु गुटिकेयं प्रशस्यते // र्मञ्ज्_६.१६७ //
<आनन्दभैरवरसः>
दरदं वत्सनाभं च मरिचं टङ्कणं कणा // र्मञ्ज्_६.१६८ //
चूर्णयेत्समभागेन रसो ह्यानन्दभैरवः /
गुञ्जैकं तु द्विगुञ्जं वा बलं ज्ञात्वा प्रयोजयेत् // र्मञ्ज्_६.१६९ //
मधुना लेहयेच्चानु कुटजस्य फलत्वचम् /
चूर्णितं कर्षमात्रं तु त्रिदोषस्यातिसारजित् // र्मञ्ज्_६.१७० //
दध्यन्नं दापयेत्पथ्यं गव्याजं तक्रमेव च /
पिपासायां जलं शीतं हिता च विजया निशि // र्मञ्ज्_६.१७१ //
<मेघडम्बररसः>
तण्डुलीयजलैः पिष्टं सूततुल्यं च गन्धकम् /
अन्धमूषागतं पक्त्वा भूधरे भस्मतां नयेत् // र्मञ्ज्_६.१७२ //
दशमूलकषायेण भावयेत्प्रहरद्वयम् /
गुञ्जाद्वयं हरत्याशु हिक्कां कासं ज्वरं तथा /
अनुपानेन दातव्यो रसो'यं मेघडम्बरः // र्मञ्ज्_६.१७३ //
<त्रिगुणाख्यरसः>
गन्धकाद् द्विगुणं सूतं शुद्धं मृद्वग्निना क्षणम् // र्मञ्ज्_६.१७४ //
पक्त्वावतार्य सञ्चूर्ण्य चूर्णतुल्याभयायुतम् /
सप्तगुञ्जामितं खादेद्वर्धयेच्च दिने दिने // र्मञ्ज्_६.१७५ //
गुञ्जैकं च क्रमेणैव यावत्स्याद् एकविंशतिः /
क्षीराज्यशर्करामिश्रं शाल्यन्नं पथ्यमाचरेत् // र्मञ्ज्_६.१७६ //
कम्पवातप्रशान्त्यर्थं निर्वाते निवसेत्सदा /
त्रिगुणाख्यो रसो नाम त्रिपक्षात्कम्पवातनुत् // र्मञ्ज्_६.१७७ //
<वातरिपुरसः>
सूतहाटकवज्राणि तारं लोहं च माक्षिकम् /
तालं नीलाञ्जनं तुत्थमब्धिफेनं समांशकम् // र्मञ्ज्_६.१७८ //
पञ्चानां लवणानां च भागमेकं विमर्दयेत् /
वज्रक्षीरैर् दिनैकं तु रुद्ध्वा तं भूधरे पुटेत् // र्मञ्ज्_६.१७९ //
माषैकमार्द्रकद्रावैर् लेहयेद् वातनाशनम् /
पिप्पलीमूलजं क्वाथं सकृष्णमनुपाययेत् // र्मञ्ज्_६.१८० //
सर्ववातविकारांस्तु निहन्त्याक्षेपकादिकान् /
रसः सर्वत्र विख्यातो नाम वातरिपुः स्मृतः // र्मञ्ज्_६.१८१ //
<वातगजाङ्कुशरसः>
मृतं लोहं सूतगन्धं ताम्रतालकमाक्षिकम् /
पथ्यां शृङ्गीविषं त्र्यूषम् अग्निमन्थं च टङ्कणम् // र्मञ्ज्_६.१८२ //
तुल्यं खल्वे दिनं मर्द्य मुण्डीनिर्गुण्डिजैर् द्रवैः /
द्विगुञ्जां वटिकां खादेत्सर्ववातप्रशान्तये /
साध्यासाध्यं निहन्त्याशु रसो वातगजाङ्कुशः // र्मञ्ज्_६.१८३ //
<अम्लपित्तान्तकरसः>
मृतसूताभ्रलोहानां तुल्यां पथ्यां विचूर्णयेत् /
माषत्रयं लिहेत्क्षौद्रैरम्लपित्तप्रशान्तये // र्मञ्ज्_६.१८४ //
<अग्निकुमाररसः>
सूतं गन्धं च नागानां चूर्णं हंसाङ्घ्रिवारिणा /
दिनं घर्मे विमर्द्याथ गोलिकां तस्य योजयेत् // र्मञ्ज्_६.१८५ //
काचकुप्यां च संवेष्ट्य तां त्रिभिर् मृत्पुटैर् दृढम् /
मुखं संरुध्य संशोष्य स्थापयेत्सिकताह्वये // र्मञ्ज्_६.१८६ //
सार्धं दिनं क्रमेणाग्निं ज्वालयेत् तदधस्ततः /
स्वाङ्गशीतं समुद्धृत्य षडंशेनामृतं क्षिपेत् // र्मञ्ज्_६.१८७ //
मरिचान्यर्द्धभागेन समं वास्याथ मर्दयेत् /
अयमग्निकुमाराख्यो रसो मात्रास्य रक्तिका // र्मञ्ज्_६.१८८ //
ताम्बूलीरससंयुक्तो हन्ति रोगानमून् अयम् /
वातरोगान् क्षयं श्वासं कासं पाण्डुकफोल्बणम् // र्मञ्ज्_६.१८९ //
अग्निमान्द्यं सन्निपातं पथ्यं शाल्यादिकं लघु /
जलयोगप्रयोगो'पि शस्तस्तापप्रशान्तये // र्मञ्ज्_६.१९० //
<अग्निकुमाररसः>
टङ्कणं रसगन्धौ च समभागं त्रयो विषम् /
कपर्दिसर्जिकाक्षारमागधीविश्वभेषजम् // र्मञ्ज्_६.१९१ //
पृथक्पृथक् कर्षमात्रं त्वष्टभागं मरीचकम् /
जम्बीराम्लैर्दिनं पिष्टं भवेदग्निकुमारकः /
विषूचिशूलवातादिवह्णिमान्द्यप्रशान्तये // र्मञ्ज्_६.१९२ //
<लीलाविलासरसः>
रसो रविर्व्योम बलिः सुलोहं धात्र्यक्षनीरैस्त्रिदिनं विमर्द्य /
तदल्पघृष्टं मृदुमार्कवेण सम्मर्दयेद् अस्य च वल्लयुग्मम् // र्मञ्ज्_६.१९३ //
हन्त्यम्लपित्तं मधुनावलीढं लीलाविलासो रसराज एषः /
दुग्धं सकूष्माण्डरसं सधात्रीफलं शनैस्तत्सहितं भजेद्वा // र्मञ्ज्_६.१९४ //
<मन्थानभैरवरसः>
मृतं सूतं मृतं ताम्रं हिङ्गुपुष्करमूलकम् /
सैन्धवं गन्धकं तालं टङ्कणं चूर्णयेत्समम् // र्मञ्ज्_६.१९५ //
पुनर्नवादेवदारुनिर्गुण्डीतण्डुलीयकैः /
तिक्तकोशातकीद्रावैर्दिनैकं मर्दयेद् दृढम् // र्मञ्ज्_६.१९६ //
माषमात्रं लिहेत्क्षौद्रै रसो मन्थानभैरवः /
कफरोगप्रशान्त्यर्थं निम्बक्वाथं पिबेदनु // र्मञ्ज्_६.१९७ //
<क्रव्यादरसः>
पलं रसस्य द्विपलं बलेः स्याच्छुल्वायसी चार्धपलप्रमाणे /
सञ्चूर्ण्य सर्वं द्रुतम् अग्नियोगाद् एरण्डपत्रेषु निवेशनीयम् // र्मञ्ज्_६.१९८ //
पिष्ट्वाथ तां पर्पटिकां निदध्याल्लोहस्य पात्रे वरपूतम् अस्मिन् /
जम्बीरजं पक्वरसं पलानां शतं नियोज्याग्निमथाम्लमात्रम् // र्मञ्ज्_६.१९९ //
जीर्णे रसे भावितमेतदेतैः सपञ्चकोलोद्भववारिपूरैः /
सवेतसाम्लैः शतमत्र योज्यं समं रजष् टङ्कणजं सुभृष्टम् // र्मञ्ज्_६.२०० //
विडं ददर्धं मरिचं समं च तत्सप्तधार्द्रं चणकाम्लवारि /
क्रव्यादनामा भवति प्रसिद्धो रसः सुमन्थानकभैरवोक्तः /
माषद्वयं सैन्धवतक्रपीतम् एतसुधन्यं खलु भोजनान्ते // र्मञ्ज्_६.२०१ //
गुरूणि मांसानि पयांसि पिष्टीघृतानि खाद्यानि फलानि वेगात् /
मात्रातिरिक्तान्यपि सेवितानि यामद्वयाज्जारयति प्रसिद्धः // र्मञ्ज्_६.२०२ //
<अग्नितुण्डीवटी>
शुद्धं सूतं विषं गन्धमजमोदाफलत्रयम् /
सर्जिक्षारं यवक्षारं वह्णिसैन्धवजीरकम् // र्मञ्ज्_६.२०३ //
सौवर्चलं विडङ्गानि सामुद्रं टङ्कणं समम् /
विषमुष्टिं सर्वतुल्यं जम्बीराम्लेन मर्दयेत् // र्मञ्ज्_६.२०४ //
मरिचाभां वटीं खादेद्वह्णिमान्द्यप्रशान्तये // र्मञ्ज्_६.२०५ //
<आनन्दोदयरसः>
पारदं गन्धकं लोहमभ्रकं विषमेव च /
समांशं मरिचं चाष्टौ टङ्कणं च चतुर्गुणम् // र्मञ्ज्_६.२०६ //
भृङ्गराजरसैः सप्त भावनाश्चाम्लदाडिमैः /
गुञ्जाद्वयं पर्णखण्डैर्हन्ति सायं तु भक्षितः // र्मञ्ज्_६.२०७ //
वातश्लेष्मोद्भवान्रोगान्मन्दाग्निग्रहणीज्वरान् /
अरुचिं पाण्डुतां चैव जयेदचिरसेवनात् // र्मञ्ज्_६.२०८ //
<चिन्तामणिरसः>
रसं गन्धं मृतं शुल्बं मृतमभ्रं फलत्रिकम् /
त्र्यूषणं बीजजैपालं समं खल्वे विमर्दयेत् // र्मञ्ज्_६.२०९ //
द्रोणपुष्पीरसैर्भाव्यं शुष्कं तद्वस्त्रगालितम् /
चिन्तामणिरसो'प्येष अजीर्णानां प्रशस्यते // र्मञ्ज्_६.२१० //
ज्वरम् अष्टविधं हन्ति सर्वशूलेषु शस्यते /
गुञ्जैको वा द्विगुञ्जो वा आमरोगहरः परः // र्मञ्ज्_६.२११ //
<राजवल्लभरसः>
रसनिष्कैकगन्धैकं निष्कमात्रः प्रदीपनः // र्मञ्ज्_६.२१२ //
सार्धं पलं प्रदातव्यं चूलिकालवणं भिषक् /
खल्वे सम्मर्दयेत्तत्तु शुष्कवस्त्रेण गालयेत् // र्मञ्ज्_६.२१३ //
माषमात्रं प्रदातव्यो मुक्तमांसादिजारकः /
अजीर्णेषु त्रिदोषेषु देयो'यं राजवल्लभः // र्मञ्ज्_६.२१४ //
<त्रिनेत्राख्यरसः>
टङ्कणं हारिणं शृङ्गं स्वर्णं शुल्बं मृतं रसम् /
दिनैकमार्द्रकद्रावैर्मर्द्य रुद्ध्वा पुटे पचेत् // र्मञ्ज्_६.२१५ //
त्रिनेत्राख्यो रसो नाम्ना माषैकं मधुसर्पिषा /
सैन्धवं जीरकं हिङ्गुमध्वाज्याभ्यां लिहेदनु /
पक्तिशूलहरं ख्यातं याममात्रान्न संशयः // र्मञ्ज्_६.२१६ //
<मेहवज्ररसः>
भस्म सूतं मृतं कान्तं शुल्बभस्म शिलाजतु /
शुद्धताप्यं शिला व्योषं त्रिफलाङ्कोलबीजकम् // र्मञ्ज्_६.२१७ //
कपित्थरजनीचूर्णं भृङ्गराजेन भावयेत् /
विंशद्वारं विशोष्याथ मधुयुक्तं लिहेत्सदा // र्मञ्ज्_६.२१८ //
निष्कमात्रं लिहेन्मेही मेहवज्रो महारसः /
महानिम्बस्य बीजानि पिष्ट्वा कर्षमितानि च // र्मञ्ज्_६.२१९ //
पलं तण्डुलतोयेन घृतनिष्कद्वयेन च /
एकीकृत्य पिबेत्तोयं हन्ति मेहं चिरोत्थितम् // र्मञ्ज्_६.२२० //
<इन्द्रवटीरसः>
मृतं सूतं मृतं वङ्गमर्जुनस्य त्वचान्वितम् /
तुल्यांशं मर्दयेत् खल्वे शाल्मल्या मूलजैर् द्रवैः // र्मञ्ज्_६.२२१ //
दिनान्ते वटिका कार्या माषमात्रा प्रमेहहा /
एषा इन्द्रवटी नाम्ना मधुमेहप्रशान्तये // र्मञ्ज्_६.२२२ //
<रसेन्द्रमङ्गलरसः>
तालसत्त्वं मृतं ताम्रं मृतं लोहं मृतं रसम् /
हतमभ्रं हतं तारं गन्धं तुत्थं मनःशिला // र्मञ्ज्_६.२२३ //
सौवीराञ्जनकासीसं नीलीभल्लातकानि च /
शिलाजत्वर्कमूलं तु कदलीकन्दचित्रकम् // र्मञ्ज्_६.२२४ //
त्वचम् अङ्कोलजां कृष्णां कृष्णधत्तूरमूलकम् /
अवल्गुजानि बीजानि गौरीमाध्वीफलानि च // र्मञ्ज्_६.२२५ //
हेमाह्वां फेनजात्यां च फलिनीं विषतिन्दुकम् /
तैलिन्यो लोहकिट्टं च पुराणममृतं च तत् // र्मञ्ज्_६.२२६ //
त्वचा च मीनकाक्षस्य पुनरुक्तं पलं पृथक् /
तैलिन्यो वटकास्तासु सर्वमेकत्र चूर्णयेत् // र्मञ्ज्_६.२२७ //
खल्वे निधाय दातव्या पुनरेषां च भावना /
ब्रह्मदण्डी शिखापुङ्खा देवदाली च नीलिका // र्मञ्ज्_६.२२८ //
वाणशोना नृपतरु निम्बसारो विभीतकः /
करञ्जो भृङ्गराजश्च गायत्री तिन्तडीफलम् // र्मञ्ज्_६.२२९ //
मलयूमूलमेतेषां तिस्रस्तिस्रस्तु भावनाः /
दातव्या कुप्पिकां कृत्वा सम्यक् संशोष्य चातपे // र्मञ्ज्_६.२३० //
भाण्डे तद्धारयेद्भाण्डं मुद्रितं चाथ कारयेत् /
यामं मन्दाग्निना पच्यात् पुटमध्ये ह्यसौ रसः // र्मञ्ज्_६.२३१ //
पुण्डरीकं निहन्त्येव नात्र कार्या विचारणा /
द्विमासाभ्यन्तरे पुंसामपथ्यं न तु भोजयेत् // र्मञ्ज्_६.२३२ //
रोगाः सर्वे विलीयन्ते कुष्ठानि सकलानि च /
भानुभक्तिप्रवृत्तानां गुरुभक्तिकृतां सदा // र्मञ्ज्_६.२३३ //
रसेन्द्रमङ्गलो नाम्ना रसो'यं प्रकटीकृतः /
अनुग्रहाय भक्तानां शिवेन करुणात्मना // र्मञ्ज्_६.२३४ //
<सर्वेश्वररसः>
मृतताम्राभ्रलोहानां हिङ्गुलं च पलं पलम् /
जम्बीरोन्मत्तभार्गीभिः स्नुह्यर्कविषमुष्टिभिः // र्मञ्ज्_६.२३५ //
मर्द्यं हयारिजैर् द्रावैः प्रत्येकं च दिनं दिनम् /
एवं सप्तदिनं मर्द्यं तद्गोलं वस्त्रवेष्टितम् // र्मञ्ज्_६.२३६ //
वालुकायन्त्रगं स्वेद्यं त्रिदिनं लघुवह्णिना /
आदाय चूर्णयेत् सर्वं पलैकं योजयेद्विषम् // र्मञ्ज्_६.२३७ //
द्विपलं पिप्पलीचूर्णमिश्रं सर्वेश्वरं रसम् /
द्विगुञ्जं लेहयेत् क्षौद्रैर् मुनिमण्डलकुष्ठनुत् // र्मञ्ज्_६.२३८ //
वाकुची चैव दारू च कर्षमात्रं विचूर्णितम् /
लिहेद् एरण्डतैलेन ह्यनुपानं सुखावहम् // र्मञ्ज्_६.२३९ //
रक्ताधिक्ये सिरामोक्षः पादे बाहौ ललाटके /
कर्तव्यो दृष्टिरोगेषु कुष्ठिनां च विशेषतः // र्मञ्ज्_६.२४० //
बलिनो बहुदोषस्य वयःस्थस्य शरीरिणः /
एतत्प्रमाणमिच्छन्ति प्रस्थं शोणितमोक्षणे // र्मञ्ज्_६.२४१ //
व्यभ्रे वर्षासु विद्यात्तु ग्रीष्मकाले तु शीतले /
हेमन्तकाले मध्याह्ने शस्त्रकालास्त्रयः स्मृताः // र्मञ्ज्_६.२४२ //
<तलेश्वररसः (१)>
कर्षा द्वादश तालस्य कूष्माण्डरससम्भृते /
स्वेदयेद् दोलिकायन्त्रे यावत्तोयं न विद्यते // र्मञ्ज्_६.२४३ //
पश्चात्तं शोषयेत्खल्वे सूतं कर्षद्वयं क्षिपेत् /
घर्षयेद् बहुधा तत्तु यावत्कज्जलिका भवेत् // र्मञ्ज्_६.२४४ //
स्नुहीक्षीरं रविक्षीरं छागी गोक्षुरवाकुची /
पातालगरुडाङ्कोलचक्रमर्दकहिज्जलाः // र्मञ्ज्_६.२४५ //
कुमार्युन्मत्तभल्लातत्रिफलाम्बुपुनर्नवाः /
निम्बत्वग् एभिर् भैषज्यैः पुटनाच्च त्रयं त्रयम् // र्मञ्ज्_६.२४६ //
षट्कर्षं कलिकाचूर्णं हण्डिकायां तु धारयेत् /
चतुर्थांशमधः स्थाप्यं मध्ये स्थाप्यं तु तालकम् // र्मञ्ज्_६.२४७ //
पश्चादुपरिचूर्णं तु सर्वं स्थाप्यं प्रयत्नतः /
हण्डिकां कण्ठपर्यन्तां कुमारीरसयोगतः // र्मञ्ज्_६.२४८ //
पूरयेच्च ततो मुद्रां दृढां कुर्यात्प्रयत्नतः /
तस्योपरि शिलां दत्त्वा दृढा न च चलेद्यथा // र्मञ्ज्_६.२४९ //
दीपाग्ना चतुर्यामं विंशद्यामं हठाग्निना /
स्वाङ्गशीतलमुद्धृत्य कुष्ठे तालेश्वरो रसः // र्मञ्ज्_६.२५० //
कुष्ठनाशः परः ख्यातो भैरवानन्दयोगिना /
पथ्यं मुद्गाम्बुशाल्यन्नं भिषगत्र प्रयोजयेत् // र्मञ्ज्_६.२५१ //
<तालेश्वररसः (२)>
सूतो द्वौ वल्गुजा त्रीणि कणा विश्वा त्रिकं त्रिकम् // र्मञ्ज्_६.२५२ //
सार्धैकं ब्रह्मपुत्रस्य मरिचस्य चतुष्टयम् /
एकैकं निम्बधत्तूरबीजतो गन्धकत्रयम् // र्मञ्ज्_६.२५३ //
जातीटङ्कणतालाया भागा दश दश स्मृताः /
युक्त्या सर्वं विमर्द्याथामृतास्वरसभाविताः // र्मञ्ज्_६.२५४ //
सप्तधा शोषयित्वाथ धत्तूरस्यैव दापयेत् /
सम्मर्द्य गोलकं सार्द्रं धत्तूरैर् वेष्टयेद्दलैः // र्मञ्ज्_६.२५५ //
गोमये वेष्टयेत्तच्च कुक्कुटाख्यपुटे पचेत् /
रसः कुष्ठहरः सेव्यः सर्वदा भोजनप्रियैः // र्मञ्ज्_६.२५६ //
<स्वर्णक्षीरीरसः>
हेमाह्वां पञ्चपलिकां क्षिप्त्वा तक्रघटे पचेत् /
तक्रे जीर्णे समुद्धृत्य पुनः क्षीरघटे पचेत् // र्मञ्ज्_६.२५७ //
क्षीरे जीर्णे समुद्धृत्य क्षालयित्वा विशोषयेत् /
चूर्णितं तत्पञ्चपलं मरिचानां पलद्वयम् // र्मञ्ज्_६.२५८ //
पलैकं मूर्छितं सूतम् एकीकृत्वा च भक्षयेत् /
निष्कैकं सुप्तिकुष्ठार्तः स्वर्णक्षीरीरसो ह्ययम् // र्मञ्ज्_६.२५९ //
<शूलगजकेसरीरसः>
शुद्धसूतं द्विधा गन्धं यामैकं मर्दयेद् दृढम् /
ताम्रभस्म द्वयोस्तुल्यं सम्पुटे तं निरोधयेत् // र्मञ्ज्_६.२६० //
ऊर्ध्वाधोलवणं दत्त्वा मृद्भाण्डे धारयेद्भिषक् /
ततो गजपुटे पक्त्वा स्वाङ्गशीतं समुद्धरेत् // र्मञ्ज्_६.२६१ //
सम्पुटं चूर्णयेत् सूक्ष्मं पर्णखण्डे द्विगुञ्जके /
भक्षयेच्छूलपीडार्थे हिङ्गुशुण्ठीसजीरकम् // र्मञ्ज्_६.२६२ //
वचामरिचजं चूर्णं कर्षमुष्णजलैः पिबेत् /
असाध्यं साधयेच्छूलं रसः स्याच्छूलकेसरी // र्मञ्ज्_६.२६३ //
व्यायामं मैथुनं मद्यं लवणं कटुकानि च /
वेगरोधं शुक्ररोधं वर्जयेच्छूलवान्नरः // र्मञ्ज्_६.२६४ //
<ब्रह्मरसः>
भागैकं मूर्छितं सूतं गन्धावल्गुजचित्रकान् /
चूर्णं तु ब्रह्मबीजानां प्रतिद्वादशभागिकम् // र्मञ्ज्_६.२६५ //
भागांस्त्रिंशद्गुडस्यापि क्षौद्रेण गुटिका कृता /
अयं ब्रह्मरसो नाम्ना ब्रह्महत्याविनाशनः // र्मञ्ज्_६.२६६ //
द्विनिष्कभक्षणाद्धन्ति प्रसुप्तिं कुष्ठमण्डलम् /
पातालगरुडीमूलं जले पिष्ट्वा पिबेदनु // र्मञ्ज्_६.२६७ //
<इन्दुधररसः>
शुद्धं सूतं समं गन्धं तुत्थं च मृतताम्रकम् /
मर्दितं वाकुचीक्वाथैर्दिनैकं वटकीकृतम् // र्मञ्ज्_६.२६८ //
निष्कमात्रं सदा खादेच्छ्वेतघ्नेन्दुधरो रसः /
वाकुचीतैलकर्षैकं सक्षौद्रमनुपाययेत् // र्मञ्ज्_६.२६९ //
<पारिभद्ररसः>
मूर्छितं सूतकं धात्रीफलं निम्बस्य चाहरेत् /
तुल्यांशं खादिरक्वाथैर्दिनं मर्द्यं च भक्षयेत् /
निष्कैकं दद्रुकुष्ठघ्नः पारिभद्राह्वयो रसः // र्मञ्ज्_६.२७० //
<श्वेतारिरसः>
शुद्धं सूतं समं गन्धं त्रिफलाभ्रं च वाकुची /
भल्लातं च शिला कृष्णा निम्बबीजं समं समम् // र्मञ्ज्_६.२७१ //
मर्दयेद्भृङ्गजद्रावैः पेष्यं शोष्यं पुनः पुनः /
इत्थं कुर्यात् त्रिसप्ताहं रसं श्वेतारिको भवेत् // र्मञ्ज्_६.२७२ //
मध्वाज्यैः खादयेन्निष्कं श्वेतकुष्ठं विनाशयेत् // र्मञ्ज्_६.२७३ //
<कालाग्निरुद्ररसः>
सूताभ्रं ताम्रतीक्ष्णानां भस्म माक्षिकगन्धकम् /
वन्ध्याकर्कोटकीद्रावै रसो मर्द्यो दिनावधि // र्मञ्ज्_६.२७४ //
वन्ध्याकर्कोटकीकन्दे क्षिप्त्वा लिप्त्वा मृदा बहिः /
भूधराख्ये पुटे पाच्यं दिनैकं तु विचूर्णयेत् // र्मञ्ज्_६.२७५ //
दशमांशं विषं योज्यं माषमात्रं च भक्षयेत् /
रसः कालाग्निरुद्रो'यं दशाहेन विसर्पनुत् // र्मञ्ज्_६.२७६ //
पिप्पलीमधुसंयुक्तं ह्यनुपानं प्रकल्पयेत् /
<मकरध्वजरसः>
स्वर्णादष्टगुणं सूतं मर्दयेद् द्वित्वगन्धकम् // र्मञ्ज्_६.२७७ //
रक्तकार्पासकुसुमैः कुमार्यास्त्रिदिनं ततः /
शुष्कं काचघटे रुद्ध्वा वालुकायन्त्रगं हठात् // र्मञ्ज्_६.२७८ //
भस्म कुर्याद्रसेन्द्रस्य नवार्ककिरणोपमम् /
भागो'स्य भागाश्चत्वारः कर्पूरस्य सुशोभनाः // र्मञ्ज्_६.२७९ //
लवङ्गं मरिचं जातीफलं कर्पूरमात्रया /
मेलयेन्मृगनाभिं च गद्याणकमितां ततः // र्मञ्ज्_६.२८० //
श्लक्ष्णपिष्टो रसः श्रीमाञ्जायते मकरध्वजः /
वल्लं वल्लद्वयं वास्य ताम्बूलदलसंयुतम् // र्मञ्ज्_६.२८१ //
भक्षयेन्मधुरं स्निग्धं लघुमांसमवातुलम् /
शृतं शीतं सितायुक्तं दुग्धं गोधूमम् आज्यकम् // र्मञ्ज्_६.२८२ //
माषाश्च पिष्टमपरं मद्यानि विविधानि च /
करोत्यग्निबलं पुंसां वलीपलितनाशनः // र्मञ्ज्_६.२८३ //
मेधायुःकान्तिजनकः कामोद्दीपनकृन्महान् /
अभ्यासात् साधकः स्त्रीणां शतं जयति नित्यशः // र्मञ्ज्_६.२८४ //
रतिकाले रतान्ते वा पुनः सेव्यो रसोत्तमः /
मदहानिं करोत्येष प्रमदानां सुनिश्चितम् // र्मञ्ज्_६.२८५ //
कृत्रिमं स्थावरविषं जङ्गमं विषवारिजम् /
न विकाराय भवति साधकानां च वत्सरात् // र्मञ्ज्_६.२८६ //
मृत्युञ्जयो यथाभ्यासान्मृत्युं जयति देहिनाम् /
तथायं साधकेन्द्रस्य जरामरणनाशनः // र्मञ्ज्_६.२८७ //
<मदनकामदेवरसः>
तारं वज्रं सुवर्णं च ताम्रं च सूतगन्धकम् /
लोहं च क्रमवृद्धानि कुर्यादेतानि मात्रया // र्मञ्ज्_६.२८८ //
विमर्द्य कन्यकाद्रावैर्न्यसेत्काचमये घटे /
मुद्रितं पिठरीमध्ये धारयेत्सैन्धवैर्भृते // र्मञ्ज्_६.२८९ //
पिठरीं मुद्रयेत्सम्यक् ततश् चुल्ल्यां निवेशयेत् /
वह्निं शनैः शनैः कुर्याद्दिनैकं तत उद्धरेत् // र्मञ्ज्_६.२९० //
स्वाङ्गशीतं च सञ्चूर्ण्य भावयेदर्कदुग्धकैः /
अश्वगन्धा च कङ्कोली वानरी मुशलीक्षुरः // र्मञ्ज्_६.२९१ //
त्रिवारं स्वरसं भाव्यं शतावर्या विभावयेत् /
पद्मकन्दकसेरूणां रसैर् भाव्यं तु एकधा // र्मञ्ज्_६.२९२ //
कस्तूरीव्योषकपूरैः कङ्कोलैलालवङ्गकम् /
पूर्वचूर्णादष्टमांशमितचूर्णं विमिश्रयेत् // र्मञ्ज्_६.२९३ //
सर्वैः समां शर्करां च दत्त्वा शाणोन्मितं ददेत् /
गोदुग्धद्विपलेनैव मधुराहारसेविनः // र्मञ्ज्_६.२९४ //
अस्य प्रभावात् सौन्दर्य्यं बलं तेजो विवर्धते /
तरुणी रमते बह्वीर्वीर्यहानिर्न जायते // र्मञ्ज्_६.२९५ //
<पूर्णेन्दुरसः>
शाल्मल्युत्थैर् द्रवैर् मर्द्य पक्षैकं शुद्धसूतकम् /
यामद्वयं पचेदाज्ये वस्त्रे बद्ध्वाथ मर्दयेत् // र्मञ्ज्_६.२९६ //
दिनैकं शाल्मलिद्रावैर् मर्दयित्वा वटीं कृताम् /
वेष्टयेन्नागवल्ल्या च निःक्षिपेत् काचभाजने // र्मञ्ज्_६.२९७ //
भाजनं शाल्मलीद्रावैः पूर्णं यामद्वयं पचेत् /
वालुकायन्त्रमध्ये तु द्रवे जीर्णे समुद्धरेत् // र्मञ्ज्_६.२९८ //
द्विगुञ्जं भक्षयेत्प्रातर्नागवल्लीदलान्तरे /
मुशलीं ससितां क्षीरैः पलैकं पाययेदनु // र्मञ्ज्_६.२९९ //
रसः पूर्णेन्दुनामायं सम्यग् वीर्यकरो भवेत् /
कामिनीनां सहस्रैकं नरः कामयते ध्रुवम् // र्मञ्ज्_६.३०० //
<कामिनीमदभञ्जनरसः>
शुद्धं सूतं समं गन्धं त्र्यहं कह्लारजद्रवैः /
मर्दितं वालुकायन्त्रे यामैः कूपीगतं पचेत् // र्मञ्ज्_६.३०१ //
रक्ताङ्गस्य द्रवैर्भाव्यं दिनैकं तु सितायुतम् /
यथेष्टं भक्षयेच्चानु कामयेत्कामिनीशतम् // र्मञ्ज्_६.३०२ //
<मदनोदयरस>
शुद्धं सूतं समं गन्धं रक्तोत्पलदलद्रवैः /
यामं मर्द्यं पुनर्गन्धं पूर्वाद् अर्धं विनिष्क्षिपेत् // र्मञ्ज्_६.३०३ //
दिनैकं मर्दयेत्तत्तु पुनर्गन्धं च मर्दयेत् /
पूर्वद्रावैर्दिनैकं तु काचकुप्यां निरुध्य च // र्मञ्ज्_६.३०४ //
दिनैकं वालुकायन्त्रे पक्वम् उद्धृत्य भक्षयेत् /
पञ्चगुञ्जा सिता सार्द्धं रसो'यं मदनोदयः // र्मञ्ज्_६.३०५ //
समूलं वानरीबीजं मुशली शर्करासमम् /
गवां क्षीरेण तत्पेयं पलार्द्धमनुपानकम् // र्मञ्ज्_६.३०६ //
<अनङ्गसुन्दररसः>
पलद्वयं द्वयं शुद्धं पारदं गन्धकं तथा /
मृतहेम्नस्तु कर्षैकं पलैकं मृतताम्रकम् // र्मञ्ज्_६.३०७ //
मृततारं चतुर्निष्कं मर्द्यं पञ्चामृतैर्दिनम् /
रुद्ध्वा तु वै पुटे पश्चाद्दिनैकं तु समुद्धरेत् // र्मञ्ज्_६.३०८ //
पिष्ट्वा पञ्चामृतैः कुर्याद्वटिकां बदराकृतिम् /
अनङ्गसुन्दरो नाम परं पुष्टिप्रदायकः // र्मञ्ज्_६.३०९ //
<कामेश्वररसः>
सम्यङ्मारितमभ्रकं कट्फलं कुष्ठाश्वगन्धामृता मेथी मोचरसो विदारिमुशली गोक्षूरकं क्षूरकम् /
रम्भाकन्दशतावरी ह्यजमुदा माषास्तिला धान्यकं यष्टी नागबला कचूरमदनं जातीफलं सैन्धवम् // र्मञ्ज्_६.३१० //
भार्ङ्गी कर्कटशृङ्गिका त्रिकटुकं जीरद्वयं चित्रकं चातुर्जातपुनर्नवा गजकणा द्राक्षा शटी वासकम् /
शाल्मल्यन्ध्रिफलत्रिकं कपिभवं बीजं समं चूर्णयेच्चूर्णांशा विजया सिता द्विगुणिता मध्वाज्ययोः पिण्डितम् // र्मञ्ज्_६.३११ //
कर्षार्द्धा गुटिकावलेहम् अथवा सेव्यं सदा सर्वथा पेयं क्षीरसिता तु वीर्यकरणं स्तम्भो'प्ययं कामिनी /
रामावश्यकरं सुखातिसुखदं प्रौढाङ्गनाद्रावकम् क्षीणे पुष्टिकरं क्षये क्षयहरं सर्वामयध्वंसनम् // र्मञ्ज्_६.३१२ //
कासश्वासमहातिसारशमनं मन्दाग्निसन्दीपनं धातोर्वृद्धिकरं रसायनवरं नास्त्यन्यदस्मात्परम् /
अर्शांसि ग्रहणीप्रमेहनिचयश्लेष्मातिरक्तप्रणुन् नित्यानन्दकरं विशेषविदुषां वाचां विलासोद्भवम् // र्मञ्ज्_६.३१३ //
अभ्यासेन निहन्ति मृत्युपलितं कामेश्वरो वत्सरात् सर्वेषां हितकारको निगदितः श्रीवैद्यनाथेन यः /
वृद्धानां मदनोदयोदयकरः प्रौढाङ्गनासङ्गमे सिंहो'यं समदृष्टिः प्रत्ययकरो भूपैः सदा सेव्यताम् // र्मञ्ज्_६.३१४ //
<अजीर्णकण्टकरसः>
शुद्धं सूतं विषं गन्धं समचूर्णं विचूर्णयेत् /
मरिचं सर्वतुल्यांशं कण्टकार्याः फलद्रवैः // र्मञ्ज्_६.३१५ //
मर्दयेद्भावयेत्सर्वानेकविंशतिवारकान् /
वटीं गुञ्जात्रयां खादेत्सर्वाजीर्णप्रशान्तये // र्मञ्ज्_६.३१६ //
<उदयभास्कररसः>
गन्धकेन मृतं ताम्रं दशभागं समुद्धरेत् /
ऊषणं पञ्चभागं स्यादमृतं च द्विभागकम् // र्मञ्ज्_६.३१७ //
श्लक्ष्णचूर्णीकृतं सर्वं रक्तिकैकप्रमाणतः /
दातव्यं कुष्ठिने सम्यगनुपानस्य योगतः // र्मञ्ज्_६.३१८ //
गलिते स्फुटिते चैव विषूच्यां मण्डले तथा /
विचर्चिकादद्रुपामाकुष्ठाष्टकप्रशान्तये // र्मञ्ज्_६.३१९ //
<रौद्ररसः>
शुद्धं सूतं समं गन्धं मर्द्यं यामचतुष्टयम् /
नागवल्लीरसैर्युक्तं मेघनादपुनर्नवैः // र्मञ्ज्_६.३२० //
गोमूत्रे पिप्पलीयुक्ते मर्द्यं रुद्ध्वा पुटेल्लघु /
लिहेत्क्षौद्रे रसो रौद्रो गुञ्जामात्रो'र्बुदं जयेत् // र्मञ्ज्_६.३२१ //
<नित्योदितरसः>
मृतसूतार्कलोहाभ्रविषगन्धं समं समम् /
सर्वतुल्यांशभल्लातफलमेकत्र चूर्णयेत् // र्मञ्ज्_६.३२२ //
द्रवैः सूरणकन्दोत्थैः खल्वे मर्द्यं दिनत्रयम् /
माषमात्रं लिहेदाज्ये रसश्चार्शांसि नाशयेत् // र्मञ्ज्_६.३२३ //
रसो नित्योदितो नाम्ना गुदोद्भवकुलान्तकः /
हस्ते पादे मुखे नाभ्यां गुदवृषणयोस्तथा // र्मञ्ज्_६.३२४ //
शोथो हृत्पार्श्वशूलं च यस्यासाध्यार्शसां हितः /
असाध्यस्यापि कर्तव्या चिकित्सा शङ्करोदिता // र्मञ्ज्_६.३२५ //
<अर्शकुठाररसः>
शुद्धसूतं पलैकं तु द्विपलं शुद्धगन्धकम् /
मृतं ताम्रं मृतं लोहं प्रत्येकं तु पलत्रयम् // र्मञ्ज्_६.३२६ //
त्र्यूषणं लाङ्गली दन्ती पीलुकं चित्रकं तथा /
प्रत्येकं द्विपलं योज्यं यवक्षारं च टङ्कणम् // र्मञ्ज्_६.३२७ //
उभौ पञ्चपलौ योज्यौ सैन्धवं पलपञ्चकम् /
द्वाविंशत्पलगोमूत्रं स्नुहीक्षीरं च तत्समम् // र्मञ्ज्_६.३२८ //
मृद्वग्निना पचेत्सर्वं स्थाल्यां यावत्सुपिण्डितम् /
माषद्वयं सदा खादेद्रसो'प्यर्शःकुठारकः // र्मञ्ज्_६.३२९ //
<विद्याधररसः>
गन्धकं तालकं ताप्यं मृतं ताम्रं मनःशिला /
शुद्धं सूतं च तुल्यांशं मर्दयेद्भावयेद्दिनम् // र्मञ्ज्_६.३३० //
पिप्पल्याश्च कषायेण वज्रीक्षीरेण भावयेत् /
वल्लं च भक्षयेत् क्षौद्रैः प्लीहगुल्मादिकं जयेत् // र्मञ्ज्_६.३३१ //
रसो विद्याधरो नाम गोदुग्धं च पिबेदनु /
<वङ्गेश्वररसः>
भस्मसूतं भस्मवङ्गं भागैकैकं प्रकल्पयेत् // र्मञ्ज्_६.३३२ //
गन्धकं मृतताम्रं च प्रत्येकं च चतुर्गुणम् /
अर्कक्षीरैर्दिनं मर्द्यं सर्वं तद्गोलकीकृतम् // र्मञ्ज्_६.३३३ //

एवं वङ्गेश्वरो नाम्ना प्लीहगुल्मोदरं जयेत् // र्मञ्ज्_६.३३४ //
घृतैर् गुञ्जाद्वयं लिह्यान्निष्कं श्वेतपुनर्नवाम् /
गवां मूत्रैः पिबेच्चानु रजनी वा गवां जलैः // र्मञ्ज्_६.३३५ //
<उदरारिरसः>
पारदं शिखितुत्थं च जैपालं पिप्पलीसमम् /
आरग्वधफलान्मज्जा वज्रीदुग्धेन मर्दयेत् // र्मञ्ज्_६.३३६ //
माषमात्रां वटीं खादेद्धरेत्स्त्रीणां जलोदरम् /
चिञ्चाफलरसं चानु पथ्यं दध्योदनं हितम् /
जलोदरहरं चैव तीव्रेण रेचनेन तु // र्मञ्ज्_६.३३७ //
<जलोदरारिरसः>
पिप्पलीमरिचं ताम्रं काञ्चनीचूर्णसंयुतम् // र्मञ्ज्_६.३३८ //
स्नुहीक्षीरैर्दिनं मर्द्यं तुत्थं जैपालबीजकम् /
निष्कं खादेद् विरेकं स्यात् सद्यो हन्ति जलोदरम् // र्मञ्ज्_६.३३९ //

दिनान्ते च प्रदातव्यमन्नं वा मुद्गयूषकम् // र्मञ्ज्_६.३४० //
<नाराचरसः>
सूतटङ्कणतुल्यांशं मरिचं सूततुल्यकम् /

द्विगुञ्जं रेचने सिद्धं नाराचो'यं महारसः /
गुल्मप्लीहोदरं हन्ति पिबेत्तमुष्णवारिणा // र्मञ्ज्_६.३४२ //

शुण्ठीमरिचसंयुक्तं रसगन्धकटङ्कणम् /
जैपालस्त्रिगुणः प्रोक्तः सर्वमेकत्र चूर्णयेत् // र्मञ्ज्_६.३४३ //

दोषाः कदाचित्कुप्यन्ति जिता लङ्घनपाचनैः // र्मञ्ज्_६.३४५ //


र्मञ्ज्, ७

अमृतं विधिसंयुक्तं विधिहीनं तु तद्विषम् // र्मञ्ज्_७.२ //
रेचनान्ते इदं सेव्यं सर्वदोषापनुत्तये /

कर्तव्यं क्षेत्रकरणं सर्वस्मिंश्च रसायने // र्मञ्ज्_७.५ //

रुद्ध्वा लघुपुटे पच्याद् उद्धृत्य मधुसर्पिषा // र्मञ्ज्_७.६ //

समूलं भृङ्गराजं तु छायाशुष्कं विचूर्णयेत् // र्मञ्ज्_७.७ //

<हेमसुन्दररसः>

क्षीराज्यं मधुना युक्तं माषैकं कांस्यपात्रके /
लेहयेन्मासषट्कं तु जरामृत्युविनाशनम् // र्मञ्ज्_७.१० //
वाकुचीचूर्णकर्षैकं धात्रीरसपरिप्लुतम् /
अनुपानं लिहेन्नित्यं स्याद्रसो हेमसुन्दरः // र्मञ्ज्_७.११ //
<मृतसञ्जीवनी गुटीका>
शुद्धसूतं वज्रभस्म सत्त्वमभ्रकताप्ययोः /
कान्तलोहसमं हेम जम्बीरे मर्दयेद् दृढम् // र्मञ्ज्_७.१२ //

तत्कल्कैश्छादितं कृत्वा पक्षैकं भूधरे पचेत् // र्मञ्ज्_७.१४ //
यामं यामं समुद्धृत्य लिप्त्वामूषां पुनः पुनः /
रुद्ध्वाथ पूर्ववत्पाच्यमेनं पक्षात्समुद्धरेत् // र्मञ्ज्_७.१५ //

एतैः प्रलेपयेन्मूषां गुटिकां तत्र निक्षिपेत् // र्मञ्ज्_७.१६ //
टङ्कणं श्वेतकाचं च दत्त्वा यामे दृढं दृढम् /
खदिराङ्गारयोगेन द्रुतो 'यं जायते रसः // र्मञ्ज्_७.१७ //
मूषायां बिडयोगेन समं हेम च जारयेत् /
ततस्त्रियामकैर्मर्द्यं सगोमूत्रं दिनैकतः // र्मञ्ज्_७.१८ //
अन्धमूषागतो ध्मातो बद्धो भवति वज्रवत् /
मृतसञ्जीवनी नाम गुटिका वक्रमध्यगा // र्मञ्ज्_७.१९ //
कर्षमात्रा जरां मृत्युं हन्ति सत्यं शिवोदितम् /
शस्त्रस्तम्भं च कुरुते ब्रह्मायुर्भवते नरः // र्मञ्ज्_७.२० //
<वीर्यरोधिनीगुटिका>
नागवल्लीदलद्रावैः सप्ताहं शुद्धसूतकम् /
मर्दयेत् क्षालयेद् अम्लैश्चतुर्निष्कप्रमाणकम् // र्मञ्ज्_७.२१ //

ततः शूकरमांसस्य गर्भे कृत्वाथ सिञ्चयेत् // र्मञ्ज्_७.२२ //

ततश्चुल्ल्यां लोहपात्रे तैले धत्तूरसंयुते // र्मञ्ज्_७.२३ //
क्षिप्त्वा त्रिंशत्पले पाच्यं तद्रसं मांसपिण्डकम् /
सन्ध्याम् आरभ्य मन्दाग्नौ यावत्सूर्योदयं पचेत् // र्मञ्ज्_७.२४ //
हठाज्जागरणं कुर्यादन्यथा तन्नसिद्धिभाक् /
प्रातरुद्धृत्य गुटिकां क्षीरभाण्डे विनिक्षिपेत् // र्मञ्ज्_७.२५ //
तत्क्षीरं शुष्यति क्षिप्रमेतत्प्रत्ययकारकम् /
रतिकाले मुखे धार्या गुटिका वीर्यरोधिनी // र्मञ्ज्_७.२६ //
क्षीरं पीत्वा रमेद्रामां कामाकुलकुलान्विताम् /
स्वमुखाद्धारयेद्धस्ते तदा वीर्यं विमुञ्चति // र्मञ्ज्_७.२७ //


र्मञ्ज्, ८

हेमन्ते शिशिरे चैव मध्याह्ने'ञ्जनमिष्यते // र्मञ्ज्_८.१ //
पूर्वाह्ने चापराह्णे च ग्रीष्मे शरदि चेष्यते /
वर्षासु कुर्यादत्युष्णे वा वसन्ते सदैव हि // र्मञ्ज्_८.२ //
श्रान्ते प्ररुदिते भीते पीतमद्ये नवज्वरे /
अजीर्णे वेगघाते च अञ्जनं न प्रशस्यते // र्मञ्ज्_८.३ //
हरेणुमात्रां कुर्वीत वर्तिं तीक्ष्णाञ्जने भिषक् /
प्रमाणं मध्यमं सार्धं द्विगुणं च मृदौ भवेत् // र्मञ्ज्_८.४ //
सूतकं गन्धकः पेतं चाङ्गेरीरससम्मूर्छितम् /
अञ्जनं दृष्टिदं नृणां नेत्रामयविनाशनम् // र्मञ्ज्_८.५ //

सूततुर्यांशं कर्पूरमञ्जनं नयनामृतम् // र्मञ्ज्_८.६ //
कृष्णसर्पवसा शङ्खः कतकं कट्फलमञ्जनम् /
रस एव मरीचेन अन्धानां दर्शनं परम् // र्मञ्ज्_८.७ //
शङ्खस्य भागाश्चत्वारस् तदर्धेन मनःशिला /
मनःशिलार्धं मरिचं मरिचार्द्धेन पिप्पली // र्मञ्ज्_८.८ //
वारिणा तिमिरं हन्ति अर्बुदं हन्ति मस्तुना /
चिपिटं मधुना हन्ति स्त्रीक्षीरेण च पुष्पकम् // र्मञ्ज्_८.९ //

ताम्रपात्रे कृता नेत्रे हन्ति पीडां सुविस्तरात् // र्मञ्ज्_८.१० //
दन्तैर्दन्तिवराहोष्ट्रगोहयाजखरोद्भवैः /
शङ्खमुक्ताम्भोधिफेनयुतैः सर्वैर् विचूर्णयेत् /
हन्ति वर्तिः कृता श्लक्ष्णं शुक्राणां नाशिनी परम् // र्मञ्ज्_८.११ //
तुत्थमाक्षिकसिन्धूत्थशिवाशङ्खमनःशिला /
गैरिकोदकफेनं च मरिचं चेति चूर्णयेत् // र्मञ्ज्_८.१२ //

वर्त्मरोगं च तिमिरं काचशुक्रहरं परम् // र्मञ्ज्_८.१३ //

राज्यन्धं वार्षिकं पुष्पं वर्तिचन्द्रोदया जयेत् // र्मञ्ज्_८.१६ //

कृष्णाञ्जनं तयोस्तुल्यं सर्वमेकत्र चूर्णयेत् // र्मञ्ज्_८.१७ //

एतत् प्रत्यञ्जनं नेत्रगदजिन्नयनामृतम् // र्मञ्ज्_८.१८ //

जाता रोगाः प्रणश्यन्ति न भवन्ति कदाचन // र्मञ्ज्_८.१९ //

अचिरेणैव तद्वारि तिमिराणि व्यपोहति // र्मञ्ज्_८.२० //

द्वयोस्तुल्येन तैलेन पचेन्मृद्वग्निना क्षणम् // र्मञ्ज्_८.२१ //
तैलतुल्ये भृङ्गरसे तत्तैलं तु विपाचयेत् /
स्निग्धभाण्डगतं भूमौ स्थित्वा मासात्समुद्धरेत् // र्मञ्ज्_८.२२ //
सप्ताहं लेपयेद्वेष्ट्य कदल्याश्च दलैः शिरः /
निर्वाते क्षीरभोजी स्यात् छालयेत् त्रिफलाजलैः // र्मञ्ज्_८.२३ //
नित्यमेव प्रकर्तव्यं सप्ताहं रञ्जनं भवेत् /
यावज्जीवं न सन्देहः कचाः स्युर्भ्रमरोपमाः // र्मञ्ज्_८.२४ //
काकमाची यवा जाती समं कृष्णतिलं ततः /
तत्तैलं ग्राहयेद्यन्त्रे तेन स्यात् केशरञ्जनम् // र्मञ्ज्_८.२५ //
लोहमलामलकल्कः सजपा कुसुमैर्नरः सदा स्नायी /
पलितानीह निहन्याद् गङ्गास्नायीव नरकौघम् // र्मञ्ज्_८.२६ //
काश्मर्या मूलमादौ सहचरकुसुमं केतकीनां च मूलं लौहं चूर्णं सभृङ्गं त्रिफलजलयुतं तैलमेभिर् विपक्वम् /
कृत्वा वै लोहभाण्डे क्षितितलनिहितं मासम् एकं निधाय केशाः काशप्रकाशा भ्रमरकुलनिभा लेपनाद् एव कृष्णाः // र्मञ्ज्_८.२७ //
वज्रीक्षीरेण सप्ताहं सुश्वेतान् भावयेत्तिलान् /
तैलेन लिप्ताः केशाः स्युः शुक्ला वै नात्र संशयः // र्मञ्ज्_८.२८ //

मर्दयित्वा लिपेल्लिङ्गं स्थित्वा यामं तथैव च // र्मञ्ज्_९.१ //
ततः प्रक्षालयेल्लिङ्गं रमेद्रामां यथोचिताम् /
वीर्यस्तम्भकरं पुंसां सम्यङ् नागार्जुनोदितम् // र्मञ्ज्_९.२ //
कृकलासस्य पुच्छाग्रं मुद्रिका प्रोततन्तुभिः /
वेष्ट्या कनिष्ठिका धार्या नरो वीर्यं न मुञ्चति // र्मञ्ज्_९.३ //
मधुना पद्मबीजानि पिष्ट्वा नाभिं प्रलेपयेत् /
यावत्तिष्ठत्यसौ लेपस्तावद्वीर्यं न मुञ्चति // र्मञ्ज्_९.४ //
चटकाण्डं तु सङ्ग्राह्य नवनीतेन पेषयेत् /
तेन प्रलेपयेत् पादौ शुक्रस्तम्भः प्रजायते /
यावन्न स्पृशते भूमिं तावद्वीर्यं न मुञ्चति // र्मञ्ज्_९.५ //
वनक्रोडस्य दंष्ट्राग्रं दक्षिणं च समाहरेत् /
कट्यामुपरि यद् बद्ध्वा शुक्रस्तम्भः प्रजायते // र्मञ्ज्_९.६ //
डुण्डुभो नामतः सर्पः कृष्णवर्णस्तमाहरेत् // र्मञ्ज्_९.७ //
तस्यास्थि धारयेत् कट्यां नरो वीर्यं न मुञ्चति /
विमुञ्चति विमुक्तेन सिद्धयोग उदाहृतः // र्मञ्ज्_९.८ //
रक्तापामार्गमूलं तु सोमवाराभिमन्त्रितम् /
भौमे प्रातः समुद्धृत्य कट्यां बद्ध्वा न वीर्यमुक् // र्मञ्ज्_९.९ //

कुरुते रतौ न पुंसो रेतः पतनं विनाम्लेन // र्मञ्ज्_९.१० //
सूरणं तुलसीमूलं ताम्बूलेन तु भक्षयेत् /

वराहवसया लिङ्गं मधुना सह लेपयेत् /

ध्वस्तं कुर्यात्ततो मेढ्रं रत्यन्यत्राप्यसंशयः // र्मञ्ज्_९.१३ //
अनुराधासुनक्षत्रे लाङ्गलीमूलिका ध्रुवम् /
निखाता मैथुनस्थाने पुंसत्त्वखण्डकारिणी // र्मञ्ज्_९.१४ //
निशा षट्तिन्दुचूर्णेन भावितेनाजवारिणा /
पानाशनं प्रयुक्तेन षण्ढत्वं जायते नृणाम् // र्मञ्ज्_९.१५ //
तिलगोक्षुरयोश्चूर्णं छागीदुग्धेन पाचितम् /
शीतलं मधुना युक्तं भुक्तं षण्ढत्वनाशनम् // र्मञ्ज्_९.१६ //
ऊर्णनाभिस्तु यो जीवो मधुना सह लेपयेत् /
तेन लेपयतो नाभिं बद्धषण्ढ्यं विमुच्यते // र्मञ्ज्_९.१७ //
एक एव महाद्रावी मालतीसम्भवो रसः /
किं पुनर्यदि युज्यते मधुकर्पूरपारदैः // र्मञ्ज्_९.१८ //
आर्द्रकं गन्धकं चैव राजिकं चाथ टङ्कणम् /
सम्पिष्टा सममात्राणि क्षेपयेन्निम्बुजे जले // र्मञ्ज्_९.१९ //
स्थापयेद्घटिकां तिस्रो हस्ते वा धारयेत्ततः /
यावन्त्यो ललनाः पञ्च आजिघ्रन्ति द्रवन्ति वै // र्मञ्ज्_९.२० //
चूर्णिते मधुसंयुक्ते महारिष्टफलछदैः /
लिङ्गलेपेन सुरते द्रवो भवति योषिताम् // र्मञ्ज्_९.२१ //
मधुसैन्धवसंयुक्तं पारावतमलान्वितम् /
एतल्लिप्तेन्द्रियो रामां दासीवत् कुरुते रतौ // र्मञ्ज्_९.२२ //
कपीन्द्रियं शशी सूतं कुङ्कुमं कनकं मधु /
एतल्लिप्तेन्द्रियो रामां दासीवत् कुरुते रतौ // र्मञ्ज्_९.२३ //
वृत्तमध्यस्थितं नाम तद्बाह्ये ह्रीं चतुष्टयम् /
तद्बहिः क्लीं चतुष्कं च लिखित्वा शिलयाखिलम् // र्मञ्ज्_९.२४ //
शुभनक्षत्रसंयोगे स्थापितं मधुनि ध्रुवम् /
स्त्रियम् आकर्षति क्षिप्रं यन्त्रमेतन्न संशयः // र्मञ्ज्_९.२५ //
चन्दनं तगरं कुष्ठं प्रियङ्गुं नागकेशरम् /
कृष्णां तिन्तिडिकं चैव समभागानि कारयेत् // र्मञ्ज्_९.२६ //
दापयेच्चैव सप्ताहमात्मपञ्चमलेन तु /
खाने पाने प्रदातव्यं स्त्रियं वा पुरुषं तथा // र्मञ्ज्_९.२७ //
शङ्खपुष्पी मधुपुष्पी तथा कुञ्चिकिपत्रिका /
श्वेतगिरिसमायुक्ता समभागानि कारयेत् // र्मञ्ज्_९.२८ //
सप्ताहं दापयेद्युक्ता ह्यात्मपञ्चमलेन च /
खाने पाने प्रदातव्यं वशीकरणमुत्तमम् // र्मञ्ज्_९.२९ //
पूर्वोक्तमन्त्रेण सप्तभिःकृत्वा मन्त्रितदापयेत् /
गर्दभस्य रजो गृह्य लुलितं गात्रसम्भवम् /
मृतकस्य तथा भस्म नारीरजःसमन्वितम् // र्मञ्ज्_९.३० //
एकीकृत्य क्षिपेद्रात्रौ शय्यायामासने'पि वा /
नूनं सञ्जायते द्वेषः कथितो मालतीमते // र्मञ्ज्_९.३१ //
अन्यद् योगवरं वक्ष्ये विद्वेषकरणं परम् /
युध्यमानावुभौ श्वानौ परस्परविरोधिनौ // र्मञ्ज्_९.३२ //
तयोर् धूलिं समादाय हन्यते योषितां रतिः /
सत्यं भवति विद्वेषं नात्र कार्या विचारणा // र्मञ्ज्_९.३३ //
प्रक्षालने भगे नित्यं कृते चामलवल्कलैः /
वृद्धापि कामिनी कामं बालेव कुरुते रतिम् // र्मञ्ज्_९.३४ //
सपद्मबीजं सितया भक्षितं पद्मवारिणा /
दृढं स्त्रीणां स्तनद्वन्द्वं मासेन कुरुते भृशम् // र्मञ्ज्_९.३५ //

पतितं यौवनं यस्यास्तस्याः स्तनोन्नतिर्भवेत् // र्मञ्ज्_९.३६ //

निपतन्ति केशनिचयाः कौतुकमिदम् अद्भुतं कुरुते // र्मञ्ज्_९.३७ //
पलाशचिञ्चातिलमाषशङ्खं दहेदपामार्गसपिप्पलो'पि /
मनःशिलातालकचूर्णलेपात् करोति निर्लोमशिरः क्षणात् // र्मञ्ज्_९.३८ //
तण्डुलीयकमूलानि पिष्ट्वा तण्डुलवारिणा /
ऋत्वन्ते त्र्यहपीतानि वन्ध्यां कुर्वन्ति योषितम् // र्मञ्ज्_९.३९ //
काञ्जिकेन जपापुष्पं पिष्ट्वा पिबति याङ्गना /
ऋतौ त्र्यहं निपीतानि वन्ध्यां कुर्वन्ति योषितम् // र्मञ्ज्_९.४० //
धूपिते योनिरन्ध्रे तु निम्बकाष्ठेन युक्तितः /
ऋत्वन्ते रमते सा स्त्री गर्भदुःखविवर्जिता // र्मञ्ज्_९.४१ //
धत्तूरं मल्लिकापुष्पं गृहीत्वा कटिसंस्थितम् /
गर्भं निवारयत्येव रण्डावेश्यादियोषिताम् // र्मञ्ज्_९.४२ //
नागकेशरपुष्पाणां चूर्णं गोसर्पिषा सह /
सेवनाल्लभते पुत्रमृतौ दुग्धान्नभोजिनी // र्मञ्ज्_९.४३ //
बीजानि मातुलुङ्गस्य दुग्धस्विन्ना ससर्पिषा /
सगर्भामिति कुर्वन्ति पानाद्वन्ध्यामपि स्त्रियम् // र्मञ्ज्_९.४४ //
मातुलुङ्गस्य बीजानि कुमार्या सह पेषयेत् /
क्षीरेण सह दातव्यं गर्भमाप्नोत्यसंशयम् // र्मञ्ज्_९.४५ //
पिप्पली शृङ्गवेरं च मरिचं केशरं तथा /
घृतेन सह पातव्यं वन्ध्यागर्भप्रदं परम् // र्मञ्ज्_९.४६ //
पुष्योद्धृतं लक्ष्मणाया मूलं पिष्टं च कन्यका /
ऋत्वन्ते घृतदुग्धाभ्यां पीत्वा गर्भमवाप्नुयात् // र्मञ्ज्_९.४७ //

उशीरमधुयष्टी च लोध्रमिन्द्रयवानपि // र्मञ्ज्_९.४८ //
घृतं सर्जरसं चैव माक्षिकं त्रायमाणकम् /
शोभाञ्जनकमूलानि समभागानि कारयेत् // र्मञ्ज्_९.४९ //
पेषयित्वा ततो द्रव्यमजाक्षीरेण पाचयेत् /
सप्तरात्रं पिबेन्नारी यावत्तिष्ठति शोणितम् // र्मञ्ज्_९.५० //
ततो योनौ विशुद्धायां पश्चाद्दद्यान्महौषधम् /
कुमारीक्षीरसंयुक्तं नस्ये पाने प्रदापयेत् // र्मञ्ज्_९.५१ //
तेन सा लभते पुत्रं सत्यं चैव सुरार्चितम् /
लशुनं क्षीरसंयुक्तं नस्ये पाने प्रदापयेत् // र्मञ्ज्_९.५२ //
अश्वगन्धाकृतं चूर्णमजाक्षीरेण दापयेत् /
यवक्षारं विडङ्गं च गुडूची च हरेणुका // र्मञ्ज्_९.५३ //
सर्वाणि समभागानि कृत्वा च वरचूर्णितान् /
एतत्पीत्वा लभेत् पुत्रं सा नारी नात्र संशयः // र्मञ्ज्_९.५४ //
हिङ्गुं च शतवीर्या च दाडिमं सैन्धवं तथा /
त्रिकटुः शतपुष्पा च नागपुष्पं शतावरी // र्मञ्ज्_९.५५ //
मधुकं सुमना चैव कार्षकाणि प्रदापयेत् /
क्षीरेण सह दातव्या नार्याश्च पुरुषस्य च // र्मञ्ज्_९.५६ //
दिनत्रयं तु भुञ्जीत शालितण्डुलदुग्धकम् /
भुक्तं तु लभते गर्भं नारीणां नात्र संशयः // र्मञ्ज्_९.५७ //
अर्कमूलं प्रियङ्गुं च कुसुम्भं नागकेशरम् /
बला चातिबला छागीक्षीरं पीतं दिनत्रयम् // र्मञ्ज्_९.५८ //
विशोधयन्ति योनिं च ततो दद्यान्महौषधम् /
उत्पलं तगरं कुष्ठं यष्टीमधु सचन्दनम् // र्मञ्ज्_९.५९ //
अजाक्षीरेण पिष्टानि दापयेत् पञ्चवासरम् /
लक्ष्मणागोपयोयुक्ता तस्यै पाने प्रदापयेत् // र्मञ्ज्_९.६० //
तेन सा लभते पुत्रं लक्षणाढ्यं सुपण्डितम् /
गते रक्ते भगे शुद्धे सक्षीरा लक्ष्मणा तथा // र्मञ्ज्_९.६१ //
नस्ये पाने प्रदातव्या लभते सुतमङ्गना /
श्वेतार्कक्षुद्रणीश्वेता श्वेता च गिरिकर्णिका // र्मञ्ज्_९.६२ //
लक्ष्मणा वन्ध्यकर्कोटी देयं गोक्षीरसंयुतम् /
नस्ये पाने कृते गर्भं लभते रतिसङ्गमात् // र्मञ्ज्_९.६३ //
पतन्तं स्तम्भयेद् गर्भं कुलालकरमृत्तिका /
मधुच्छागीपयः पीत्वा किंवा श्वेताद्रिकर्णिका // र्मञ्ज्_९.६४ //
ललना शर्करा पाठा कन्दश्च मधुनान्वितः /
भक्षितो वारयत्येव पतन्तं गर्भसंस्थितम् // र्मञ्ज्_९.६५ //
समभागं सितायुक्तं शालितण्डुलचूर्णकम् /
उदुम्बरशिफाक्वाथे पीतं गर्भं सुरक्षति // र्मञ्ज्_९.६६ //
मातुलुङ्गस्य मूलानि मधुकं मधुसंयुतम् /
घृतेन सह पातव्यं सुखं नारी प्रसूयते // र्मञ्ज्_९.६७ //
तुषाम्बुपरिघृष्टेन कन्देन परिलेपयेत् /
लाङ्गल्याश्चरणौ सूतिं क्षिप्रमाप्नोति गर्भिणी // र्मञ्ज्_९.६८ //
यासां पुष्पागमो नास्ति ऋतुकाले च योषिताम् /
तासां कुर्याच्चिकित्सेयं पुनः पुष्पागमो भवेत् // र्मञ्ज्_९.६९ //
पिप्पलीं च यवक्षारं विडङ्गं मन्मथफलम् /
तिक्तं तु तुम्बिनीबीजं गुटिकां कारयेद्भिषक् // र्मञ्ज्_९.७० //
मुण्डी च क्षीरसंयुक्ता योनिद्वारे'ङ्गना शुभा /
त्रिपञ्चसप्तरात्रेण पुष्पं भवति नान्यथा // र्मञ्ज्_९.७१ //
बालतन्त्रं प्रवक्ष्यामि समासाद् रावणोदितम् /
पूजाद्रव्यं दिशो भागं मन्त्रं तद्ग्राह्यलक्षणम् // र्मञ्ज्_९.७२ //
प्रथमे दिवसे मासे वर्षे गृह्णाति बालकम् /
मन्दानाम्नी समाख्याता योगिणी तस्य लक्षणम् // र्मञ्ज्_९.७३ //
द्वितीये दिवसे मासे वर्षे गृह्णाति बालकम् /
सुनन्दा योगिनी नाम प्रथमं जायते ज्वरः // र्मञ्ज्_९.७४ //
सङ्कोचो हस्तपादानामक्षिरोगो'तिछर्दनम् /
सभयत्वं कृशत्वं च तद्ग्रस्त इति लक्षणम् // र्मञ्ज्_९.७५ //
तृतीये दिवसे मासे वर्षे गृह्णाति बालकम् /
पूतना योगिनी नाम गात्रभङ्गो ज्वरो'रुचिः // र्मञ्ज्_९.७६ //
प्रलापं कन्धराशोथच्छर्दिरित्यादिलक्षणम् /
अपराह्ने च वारुण्यां पञ्चरात्रं बलिं क्षिपेत् // र्मञ्ज्_९.७७ //
चतुर्थे दिवसे मासे वर्षे गृह्णाति बालकम् /
बिडाली नाम तद्ग्रस्ते चक्षुःशूलं ज्वरो'रुचिः // र्मञ्ज्_९.७८ //
गात्रमाटेनम् इत्यादि विवर्णेन बलिं क्षिपेत् /
पञ्चमे दिवसे मासे वर्षे गृह्णाति बालकम् // र्मञ्ज्_९.७९ //
नर्तकीति समाख्याता योगिनी तस्य लक्षणम् /
अथ षष्ठदिने मासे वर्षे गृह्णाति बालकम् // र्मञ्ज्_९.८० //

सप्तमे दिवसे मासे वर्षे शुष्काशिवा शिशुम् // र्मञ्ज्_९.८२ //

पश्चिमायां दिशि पञ्च बलौ दत्ते शिशुः सुखी // र्मञ्ज्_९.८३ //

तया गृहीतमात्रस्य प्रथमं जायते ज्वरः // र्मञ्ज्_९.८४ //
शिरःपीडाक्षिरोगश्च चक्षुरुत्पाटचेष्टितम् /
दक्षिणां दिशिम् आश्रित्य बलिं तस्यै प्रदापयेत् // र्मञ्ज्_९.८५ //
नवमे दिवसे मासे वर्षे गृह्णाति बालकम् /
अचिन्ता योगिनी नाम गात्रभङ्गः शिरो'क्षिरुक् // र्मञ्ज्_९.८६ //
छर्दिः प्रलाप इत्यादि तद्गृहीतस्य लक्षणम् /
उत्तरां दिशिम् आश्रित्य बलिं तस्यै प्रदापयेत् // र्मञ्ज्_९.८७ //
दशमे दिवसे मासे वर्षे गृह्णाति बालकम् /
नाम्ना कापालिका ख्याता योगिनी तस्य लक्षणम् // र्मञ्ज्_९.८८ //
रोदनं कम्पनं छर्दिर्ज्वरो दुर्बलताक्षिरुक् /
पूर्वां दिशं समाश्रित्य पञ्चरात्रं बलिं क्षिपेत् // र्मञ्ज्_९.८९ //
एकादशे दिने मासे वर्षे गृह्णाति लिप्सिता /
गात्रकम्पो ज्वरस्तीव्रस्तद्गृहीतस्य लक्षणम् // र्मञ्ज्_९.९० //
भानुसङ्ख्ये दिने मासे वर्षे गृह्णाति बालकम् /
पीतली योगिनी नाम रोदनं वेदना ज्वरः // र्मञ्ज्_९.९१ //
निद्रा क्षीणस्वरः पीतो वमनाहारशून्यता /
उत्तरां दिशिमाश्रित्य सप्तरात्रं बलिं क्षिपेत् // र्मञ्ज्_९.९२ //
कामसङ्ख्ये दिने मासे वर्षे गृह्णति बालकम् /
भद्रकाली ज्वरो नाम वामहस्तस्य कम्पम् // र्मञ्ज्_९.९३ //
वेदनारुचिनिःश्वासाः कायः पीतो विचेष्टितम् /
पूर्वां दिशः समाश्रित्य बलिं तस्यै प्रदापयेत् // र्मञ्ज्_९.९४ //
पुरन्दरदिने मासे वर्षे गृह्णाति बालकम् /
तारा हि योगिनी नाम ज्वरः शोषो'रुचिर्भृशम् // र्मञ्ज्_९.९५ //
चक्षुःपीडेङ्गितं तस्यै पश्चिमे बलिमाहरेत् /
पक्षे च दिवसे मासे वर्षे गृह्णाति बालकम् // र्मञ्ज्_९.९६ //
योगिनी शर्वरी नाम श्वासः कासो'रुचिर्ज्वरः /
तद्ग्रस्ते चिह्नम् इत्यादि दक्षिणस्यां बलिं क्षिपेत् // र्मञ्ज्_९.९७ //
विकारदिवसे मासे वर्षे गृह्णति योगिनी /
कुमारी नयनोद्वेगो ज्वरशोषादिचेष्टितम् // र्मञ्ज्_९.९८ //
नैरृतीं दिशिमाश्रित्य सप्तरात्रं बलिं क्षिपेत् /
बालं च स्नपयेत्पश्चाच्छान्तितोयेन मन्त्रवित् // र्मञ्ज्_९.९९ //
नदीतीरद्वयाकृष्टमृदा देवीस्वरूपकम् /
कृत्वा पूजा च कर्तव्या भूपपुष्पाक्षतादिभिः // र्मञ्ज्_९.१०० //
वटका लड्डुकापूपा अग्रभक्तं गुडं दधि /
चातुर्वर्ण्यपताकाश्च प्रदीप्ताः पुष्पचन्दनम् // र्मञ्ज्_९.१०१ //
पूजयेत्सर्वरोगाणामपराह्ने यथाबलि /
सर्वत्र नामभेदेन बलिदानं प्रजायते // र्मञ्ज्_९.१०२ //


र्मञ्ज्, १०
अथ कालस्य विज्ञानं प्रवक्ष्यामि यथासुखम् /
जीवितं मरणं योगी यतो जानाति निश्चयात् // र्मञ्ज्_१०.१ //
कालग्रहस्य यस्येदं दंष्ट्रासम्पुटके जगत् /
अद्यैव वा प्रभाते वा सो'वश्यं भविष्यति // र्मञ्ज्_१०.२ //
रसं रसायनं योगं कालं ज्ञात्वा समाचरेत् /
यस्माज्ज्ञानं विना व्यर्थं तत्तस्मात्प्रोच्यते'धुना // र्मञ्ज्_१०.३ //
दूतो रक्तकषायकृष्णवसनो दन्ती जरामर्दितस् तैलाभ्यक्तशरीरकायुधकरो दीनाश्रुपूर्णाननः /
भस्माङ्गारकपालपांशुमुशलः सूर्यास्तसूर्योदये यः सूर्यस्वरसंस्थितो गदवतां कालाय स स्यादसौ // र्मञ्ज्_१०.४ //
अकस्माच्चित्तविकृतिरकस्मात्पुरुषोत्तमः /
अकस्मादिन्द्रियोत्पत्तिः सन्निपातस्य लक्षणम् // र्मञ्ज्_१०.५ //
शरीरं शीतलं यस्य प्रकृतेर् विकृतिर् भवेत् /
तत्रारिष्टं समासेन व्याप्तं तत्र निबोध मे // र्मञ्ज्_१०.६ //
दुष्टशब्देन रमते साधुशब्देन कुप्यति /
यश्चाकस्मान्न शृणुते तं गतायुषमादिशेत् // र्मञ्ज्_१०.७ //
यो वा गन्धं न गृह्णाति दीपे शान्ते च मानवः /
दिवाज्योतींषि यश्चापि ज्वलितानि च पश्यति // र्मञ्ज्_१०.८ //
चन्द्रं सूर्यप्रभं पश्येत् सूर्यं वा चन्द्रवर्चसम् /
तडिद्वातोषितान् मेघान् निर्मले गगने चरान् // र्मञ्ज्_१०.९ //
विमानयानप्रासादैर्यश्च सङ्कुलमम्बरम् /
यश्च नीलं मूर्तिमन्तम् अन्तरिक्षं च पश्यति // र्मञ्ज्_१०.१० //
यस्तूष्णम् इव गृह्णाति शीतमुष्णं च शीतवत् /
सञ्जातः संशयो यस्य तं वदन्ति गतायुषम् // र्मञ्ज्_१०.११ //
विपरीतेन गृह्णाति भावानन्यांश्च यो नरः /
धूमनीहारवासोभिरावृतामिव मेदिनीम् // र्मञ्ज्_१०.१२ //
प्रदीप्तमिव लोकं च यो'वलुप्तमिवाम्भसा /
भूमिम् अष्टादशाकारां लेखाभिर्यस्तु पश्यति // र्मञ्ज्_१०.१३ //
ज्योत्स्नादर्शे हि तोयेषु छायां यश्च न पश्यति /
पश्यत्येकाङ्गहीनां च वैकृतं चापि पश्यति // र्मञ्ज्_१०.१४ //
श्वकाककङ्कगृध्राणां प्रयातं यक्षराक्षसाम् /
पिशाचोरगनागानां विकृतामपि यो नरः // र्मञ्ज्_१०.१५ //
ह्रीश्रियौ यस्य नश्येतां तेज ओजः स्मृतिस् तथा /
अकस्माज्जृम्भते यश्च स परासुरसंशयम् // र्मञ्ज्_१०.१६ //
यस्याधरोष्ठः पतति स्थितश्चोर्ध्वं तथोत्तरम् /
पाशशङ्का भवेद्यस्य दुर्लभं तस्य जीवितम् // र्मञ्ज्_१०.१७ //
कुटिला स्फुटिता वापि सुप्ता यस्य च नासिका /
अवस्फूर्यति मग्ना वा स परासुरसंशयम् // र्मञ्ज्_१०.१८ //
स्वरूपं परनेत्रेषु पुत्तिकां यो न पश्यति /
यदा हि पटुदृष्टिश्च तदा मृत्युर् अदूरतः // र्मञ्ज्_१०.१९ //
कर्णहीनं यदात्मानं पश्यत्यात्मा कथञ्चन /
न स जीवति लोके'स्मिन् कालेन कवलीकृतः // र्मञ्ज्_१०.२० //
रात्रौ दाहो भवेद्यस्य दिवा शीतं च जायते /
कफपूरितकण्ठस्य मृत्युश्चैव न संशयः // र्मञ्ज्_१०.२१ //
चरणौ शीतलौ यस्य शीतलं नाभिमण्डलम् /
शिरस्तापो भवेद्यस्य तस्य मृत्युर्न संशयः // र्मञ्ज्_१०.२२ //
हुङ्कारः शीतलो यस्य फुत्कारो वह्णिसन्निभः /
सदा दाहो भवेद्यस्य तस्य मृत्युर्न संशयः // र्मञ्ज्_१०.२३ //
अरुन्धतीं ध्रुवं चैव विष्णोस्त्रीणि पदानि च /
हीनायुधो न पश्यन्ति चतुर्थं मातृमण्डलम् // र्मञ्ज्_१०.२४ //
अरश्मिबिम्बं सूर्यस्य वह्नेश्चैवांशुवर्जितम् /
दृष्ट्वैकादशमासास्तु नरश्चोर्ध्वं न जीवति // र्मञ्ज्_१०.२५ //
वातं मूत्रं पुरीषं यः सुवर्णं रजतं तथा /
प्रत्यक्षमथवा स्वप्ने दशमासं न जीवति // र्मञ्ज्_१०.२६ //
क्वचित् पश्यति यो दीप्तं स्वर्णवत् काननं नरः /
विरूपाणि च भूतानि नवमासं न जीवति // र्मञ्ज्_१०.२७ //
स्थूलाङ्गो'पि कृशः कृशो'पि सहसा स्थूलत्वमालम्बते श्यामो वा कनकप्रभो यदि भवेद्गौरो'पि कृष्णच्छविः /
धीरो धीरतयार्थधर्मनिपुणः शान्तोपकारी पुमान् इत्येवं प्रकृते तु शान्तचलनं मास्यष्टमे मृत्युदम् // र्मञ्ज्_१०.२८ //
पीडा भवेत्पाणितले च जिह्वामूलं समूलं रुधिरं च कृष्णम् /
वृद्धिं नरः कामपि यन्न दृष्ट्वा जीवेन्मनुष्यः स हि सप्तमासान् // र्मञ्ज्_१०.२९ //
मध्याङ्गुलीनां त्रितयं विरक्तं रोगं विना शुष्यति यस्य कण्ठः /
मुहुर्मुहुः प्रस्रवणं च जाड्यं षष्ठे च मासे प्रलयं प्रयाति // र्मञ्ज्_१०.३० //
यस्य न स्फुरणं किञ्चिद् विद्यते यस्य कर्मणि /
सो'वश्यं पञ्चमे मासि स्कन्धारूढो गमिष्यति // र्मञ्ज्_१०.३१ //
यस्य न स्फुरते ज्योतिः पीडिते नयनद्वये /
मरणं तस्य निर्दिष्टं चतुर्थे मासि निश्चितम् // र्मञ्ज्_१०.३२ //
स्पन्दते वृषणो यस्य न किञ्चिदपि पीडितः /
तृतीये मासि सो'वश्यं यमलोके गमिष्यति // र्मञ्ज्_१०.३३ //
तारा दिवा चन्द्रप्रभं निशान्ते यो विद्युतं पश्यति चैव श्वभ्रे /
इन्द्रायुधं वा स्वयमेव रात्रौ मासद्वये तस्य वदन्ति नाशम् // र्मञ्ज्_१०.३४ //
यस्य जानुगतं मर्म न किञ्चिदपि चेष्टितम् /
मासान्ते मरणं तस्य न केनापि विलम्ब्यते // र्मञ्ज्_१०.३५ //
कनिष्ठाङ्गुलिपर्व स्यात् कृष्णं च मध्यमं यदा /
गतायुः प्रोच्यते पुंसाम् अष्टादशदिनावधिः // र्मञ्ज्_१०.३६ //
घृते तैले जले वापि दर्पणे यस्य दृश्यते /
शिरोरहितमात्मानं पक्षमेकं स जीवति // र्मञ्ज्_१०.३७ //
शैत्यं दध्यन्नपानानि यस्य तापकराणि च /
शीतरश्मि भवेच्चारुहासं चाथ सुनिर्मलम् // र्मञ्ज्_१०.३८ //
न वेत्ति वै चारुहितं न चोष्णं वेत्ति यो नरः /
कालज्ञानेन सम्प्रोक्तं पक्षमेकं स जीवति // र्मञ्ज्_१०.३९ //
स्नातमात्रस्य यस्यैते त्रयः शुष्यन्ति तत्क्षणात् /
हृदयं हस्तपादौ च दशरात्रं स जीवति // र्मञ्ज्_१०.४० //
नासाग्रं रसनाग्रं च चक्षुश्चैवौष्ठसम्पुटम् /
यो न पश्येत्पुरादृष्टं सप्तरात्रं स जीवति // र्मञ्ज्_१०.४१ //
धारा बिन्दुसमा यस्य पतते च महीतले /
सप्ताहाज्जायते मृत्युः कालज्ञानेन कथ्यते // र्मञ्ज्_१०.४२ //
अथातः सम्प्रवक्ष्यामि छायापुरुषलक्षणम् /
यस्य विज्ञानमात्रेण त्रिकालज्ञो भवेन्नरः // र्मञ्ज्_१०.४३ //
कालो दूरस्थितो'स्यापि येनोपायेन लक्ष्यते /
तं वदन्ति समासेन यथोद्दिष्टं शिवागमे // र्मञ्ज्_१०.४४ //
एकान्ते विजने गत्वा कृत्वादित्यं स्वपृष्ठतः /
सन्निरीक्ष्य निजच्छायां कण्ठदेशसमाहिताम् // र्मञ्ज्_१०.४५ //
ततश्चाकाशमीक्षेत ततः पश्यति शङ्करम् /
अष्टोत्तरशतं जप्त्वा ततो वै दृश्यते शुभम् // र्मञ्ज्_१०.४६ //
शुद्धस्फटिकसङ्काशं नानारूपधरो हरम् /
षण्मासाभ्यासयोगेन भूचराणां पतिर्भवेत् // र्मञ्ज्_१०.४७ //
वर्षद्वयेन हे नाथ कर्ता हर्ता स्वयं प्रभुः /
त्रिकालज्ञत्वम् आप्नोति स योगी नात्र संशयः // र्मञ्ज्_१०.४८ //
यत्कृताभ्यासयोगेन नास्ति किञ्चित् सुदुर्लभम् /
तद्रूपं कृष्णवर्णं च पश्यति व्योम्नि निर्मले // र्मञ्ज्_१०.४९ //
षण्मासान् मृत्युमाप्नोति स योगी नात्र संशयः /
पीतो व्याधिभयं रक्तो नीलो हत्यां विनिर्दिशेत् // र्मञ्ज्_१०.५० //
